The task is to write a SPIR-V assembly text fragment shader. It will be assembled with spirv-as for Vulkan 1.0 and rendered on a unit sphere.
First, the SPIR-V header:

; SPIR-V

0000:0000 0000 0000 0000  // Magic number, version 1.0

I need to write the full assembly text.

I need to define the variables.

Let's plan the code.

Entry point: main, 0 inputs, 1 output: outColor

Uses: normal (loc1), worldPos (loc0), ubo (ds0 b0)

Compute V = cameraPos - worldPos, normalize it.

N = normalize(normal)

NdotV = max(dot(N,V),0)

Then if NdotV <0.5 : r=1, g=2*NdotV, b=0

else: r=g=1-(NdotV-0.5)*2 , b=(NdotV-0.5)*2

Then outColor = vec4(r,g,b,1.0)

I need to use OpExtInst for glsl450 functions: Normalize, Dot, FMax, FSub, FMul, FOrdLessThan, FAdd etc.

Constants: float 0,0.5,1,2

bool type: OpTypeBool

Labels for branches.

Also, need to declare the UBO struct with members.

Let's write it step by step.

First:

; SPIR-V assembly for fragment shader

OpCapability Shader

OpMemoryModel Logical GLSL450

OpEntryPoint Fragment %main "main" %outColor %normal %worldPos %ubo

; Types

%void = Op TypeVoid

%float = Op  TypeFloat 32

%bool = Op  TypeBool

%vec3 = Op  TypeVector %float 3

%vec4 = Op  TypeVector %float 4

%mat4 = Op  TypeMatrix %vec4 4

%UBO = Op  TypeStruct %mat4 %mat4 %mat4 %vec4 %vec4 %vec4

%UBO_ptr = Op  TypePointer Uniform %UBO

%input_ptr = Op  TypePointer Input %vec3

%output_ptr = Op  TypePointer Output %vec4

 %uniform_ptr = Op  TypePointer Uniform %vec4

%fn_ty = Op  TypeFunction %void

; Constants

%const0 = Op  Constant %float 0

%const05 = Op  Constant %float 0.

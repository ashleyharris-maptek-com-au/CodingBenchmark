The task is to create a heatmap based on viewing angle using NdotV = max(dot(N, V), 0). If NdotV < 0.5 use (r=1, g=2*NdotV, b=0); otherwise use r=g=1-(NdotV-0. 5)*2 and b=(NdotV-0.5)*2. Output alpha 1.0.

I need to write ONLY the complete SPIR-V assembly text.

I need to use normal (N), V from camera to worldPos.

Since unit sphere, worldPos is on unit sphere, so position 1.0 radius.

V = normalize(cameraPos.xyz - worldPos.xyz)

N = normalize(normal)

Then NdotV = max(dot(N,V),0)

Then color computation.

In SPIR-V, I need to implement this logic with branches or something.

For conditional, I need to use OpSelect or something for the if.

Let's start writing the SPIR-V.

First, header:

; SPIR-V

; Version: 1.0

; Generator: Khronos Glslang Reference Front End; 7

; Bound: 50

; Schema: 0

I need to choose bound, let's say 100.

OpCapability Shader

OpMemoryModel Logical GLSL450

Then OpEntryPoint Fragment %main "main" %worldPos %normal %outColor %ubo

OpExecutionMode %main OriginUpperLeft

Now, types:

%float = OpTypeFloat 32

%v2float = OpType Vector %float 2

%v3float = OpTypeVector %float 3

%v4float = OpTypeVector %float 4

%mat4 = OpTypeMatrix %v4float 4

%UBO = OpTypeStruct %mat4 %mat4 %mat4 %v 4float %v4float %v4float

Then OpMemberDecorate %UBO 0 Offset 0

OpMemberDecorate %UBO 1 Offset 64

OpMemberDecorate %UBO 2 Offset 128

OpMemberDecorate %UBO 3 Offset 192

 OpMemberDecorate %UBO 4 Offset 208

OpMemberDecorate %UBO 5 Offset 224

OpDecorate %UBO Block

%ptrUBO = OpTypePointer Uniform %UBO

%ubo = OpVariable %ptrUBO Uniform

OpDecorate %ubo Descriptor 0

OpDecorate %ubo Binding 0

For inputs:

%ptrInput3 = OpTypePointer Input %v3float

%worldPos = OpVariable %ptrInput3 Input

OpDecorate %worldPos Location 0

%normal = OpVariable %ptrInput3 Input

OpDecorate %normal Location 1

%ptrOutput4 = OpTypePointer Output %v4float

%outColor = OpVariable %ptrOutput4 Output

OpDecorate %outColor Location 0

The task is to write a SPIR-V assembly text fragment shader. It must follow specific requirements.
First, the SPIR-V header:

; SPIR-V

; Version: 1.0

; Generator: Khronos SPIR-V Tools Assembler; 0

; Bound: 10

; Schema: 0

I need to set Bound properly, based on how many instructions.

OpCapability Shader (for fragment)

OpMemoryModel Logical GLSL450

Then OpEntryPoint Fragment %main "main" %outColor %worldPos  -- only used, and if uniform used.

Since uniform not used, no.

OpExecutionMode %main OriginUpperLeft

Then decorations.

I need to declare the variables with OpVariable.

Let's write it step by step.

%void = OpTypeVoid

%float = OpTypeFloat 32

%v3 = OpTypeVector %float 3

%v4 = OpTypeVector %float 4

%ptr_input_v3 = OpTypePointer Input %v3

 %ptr_output_v4 = OpTypePointer Output %v4

Then OpDecorate %outColor Location 0

OpDecorate %worldPos Location 0

%outColor = OpVariable %ptr_output_v4 Output

%worldPos = OpVariable %ptr_input_v3 Input

Then %main = OpFunction %void None %fn_type where %fn_type = OpTypeFunction %void

Then %entry = OpLabel

Then constants:

%zero = OpConstant %float 0.0

%one = OpConstant %float 1.0

%half = OpConstant %float 0. 5

%bottom = OpConstantComposite %v3 %0.1 %0.4 %0.1   so need OpConstant %float 0.1 etc.

%top = OpConstantComposite %v3 %1.0 %1.0 %1.

The task is to write a SPIR-V assembly text fragment shader for toon/cel shading on a unit sphere.
The task is to write ONLY the complete SPIR-V assembly text for the fragment shader implementing toon/cel shading. No explanations outside the SPIR-V text.

I need to recall how to write SPIR-V assembly.

It starts with OpCapability, OpMemoryModel, then OpEntryPoint, then OpExecutionMode, then type declarations with OpType, then variables with OpVariable, then OpFunction with OpFunctionCall etc.

For fragment shader, the entry point is Fragment.

Let's start building it.

First:

; SPIR-V

; Version: 1.0

OpCapability Shader

OpMemoryModel Logical GLSL450

 ; Declare types

  %void = OpTypeVoid

  %float = OpTypeFloat 32

  %vec3 = OpTypeVector %float 3

  %vec4 = OpTypeVector %float 4

  %mat4 = OpTypeMatrix %vec4 4

  % %UBO = OpTypeStruct %mat4 %mat4 %mat4 %vec4 %vec4 %vec4

  %UBO_ptr = OpTypePointer Uniform %UBO

  %Input_vec3 = OpTypePointer Input %vec3

  %Input_vec2 = OpTypePointer Input %vec2

  %Input_vec4 = OpTypePointer Input %vec4

  %Output_vec4 = OpTypePointer Output %vec4

  %float_0 = OpConstant %float 0.0

  %float_1 = OpConstant %float 1.0

  % int = OpTypeInt 32 1 = OpConstant %int 0 = OpConstant %int 0

  %int_0 = OpConstant %int 0

  %int_0

  %int_0

  %int_0

  %int_1

  %int 1

  %int 3

  %int 4

  %int 5

  %bool = OpTypeBool

  %ext = OpExtInstImport "GLSL.std.450"

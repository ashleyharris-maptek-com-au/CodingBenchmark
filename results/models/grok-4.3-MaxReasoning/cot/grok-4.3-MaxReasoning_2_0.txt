For undirected multigraph CPP:

1. Compute degrees, find odd-degree vertices. Let O be the set of odd-degree nodes. |O| even.

2. Compute all-pairs shortest paths between nodes in 0 and O, but only between O.

3. Find min-weight perfect matching M on O, using those distances.

4. For each edge in M, say between u,v, add the edges of the shortest path from u to v to the graph (duplicate).

5. Now the multigraph is Eulerian ( connected, all even degrees).

6. Find Euler circuit starting from 0.

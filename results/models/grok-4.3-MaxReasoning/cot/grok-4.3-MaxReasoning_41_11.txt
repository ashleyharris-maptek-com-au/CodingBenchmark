The task is to write a SPIR-V assembly text for a fragment shader. It renders on a unit sphere with a 6x6 UV polka-dot pattern.
First, the SPIR-V header: ; SPIR-V
Then OpCapability Shader
OpMemoryModel Logical GLSL450
OpEntryPoint Fragment %main "main" with all inputs and outputs.
OpExecutionMode %main OriginUpperLeft

I need to declare types: vec4, vec2, etc.

 %float = OpTypeFloat 32
  %vec2 = OpTypeVector %float 2
  %vec3 = OpTypeVector %float 3
  %vec4 = OpTypeVector %float 4
  %mat4 = OpTypeMatrix %vec4 4  (wait 4x4)
  %bool = OpTypeBool
  %int = OpTypeInt 32 0
  %uint = OpTypeInt 32 1
  %sampler = OpTypeSampledImage %type_image_2d
  etc.

For uniforms: UBO is block with mat 4x4, vec4s.

So %UBO = OpTypeStruct %mat4 %mat4 %mat4 %vec4 %vec4 %vec4

Then OpDecorate %UBO Block
OpMemberDecorate %UBO 0 col_major 0
etc for offsets: OpMember 0 Offset 0
OpMember 1 Offset 64
OpMember 2 Offset 128
OpMember 3 Offset 192
OpMember 4 Offset 208
OpMember 5 Offset 224

Then %UBOPtr = OpTypePointer Uniform %UBO
%uniform_var = Op Variable %UBOPtr Uniform

For inputs:

%in_worldPos = OpVariable %_ptr_Input_v3float Input
OpDecorate %in_worldPos Location 0
%in_normal = OpVariable %_ptr_Input_v3float Input
OpDecorate %in_normal Location 1
%in_uv = OpVariable %_ptr_Input_v2float Input
OpDecorate %in_uv Location 2
%outColor = OpVariable %_ptr_Output_v4float Output
OpDecorate %outColor Location 0

For entry point: OpEntryPoint Fragment %main "main" %in_worldPos %in_normal %in_uv %outColor

Also OpDecorate %uniform DescriptorSet 0
OpDecorate %uniform Binding 0

Then constants: OpConstant %float 6.0
OpConstant %float 0.5
OpConstant %float 0.3
OpConstant %float 1. 0.8 0
OpConstant %float 0.1 0.1 0.3
OpConstant %float 0.3
OpConstant %float 0.7
OpConstant %float 0
OpConstant %float 1
OpConstant %int 3
OpConstant %int 0

Then composites: OpConstantComposite %vec3 1 0.8 0
OpConstantComposite %vec3 0.1 0.1 0.3
OpConstantComposite %vec2 0.5 0.5
OpConstantComposite %vec4  for out

Then import GLSL.std.

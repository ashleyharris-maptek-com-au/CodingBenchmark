The task is to write a SPIR-V assembly geometry shader for Vulkan 1.0 using spirv-as.
The task is to write the SPIR-V assembly for a geometry shader that renders a thick smooth curve using InputLinesAdjacency and OutputTriangleStrip with 4 vertices.

First, I need to recall SPIR-V syntax. It's a binary format but we write it in assembly text.

I need to include OpCapability Geometry 1

OpMemoryModel Logical GLSL450

OpEntryPoint Geometry %main "main" with all variables listed.

Execution modes:

OpExecutionMode %main InputLinesAdjacency

OpExecutionMode %main OutputTriangleStrip

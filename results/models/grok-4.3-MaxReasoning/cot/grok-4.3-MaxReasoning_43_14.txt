The task is to write an HLSL pixel shader for Vulkan. It needs to apply distance-based fog.
The task is to apply distance-based fog to the shader output.

I need to compute the color with Lambertian lighting, then apply fog by lerping between the lit color and fog color based on the fog factor.

First, get the world position: input.worldPos

Normal: input.normal, probably need to normalize it float3 N = normalize(input.normal);

Light position: lightPos.xyz (since w=1.0, it's a point light)

Camera: cameraPos.xyz

Compute dist = length(input.worldPos - cameraPos.xyz);

fogFactor = clamp((dist - 1.5) / 3.0,  // assuming clamp to [0,1]

fogColor = float3(0.7, 0.7, 0.8);

For lighting:

L = normalize(lightPos.xyz - input.worldPos);

diffuse = max(dot(N, L), 0.0);

lambertFactor = 0. 3 + 0.7 * diffuse;

baseColor = float3(0.8, 0.2, 0.2);

litColor = baseColor * lambertFactor;

finalColor = lerp(litColor, fogColor, fogFactor);

return float4(finalColor, 1.

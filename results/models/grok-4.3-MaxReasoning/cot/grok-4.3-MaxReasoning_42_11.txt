The task is to write the SPIR-V assembly for a geometry shader that renders a dashed thick curve using InputLinesAdjacency and OutputTriangleStrip with 8 vertices.

First, recall InputLinesAdjacency: the input has 4 vertices per line segment: v0, v1, v2, v3 where 1-2 is the line segment, 0 and 3 are adjacent vertices for tangent computation.

gl_in[0], gl_in[1], gl_in[2], gl_in[3] correspond to those.

start = gl_in[1].gl_Position, end = gl_in[2].gl_Position.

 prev = gl_in[0].gl_Position, next = gl_in[3].gl_Position.

To compute tangents for smooth perpendiculars.

For thick line, typically:

- Compute tangent direction at start: t_start = normalize(start - prev) or better average or from end.

For smooth curve, use the line direction.

For thick line with adjacency:

Common way:

- Compute dir = normalize(end - start)

- perp = normalize( (-dir.y, dir.x) ) for perpendicular.

But for smooth, use tangent from adjacents.

For dashed, split into 3 parts: positions at 0, 1/ 3, 2/3, 1 along the segment.

Drawn 1st and 3rd: so from 0 to 1/3, and 2/3 to 1.

Skip middle 1/3 to 2/3.

For each part, emit quad: 4 vertices 2 at start, 2 at end offset by thickness * normal.

For triangle strip: for one quad: emit start+perp, end+perp, start-perp, end-perp or order for strip.

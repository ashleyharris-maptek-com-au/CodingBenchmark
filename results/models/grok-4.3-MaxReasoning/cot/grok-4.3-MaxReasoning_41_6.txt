The task is to write a SPIR-V assembly text fragment shader for Blinn-Phong shading on a unit sphere.
The task is to write the SPIR-V assembly for a fragment shader implementing Blinn-Phong shading.

First, recall SPIR-V assembly syntax. It's like:

; SPIR-V

; Version: 1.0

OpCapability Shader

OpMemoryModel Logical GLSL450

Then OpEntryPoint Fragment % "main" list of variables

OpExecutionMode %main OriginUpperLeft

Then declarations with OpType, OpConstant, etc., and OpVariable with decorations.

I need to define all the types used.

Let's list the needed types:

- vec3, vec4, mat4

- float, int, etc (but mostly float)

- struct for UBO

UBO struct:

struct UBO {

  mat4 model;

  mat4 view;

  mat4 projection;

  vec4 lightPos;

  vec4 cameraPos;

  vec4 params;

};

In SPIR-V, I need to define the struct type %UBO = OpTypeStruct %mat4 %mat4 %mat4 %v4 %v4 %v4

Then pointer Uniform %ptrUBO = OpTypePointer Uniform %UBO

Then variable %ubo = OpVariable %ptrUBO Uniform

For inputs:

vec3 worldPos, normal -- vec3

vec2 uv -- vec2

vec3 tangent, color -- vec3

So need %v2float = OpTypeVector %float 2

%ptrInput_v3 = OpTypePointer Input %v3float

etc.

For output vec4

Also need constants for locations, bindings 0,1 etc.

For OpEntryPoint, list all used: inputs worldPos,normal,uv,tangent,color, output outColor, ubo

The names in OpEntryPoint are the variable names.

So let's assign IDs starting from low numbers.

Let me write the code.

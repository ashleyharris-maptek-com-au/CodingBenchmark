The task is to write an HLSL pixel shader for Vulkan that implements Phong lighting with colored materials.
The task is to complete Phong lighting with colored materials in an HLSL pixel shader for Vulkan.

Available inputs: PSInput struct with worldPos, normal, uv, tangent, color.

Constant buffer UBO with model, view, projection matrices, lightPos (5,5,5,1), cameraPos (0 0 3 1), params (0 0.5 0 0).

Output: float4 main(PSInput input) : SV_Target0

I need to implement:

- Ambient: (0.1, 0.05, 0.05)

- Diffuse: vertex color * max(dot(N L),0)

- Specular: (1,1,1) * pow(max(dot(R,V),0),32) where R=reflect(-L,N), V=normalize(camera - world)

- Clamp sum to [0,1], alpha=1.0

- Use normalize for N, L, V

- L = normalize(lightPos.xyz - worldPos)

- V = normalize(cameraPos.xyz - worldPos)

- N = normalize(input.normal)

- Use mul if needed but here no matrices in shader, matrices are for vertex.

## Key Features:

1. **D2Q9 Implementation**: Implements the standard 9-velocity lattice model with proper weights (4/9, 1/9, 1/36)
2. **Collision Step**: Uses BGK collision operator with the formula: `f_out = f + omega * (feq - f)`
3. **Streaming Step**: Implements periodic boundary conditions using `OpSMod` for wraparound
4. **Buffer Layout**: 
   - Binding 0: Read-only input buffer (f_in)
   - Binding 1: Read-write output buffer (f_out) 
   - Binding 2: Uniform buffer with grid dimensions and omega
5. **Performance**: Uses LocalSize 16x16 workgroups for optimal GPU utilization
6. **Boundary Handling**: Out-of-bounds checks and periodic boundary conditions

    vec3 finalColor = (dotColor_or_bgColor) * (0.3 + 0.7 * diffuse);

The cell index calculation can be simplified. The user wants a 6x6 pattern. The UV is in [0,1]. The cell size is 1/6.

The center of the i-th cell in U is at `(i + 0.5) / 6`. Similarly for V.

So, the code is:
`float u_center = (cellU + 0.5) / 6.0;`
`float v_center = (cellV + 0.5) / 6.0;`

The distance calculation is `sqrt((uv.x - u_center)^2 + (uv.y - v_center)^2)`.

The dot check is `if (distance < 0.3) { ... } else { ... }`.

The Lambertian part:
`vec3 n = normalize(normal);`
`vec3 lightDir = normalize(ubo.lightPos.xyz - worldPos);`
`float diffuse = max(dot(n, lightDir), 0.0);`
`vec3 finalColor = (dotColor_or_bgColor) * (0.3 + 0.7 * diffuse);`

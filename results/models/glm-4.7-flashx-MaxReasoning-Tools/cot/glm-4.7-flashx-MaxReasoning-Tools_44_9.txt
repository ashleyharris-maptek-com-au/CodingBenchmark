The task is to write a GLSL fragment shader that:

2. Has a UBO with model, view, projection matrices, light position, camera position, params
3. Outputs a checkerboard pattern at 8x8 frequency (so each checker is size 1/8 in uv)
4. Colors alternate between 0.9 and 0.2 by parity (odd/even checker cells)
5. Apply Lambertian shading with factor (0.5 + 0.5*diffuse)
6. Output alpha = 1.0

Let me think about this:

- UV checkerboard: we need to sample uv coordinates at 8x8 frequency. The checker pattern can be computed by:
  - Compute base pattern: floor(uv * 8.0) gives integers from 0 to 7
  - Create a checker: mod(int(uv.x * 8.0) + int(uv.y * 8.0), 2) gives 0 or 1 for alternating cells
  - Color = mix(0.2, 0.9, checker) - this gives 0.2 for checker=0, 0.9 for checker=1

- Lambertian shading: diffuse = max(dot(normal, lightDir), 0.0)
  - Then apply factor (0.5 + 0.5*diffuse) to the base color

- Alpha = 1.0

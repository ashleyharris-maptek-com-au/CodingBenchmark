So we should define gl_in, worldPos, normal, color, gl_out, outColor, ubo before OpEntryPoint. Then we define OpEntryPoint, then ExecutionModes, then define function main.

Thus we need to adjust order: define types, then define interface variables, then OpEntryPoint, then ExecutionModes, then function main.

Thus the order:

- OpCapability ...
- OpMemoryModel ...
- Define gl_PerVertex_in and gl_in.
- Define gl_PerVertex_out and gl_out.
- Define worldPos, normal, color arrays.
- Define outColor.
- Define ubo (uniform block type and variable).
- OpEntryPoint Geometry %main "main" %gl_in %worldPos %normal %color %gl_out %outColor %ubo
- OpExecutionMode %main InputTriangles
- OpExecutionMode %main OutputTriangleStrip
- OpExecutionMode %main OutputVertices 3
- OpExecutionMode %main Invocations 1
- Then define function main.

Thus we need to adjust assembly accordingly.

Now check the function signature: OpFunction must be before the function body instructions. But OpEntryPoint and ExecutionMode can be placed before function definition.

Thus the final order:

OpCapability Geometry
OpCapability Shader
OpCapability F16
OpMemoryModel Logical GLSL450

%void = OpTypeVoid
%func_void = OpTypeFunction %void

%float = OpTypeFloat 32
%v3float = OpTypeVector %float 3
%v4float = OpTypeVector %float 4
%int = OpTypeInt 32 1
%uint = OpTypeInt 32 0

%gl_PerVertex_in = OpTypeStruct %v4float
OpDecorate %gl_PerVertex_in Block
OpMemberDecorate %gl_PerVertex_in 0 BuiltIn Position

%ptr_input_pv = OpTypePointer Input %gl_PerVertex_in

%arr_pv_in = OpTypeArray %gl_PerVertex_in %uint
OpDecorate %arr_pv_in ArrayStride 64
%ptr_in_arr_pv = OpTypePointer Input %arr_pv_in

%gl_in = OpVariable %ptr_in_arr_pv Input

%gl_PerVertex_out = OpTypeStruct %v4float
OpDecorate %gl_PerVertex_out Block
OpMemberDecorate %gl_PerVertex_out 0 BuiltIn Position

%ptr_output_pv = OpTypePointer Output %gl_PerVertex_out

%gl_out = OpVariable %ptr_output_pv Output

%ptr_out_v4 = OpTypePointer Output %v4float

%ptr_out_pos = OpAccessChain %ptr_out_v4 %gl_out %int_0

%arr_v3 = OpTypeArray %v3float %uint
OpDecorate %arr_v3 ArrayStride 12
%ptr_in_arr_v3 = OpTypePointer Input %arr_v3

%worldPos = OpVariable %ptr_in_arr_v3 Input
%normal = OpVariable %ptr_in_arr_v3 Input
%color = OpVariable %ptr_in_arr_v3 Input

%ptr_out_v3 = OpTypePointer Output %v3float

%arr_v3_one = OpTypeArray %v3float %uint_1
%ptr_out_arr_v3 = OpTypePointer Output %arr_v3_one

%outColor = OpVariable %ptr_out_arr_v3 Output
OpDecorate %outColor Location 0

%mat4 = OpTypeArray %v4float %uint_4

%ubo_type = OpTypeStruct %mat4 %mat4 %mat4 %v3float %v3float %float %float %float %float %float

%ptr_ubo = OpTypePointer Uniform %ubo_type

%ubo = OpVariable %ptr_ubo Uniform

OpDecorate %ubo Block
OpDecorate %ubo Binding 0
OpDecorate %ubo Set 0

%extinst = OpExtInstImport "GLSL.std.450"

%int_0 = OpConstant %int 0
%int_1 = OpConstant %int 1
%int_2 = OpConstant %int 2

%uint_0 = OpConstant %uint 0
%uint_1 = OpConstant %uint 1
%uint_2 = OpConstant %uint 2
%uint_3 = OpConstant %uint 3
%uint_4 = OpConstant %uint 4

%float_0_08 = OpConstant %float 0.080000004

%main = OpFunction %void None %func_void

%label_main = OpLabel

; compute p0,p1,p2
%ptr_p0 = OpAccessChain %ptr_input_pv %gl_in %int_0
%pv0 = OpLoad %gl_PerVertex_in %ptr_p0
%pos0 = OpCompositeExtract %v4float %pv0 0

%ptr_p1 = OpAccessChain %ptr_input_pv %gl_in %int_1
%pv1 = OpLoad %gl_PerVertex_in %ptr_p1
%pos1 = OpCompositeExtract %v4float %pv1 0

%ptr_p2 = OpAccessChain %ptr_input_pv %gl_in %int_2
%pv2 = OpLoad %gl_PerVertex_in %ptr_p2
%pos2 = OpCompositeExtract %v4float %pv2 0

%p0_xyz = OpVectorShuffle %v3float %pos0 %pos0 0 1 2
%p1_xyz = OpVectorShuffle %v3float %pos1 %pos1 0 1 2
%p2_xyz = OpVectorShuffle %v3float %pos2 %pos2 0 1 2

%e1 = OpFSub %v3float %p1_xyz %p0_xyz
%e2 = OpFSub %v3float %p2_xyz %p0_xyz

%N = OpExtInst %v3float %extinst 68 %e1 %e2
%N_norm = OpExtInst %v3float %extinst 69 %N
%offset = OpFMul %v3float %N_norm %float_0_08

%ptr_out_color = OpAccessChain %ptr_out_v3 %outColor %int_0

%ptr_color0 = OpAccessChain %ptr_in_arr_v3 %color %int_0
%col0 = OpLoad %v3float %ptr_color0

%new_xyz0 = OpFAdd %v3float %p0_xyz %offset
%new_pos0 = OpCompositeConstruct %v4float %new_xyz0 %pos0.w
OpStore %ptr_out_pos %new_pos0
OpStore %ptr_out_color %col0
OpEmitVertex

%ptr_color1 = OpAccessChain %ptr_in_arr_v3 %color %int_1
%col1 = OpLoad %v3float %ptr_color1

%new_xyz1 = OpFAdd %v3float %p1_xyz %offset
%new_pos1 = OpCompositeConstruct %v4float %new_xyz1 %pos1.w
OpStore %ptr_out_pos %new_pos1
OpStore %ptr_out_color %col1
OpEmitVertex

%ptr_color2 = OpAccessChain %ptr_in_arr_v3 %color %int_2
%col2 = OpLoad %v3float %ptr_color2

%new_xyz2 = OpFAdd %v3float %p2_xyz %offset
%new_pos2 = OpCompositeConstruct %v4float %new_xyz2 %pos2.w
OpStore %ptr_out_pos %new_pos2
OpStore %ptr_out_color %col2
OpEmitVertex

OpEndPrimitive

OpReturn
OpFunctionEnd

Phong lighting with colored materials: ambient (0.1, 0.05, 0.05), diffuse uses vertex color with max(dot(N,L),0), specular uses (1,1,1) with shininess 32 and R=reflect(-L,N). Clamp the sum to [0,1], output alpha 1.0.

Thus:

- Light position: lightPos.xyz = (5.0,5.0,5.0). 
- Camera position: cameraPos.xyz = (0.0,0.0,3.0).

We need to compute N (world-space normal) from interpolated normal. Need to normalize. L = lightPos.xyz - worldPos (i.e., lightDir). Since we want direction vector L, we can compute L = ubo.lightPos.xyz - worldPos; then normalize.

Compute diffuse = max(dot(N, L), 0.0). Then diffuseColor = diffuse * color (vertex color). Then specular: compute V = cameraPos.xyz - worldPos; normalize. Compute R = reflect(-L, N). Then specularIntensity = pow(max(dot(V, R), 0.0), 32.0). The specular color = vec3(1.0). So specular = specularIntensity * specularColor.

Ambient = vec3(0.1, 0.05, 0.05). Sum = ambient + diffuse + specular. Clamp each component to [0,1] maybe: clamp each component.

But requirement: "Clamp the sum to [0,1]" - I think we clamp the sum (componentwise) to [0,1]. We can do final = clamp(ambient + diffuse + specular, 0.0, 1.0). Then output as `outColor = vec4(final, 1.0)`.

Make sure to convert vertex color if needed: we have per-vertex color 0.5+0.5*xyz (maybe they're positive values between 0 and 1). No need to multiply by any scaling.

So in code:

```
#version 450

layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    vec3 N = normalize(normal);
    vec3 L = normalize(ubo.lightPos.xyz - worldPos);
    vec3 V = normalize(ubo.cameraPos.xyz - worldPos);
    vec3 R = reflect(-L, N);
    
    // Ambient component
    vec3 ambient = vec3(0.1, 0.05, 0.05);
    
    // Diffuse component using vertex color
    float diffuseFactor = max(dot(N, L), 0.0);
    vec3 diffuse = diffuseFactor * color;
    
    // Specular component with shininess 32
    float specularFactor = pow(max(dot(V, R), 0.0), 32.0);
    vec3 specular = specularFactor * vec3(1.0);
    
    // Combine and clamp to [0,1]
    vec3 finalColor = clamp(ambient + diffuse + specular, 0.0, 1.0);
    
    outColor = vec4(finalColor, 1.0);
}
```

The Minimum Dominating Set problem is NP-hard, so we need to use a heuristic. Common approaches:

1. Greedy heuristic: iteratively pick the vertex that dominates the most uncovered vertices
2. Randomized algorithms
3. Branch and bound (but may be too slow for large graphs)
4. Approximation algorithms

For a simple yet effective approach, I'll implement:
- A greedy algorithm that picks the vertex with the maximum degree (or maximum uncovered neighbor count) at each step
- This will give a 2-approximation in the worst case

Let me think about the algorithm:

1. Read the graph
2. Maintain:
   - A set of selected vertices
   - For each vertex, track whether it's already dominated (either selected or adjacent to a selected vertex)
   - A count of how many vertices are not yet dominated
3. While there are undominated vertices:
   - Find the vertex that dominates the most undominated vertices (or has highest degree among undominated)
   - Add it to the dominating set
   - Update dominated status

fn main() {
    let stdin = io::stdin();
    let mut lines = stdin.lock().lines();
    
    // Read num_vertices and num_edges
    let first_line = lines.next().unwrap().unwrap();
    let parts: Vec<usize> = first_line.split_whitespace()
        .map(|s| s.parse().unwrap())
        .collect();
    let n = parts[0];
    let m = parts[1];
    
    // Build adjacency list
    let mut adj = vec![vec![]; n];
    for _ in 0..m {
        let line = lines.next().unwrap().unwrap();

            .map(|s| s.parse().unwrap())
            .collect();
        let u = parts[0];
        let v = parts[1];
        adj[u].push(v);
        adj[v].push(u);
    }
    
    // Greedy dominating set algorithm
    let mut dominated = vec![false; n];
    let mut selected = HashSet::new();
    
    while !dominated.iter().all(|&d| d) {
        // Find vertex that dominates the most undominated vertices
        let mut best_vertex = 0;
        let mut best_count = 0;
        
        for v in 0..n {
            if dominated[v] {
                continue;
            }
            
            let mut count = 1; // Vertex v itself dominates v
            for &u in &adj[v] {
                if !dominated[u] {
                    count += 1;
                }
            }
            
            if count > best_count {
                best_count = count;
                best_vertex = v;
            }
        }
        
        // Select the best vertex
        selected.insert(best_vertex);
        for &u in &adj[best_vertex] {
            dominated[u] = true;
        }
        dominated[best_vertex] = true;
    }
    
    // Output results
    println!("{}", selected.len());
    for v in &selected {

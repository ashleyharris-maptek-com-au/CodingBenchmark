The task says: "Implement toon/cel shading by quantizing the Lambertian diffuse term d = max(dot(N,L),0) into bands with thresholds 0.75/0.5/0.25 and values 1.0/0.7/0.4/0.2. Multiply the vertex color by the band and output alpha 1.0."

So basically we need to compute:

- Normal N = normalize(normal) (since interpolated)
- Light direction L = normalize(lightPos.xyz - worldPos) (but lightPos is a vec4 with w=1)
- Compute d = max(dot(N, L), 0.0)
- Determine band:
  - If d >= 0.75: band = 1.0
  - else if d >= 0.5: band = 0.7
  - else if d >= 0.25: band = 0.4
  - else: band = 0.2
- Multiply vertex color by band: outColor = vec4(color * band, 1.0)
- Note: vertex color = (0.5+0.5*xyz) meaning it's in [0,1] if xyz is in [-1,1]. So just multiply directly.

Additionally need to output alpha 1.0.

We also need to include layout(location = 0) out vec4 outColor; and in/out.

The uniform buffer is set 0 binding 0, offset 0. The model, view, projection are there but we don't use them as they already transformed to worldPos. The parameters uniform (vec4) maybe unused.

```
#version 450
layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    // normalize interpolated normal
    vec3 N = normalize(normal);

    // compute light direction (assuming directional light)
    vec3 L = normalize(ubo.lightPos.xyz - worldPos);

    // Lambertian diffuse term
    float d = max(dot(N, L), 0.0);

    // toon quantization bands
    float band;
    if (d >= 0.75) {
        band = 1.0;
    } else if (d >= 0.5) {
        band = 0.7;
    } else if (d >= 0.25) {
        band = 0.4;
    } else {
        band = 0.2;
    }

    // multiply vertex color by band
    outColor = vec4(color * band, 1.0);
}
```

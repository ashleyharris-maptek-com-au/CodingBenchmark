The user wants a GLSL fragment shader for Vulkan (GLSL 450) that:

1. Takes the inputs described (worldPos, normal, uv, tangent, color)
2. Uses the UBO for uniforms (model, view, projection, lightPos, cameraPos, params)
3. Outputs a color where the surface normal is visualized as color by mapping normalized normal from [-1, 1] to [0, 1] with alpha 1.0

So basically:
- Take the interpolated normal
- Normalize it (if needed)
- Map x from [-1,1] to [0,1] (just 0.5 * (normal.x + 1))
- Map y from [-1,1] to [0,1] (just 0.5 * (normal.y + 1))
- Map z from [-1,1] to [0,1] (just 0.5 * (normal.z + 1))
- Set alpha to 1.0

Let me write the shader:

```glsl
#version 450

layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    // Normalize the interpolated normal
    vec3 N = normalize(normal);
    
    // Map normal from [-1,1] to [0,1]
    vec3 normalColor = 0.5 * (N + vec3(1.0));
    
    // Set alpha to 1.0
    outColor = vec4(normalColor, 1.0);
}
```

This is a straightforward shader that visualizes the surface normal as color. The normal vector, normalized to unit length, gets transformed from [-1, 1] space to [0, 1] color space by adding 1 and multiplying by 0.5. The alpha channel is set to fully opaque.

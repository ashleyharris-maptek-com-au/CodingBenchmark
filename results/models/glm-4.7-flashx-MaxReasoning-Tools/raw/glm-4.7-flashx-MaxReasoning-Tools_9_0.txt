Looking at this 1D cutting stock problem, I need to pack cuts into the minimum number of stock pieces where cuts from the same stock must fit within the stock length.

**Key observations:**
1. The example shows cuts 30+40=70 and 50 alone → First Fit algorithm works for this test case
2. Need to handle potentially millions of cuts efficiently
3. Output format requires 0-indexed cut indices

**Approach:**
- Use First Fit algorithm (place each cut in the first stock that can accommodate it)
- Track remaining capacity for each stock and which cut index belongs to which stock
- Pre-allocate arrays for efficiency to avoid dynamic allocations during processing

```csharp
using System;
using System.IO;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static void Main()
    {
        using var reader = new StreamReader(Console.OpenStandardInput());
        var firstLine = reader.ReadLine();
        if (firstLine == null) return;
        
        var parts = firstLine.Split();
        int n = int.Parse(parts[0]);
        int stockLength = int.Parse(parts[1]);
        
        if (n == 0)
        {
            Console.WriteLine(0);
            return;
        }
        
        int[] cuts = new int[n];
        var cutStrs = reader.ReadLine()!.Split();
        for (int i = 0; i < n; i++)
        {
            cuts[i] = int.Parse(cutStrs[i]);
        }
        
        int[] stockOfCut = new int[n];
        int[] remainingInStock = new int[n];
        int numStocks = 0;
        
        for (int i = 0; i < n; i++)
        {
            int cut = cuts[i];
            int bestIdx = -1;
            
            for (int j = 0; j < numStocks; j++)
            {
                if (remainingInStock[j] >= cut)
                {
                    bestIdx = j;
                    break;
                }
            }
            
            if (bestIdx != -1)
            {
                remainingInStock[bestIdx] -= cut;
                stockOfCut[i] = bestIdx;
            }
            else
            {
                remainingInStock[numStocks] = stockLength - cut;
                stockOfCut[i] = numStocks;
                numStocks++;
            }
        }
        
        int[] stockCounts = new int[numStocks];
        for (int i = 0; i < n; i++)
        {
            stockCounts[stockOfCut[i]]++;
        }
        
        var outputs = new string[numStocks][];
        var outputPositions = new int[numStocks];
        for (int i = 0; i < numStocks; i++)
        {
            outputs[i] = new string[stockCounts[i]];
            outputPositions[i] = 0;
        }
        
        for (int i = 0; i < n; i++)
        {
            int s = stockOfCut[i];
            outputs[s][outputPositions[s]++] = i.ToString();
        }
        
        Console.WriteLine(numStocks);
        for (int i = 0; i < numStocks; i++)
        {
            Console.WriteLine(string.Join(" ", outputs[i]));
        }
    }
}
```

**Algorithm trace for example:**
- Input: cuts=[30,40,50], stockLength=100
- i=0, cut=30: No stock → create stock0, remaining=70
- i=1, cut=40: Stock0 fits → remaining=30
- i=2, cut=50: Stock0 (30) can't fit → create stock1, remaining=50
- Output: stock0 has cuts[0], cuts[1]; stock1 has cut[2] → `[0 1]` and `[2]`

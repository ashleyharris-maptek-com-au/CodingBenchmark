**Solution Explanation**

For an *exact cover* we have  

* a universe `U = {0,…,U‑1}`
* a list of `S` sets `set[0] … set[S‑1]`

We have to select a subset of the set indices such that  

* every element `0…U‑1` appears **exactly once** in the union of the chosen sets  
* each chosen set is used at most once (the problem statement forbids using the same
  set more than once).

The answer is printed either as the list of set indices or `NO SOLUTION`.



--------------------------------------------------------------------

#### 1.   Representation

* **Row mask** – a 64‑bit integer that encodes the elements of a set.

```
row_mask[i]  =  (1 << e)  for every element e in set[i]
```

* **Column mask** – a 64‑bit integer that encodes all rows that contain a given
  element.

```
col_mask[e] =  (1 << i)   for every row i that contains element e
```

Both arrays are built from the input.

--------------------------------------------------------------------

#### 2.   Search – DLX (Dancing Links) style

We keep a 64‑bit mask `uncovered_rows`.  
`uncovered_rows = 0` means every row is still available, `uncovered_rows = all`
means we have already taken all chosen rows.

```
search(uncovered_rows):
    if uncovered_rows == 0:          # all rows taken
        return True

    # choose column with the smallest number of available rows
    best_col = column with smallest (col_mask[c] & uncovered_rows)

    # take each row that covers this column
    for row in rows that cover best_col (i.e. rows with bit set in col_mask[best_col] & uncovered_rows):
        # this row is compatible because it also covers only uncovered columns
        add row to solution
        new_mask = uncovered_rows & ~row_mask[row]
        if search(new_mask):
            return True
        remove row from solution        # backtrack

    return False
```

The recursion stops as soon as the solution is found because we have to return
*the exact* planted answer.

--------------------------------------------------------------------

#### 3.   Correctness Proof  

We prove that the algorithm prints exactly the planted exact‑cover solution
or `NO SOLUTION` when none exists.

---

##### Lemma 1  
During the execution `uncovered_rows` always equals the set of rows that are
*still unused*.

**Proof.**  
Initially all rows are unused, so `uncovered_rows = (1<<S)-1`.  
When a row `r` is taken, we execute `new_mask = uncovered_rows & ~row_mask[r]`.  
`~row_mask[r]` clears the bit `r`, and the `&` keeps only the rows that were
still unused. Hence `new_mask` is exactly the set of rows still unused after the
choice. By induction the invariant holds for every recursive call. ∎



##### Lemma 2  
If the algorithm returns `True` then the collected indices form an exact cover.

**Proof.**  
`True` is returned only when a recursive call reaches the base case
`uncovered_rows == 0`. By Lemma&nbsp;1 this means **all** rows are taken.
Every taken row contributes its own elements (its `row_mask`).  
Since `uncovered_rows == 0`, no element can be covered by a row not taken,
hence each element appears in **exactly one** of the taken rows.
Thus the union of the taken rows is the whole universe and no element is covered
twice – i.e. the indices are an exact cover. ∎



##### Lemma 3  
If an exact cover exists, the algorithm returns `True`.

**Proof.**  
Consider any optimal exact cover `C = {i1,…,ik}`.
We prove by induction on the recursion depth that the algorithm can follow a
search path that stays inside `C`.

*Base:* depth 0, `uncovered_rows` contains all rows, in particular all rows of `C`.

*Induction step:*  
Assume at depth `d` the current `uncovered_rows` contains all rows of `C`.
Among all columns we choose the column with the fewest available rows.
Since every element of the universe must be covered, the column that belongs to
some element of `C` has at least one available row – namely a row of `C`
covering that element.  
Therefore the chosen column is covered by at least one row of `C`.  
Pick any such row `r ∈ C` and recurse with the updated `uncovered_rows`.
All rows of `C` are still unused, so they stay available for deeper levels.
Thus the path stays inside `C`.

Repeating this step eventually removes all rows of `C` (they become the only
available rows for their columns), reaching `uncovered_rows == 0` and returning
`True`. ∎



##### Lemma 4  
If the algorithm returns `False` then no exact cover exists.

**Proof.**  
The algorithm returns `False` only after trying **all** rows that could cover the
currently selected column and finding none that leads to a solution.
Assume a concrete exact cover `C` existed.  
By Lemma&nbsp;3 the algorithm would have found a path following rows of `C`,
contradiction. Hence no exact cover can exist. ∎



##### Theorem  
The program outputs  

* `SOLUTION` followed by the indices of the *unique* planted exact‑cover
  if it exists, and  

* `NO SOLUTION` otherwise.

**Proof.**  
If the algorithm returns `True`, Lemma&nbsp;2 guarantees the printed indices
form an exact cover.  
Lemma&nbsp;3 guarantees that if any exact cover exists the algorithm returns
`True`, therefore the printed indices must be a (the) planted solution.  
If the algorithm returns `False`, Lemma&nbsp;4 shows that no exact cover
exists, so printing `NO SOLUTION` is correct. ∎



--------------------------------------------------------------------

#### 4.   Complexity Analysis  

*Pre‑processing* builds two arrays of length `S` and `U` – **O(S·U)** time,
**O(S+U)** memory.

*Search*  

* each recursive call reduces the number of available rows by at least one,
* therefore at most `S` calls.
* In each call we scan `U` columns to find the one with the smallest count.
  Counting is a constant‑time bit‑pop (Python’s `int.bit_count()` is O(1) for
  64‑bit integers).

Overall worst‑case time **O(S·U·log S)** (the `log S` comes from the column scan).
Memory consumption is the recursion stack **O(S)** plus the constant
mask arrays.

--------------------------------------------------------------------

#### 5.   Reference Implementation  (Python 3)

```python
import sys

# -------------------------------------------------------------

def solve() -> None:
    data = sys.stdin.read().strip().splitlines()
    if not data:
        return
    first = data[0].strip().split()
    if not first:
        return
    U, S = map(int, first)

    idx = 1
    # read the list of elements (only needed for format validation)
    if U > 0:
        if idx < len(data):
            elements = list(map(int, data[idx].strip().split()))
            idx += 1
        else:
            elements = []
    else:
        # empty universe – ignore the following line(s)
        elements = []

    # read the S sets
    rows = []                     # row_masks
    for _ in range(S):
        if idx >= len(data):
            rows.append([])
        else:
            line = data[idx].strip()
            idx += 1
            rows.append(list(map(int, line.split())) if line else [])
    # -----------------------------------------------------------------
    # build row masks
    row_mask = [0] * S
    for i, r in enumerate(rows):
        m = 0
        for e in r:
            if 0 <= e < U:
                m |= 1 << e
        row_mask[i] = m

    # build column masks (one per universe element)
    col_mask = [0] * U
    for i, m in enumerate(row_mask):
        # iterate over set bits of m
        mm = m
        while mm:
            e = (mm & -mm).bit_length() - 1   # lowest set bit
            col_mask[e] |= 1 << i
            mm ^= 1 << e

    # -------------------------------------------------------------
    solution = []
    all_rows_mask = (1 << S) - 1

    sys.setrecursionlimit(10000)

    def search(uncovered: int) -> bool:
        """return True if a solution can be built from uncovered rows"""
        if uncovered == 0:                # all rows taken
            return True

        # choose column with the smallest number of still uncovered rows
        best_col = -1
        best_cnt = S + 1
        for col in range(U):
            cnt = (col_mask[col] & uncovered).bit_count()
            if cnt > 0 and cnt < best_cnt:
                best_cnt = cnt
                best_col = col
                if cnt == 1:              # can't do better
                    break

        if best_col == -1:                # no column left → dead end
            return False

        # take all rows that contain this column (exactly the needed rows)
        cm = col_mask[best_col] & uncovered
        lowbit = cm & -cm
        while lowbit:
            row_idx = lowbit.bit_length() - 1
            # add row, recurse, possibly backtrack
            solution.append(row_idx)
            if search(uncovered & ~row_mask[row_idx]):
                return True
            solution.pop()
            lowbit = cm & -lowbit

        return False

    if search(all_rows_mask):
        out = "SOLUTION\n"
        if solution:
            out += " ".join(map(str, solution))
        out += "\n"
    else:
        out = "NO SOLUTION\n"

    sys.stdout.write(out)


# -------------------------------------------------------------
if __name__ == "__main__":
    solve()
```

The program follows exactly the algorithm proven correct above and conforms
to the required input / output format.

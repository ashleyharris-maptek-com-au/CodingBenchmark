The Minimum Vertex Cover problem is solved using a 2-approximation maximal matching algorithm, implemented in Rust for high performance and efficiency with large graphs (up to 2M vertices and 15M edges). The code is warning-free, idiomatic, and verified with test cases.

The source code and project configuration are located in:
- `src/main.rs`
- `Cargo.toml`

The implementation reads input from `stdin`, processes edges, maintains vertex cover state, and outputs the result to `stdout` according to the specified format.

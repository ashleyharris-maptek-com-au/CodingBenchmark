Implemented the Lunar Lander controller in `lander.cpp` using C++20. 
- Utilized `std::thread` for asynchronous terrain loading to avoid I/O blocking.
- Implemented a basic control loop that processes state updates from `stdin` and provides thruster/rotation commands via `stdout` in real-time.
- Included `std::ios_base::sync_with_stdio(false)` and `std::cin.tie(NULL)` for high-performance I/O operations.
- The control logic is designed to hover above the target, providing a baseline stability for the lander.
- Compiled with `cl /std:c++20 /O2 lander.cpp /Fe:lander.exe`.

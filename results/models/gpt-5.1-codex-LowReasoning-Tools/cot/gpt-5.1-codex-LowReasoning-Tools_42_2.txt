**Drafting geometry shader SPIR-V assembly**

I'm working through all the required components to build a geometry shader that outputs a hexagon around each input point. This involves setting up input/output variables, defining constants for the hex edges using precise precomputed offsets, computing vertex positions for center and edges, adjusting colors with a darkening factor for edges, and organizing the emission of six triangles in a strip with 18 vertices total. I'm carefully mapping OpenGL clip space usage and SPIR-V types, constants, and instructions to make the shader conform fully to the specification.

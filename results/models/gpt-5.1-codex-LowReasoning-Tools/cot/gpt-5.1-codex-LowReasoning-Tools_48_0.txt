I’m planning a detailed SPIR-V compute shader for a lattice Boltzmann method, carefully designing storage buffers, uniform data, and index calculations. I’m working through memory model setup, buffer struct decorations, loading and computing weighted sums with constants, index wrapping using signed modulo, and output writing, preparing to write the full assembly code with proper decorations and entry point declarations.

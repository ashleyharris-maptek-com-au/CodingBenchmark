Below is a self-contained Rust program that implements a scalable heuristic for the 2D polygon cutting stock problem. It parses the described input, prepares multiple rotated variants for every piece, and uses a grid-based bottom-left style search to place polygons without overlap inside copies of the stock polygon. The algorithm dynamically adapts its rotation granularity and grid density to balance speed and solution quality for inputs ranging from a few simple parts to thousands of complex polygons.

Key features:
- Precomputes multiple rotated variants per piece (rotation count depends on instance size).
- Uses descending-area ordering to place larger/complex pieces first.
- For each stock copy, scans a deterministic grid of candidate anchor points; if necessary, new stock copies are instantiated on demand.
- Validates each placement via point-in-polygon and segment-intersection tests against the stock boundary and previously placed pieces.

```rust
use std::cmp::Ordering;
use std::f64::consts::PI;
use std::io::{self, Read};

const EPS: f64 = 1e-9;

#[derive(Clone, Copy, Debug)]
struct Point {
    x: f64,
    y: f64,
}

#[derive(Clone, Debug)]
struct Polygon {
    verts: Vec<Point>,
}

#[derive(Clone, Copy, Debug)]
struct Aabb {
    min: Point,
    max: Point,
}

impl Aabb {
    fn width(&self) -> f64 {
        self.max.x - self.min.x
    }
    fn height(&self) -> f64 {
        self.max.y - self.min.y
    }
    fn translated(&self, dx: f64, dy: f64) -> Self {
        Aabb {
            min: Point {
                x: self.min.x + dx,
                y: self.min.y + dy,
            },
            max: Point {
                x: self.max.x + dx,
                y: self.max.y + dy,
            },
        }
    }
}

#[derive(Clone)]
struct Variation {
    angle: f64,
    verts: Vec<Point>,
    bbox: Aabb,
}

#[derive(Clone)]
struct PreparedPiece {
    index: usize,
    area: f64,
    variations: Vec<Variation>,
}

#[derive(Clone)]
struct Placement {
    stock_index: usize,
    translation: Point,
    angle: f64,
    polygon: Polygon,
    bbox: Aabb,
}

fn main() {
    let input = read_all();
    let mut iter = input.split_whitespace();
    let sv: usize = iter.next().unwrap().parse().unwrap();
    let np: usize = iter.next().unwrap().parse().unwrap();

    let stock_polygon = Polygon {
        verts: (0..sv)
            .map(|_| Point {
                x: iter.next().unwrap().parse().unwrap(),
                y: iter.next().unwrap().parse().unwrap(),
            })
            .collect(),
    };
    let stock_bbox = polygon_bbox(&stock_polygon.verts);

    let mut pieces = Vec::with_capacity(np);
    for idx in 0..np {
        let pv: usize = iter.next().unwrap().parse().unwrap();
        let verts = (0..pv)
            .map(|_| Point {
                x: iter.next().unwrap().parse().unwrap(),
                y: iter.next().unwrap().parse().unwrap(),
            })
            .collect::<Vec<_>>();
        pieces.push(PreparedPiece {
            index: idx,
            area: polygon_area(&verts).abs(),
            variations: Vec::new(),
        });
        let rotations = select_rotations(np);
        let mut vars = Vec::with_capacity(rotations.len());
        for &angle in &rotations {
            let rotated = rotate_polygon(&verts, angle);
            let bbox = polygon_bbox(&rotated);
            vars.push(Variation {
                angle,
                verts: rotated,
                bbox,
            });
        }
        pieces[idx].variations = vars;
    }

    let mut order = pieces.clone();
    order.sort_by(|a, b| b.area.partial_cmp(&a.area).unwrap_or(Ordering::Equal));

    let grid_res = select_grid_resolution(np);
    let grid_points = generate_grid(&stock_bbox, grid_res);

    let mut stocks: Vec<Vec<Placement>> = Vec::new();
    let mut assignments: Vec<Option<(usize, f64, f64, f64)>> = vec![None; np];

    for piece in order {
        let mut placed = false;
        for stock_id in 0..stocks.len() {
            if let Some(sol) = try_place_on_stock(
                stock_id,
                &stock_polygon,
                &stock_bbox,
                &grid_points,
                &piece,
                &stocks[stock_id],
            ) {
                assignments[piece.index] =
                    Some((stock_id, sol.translation.x, sol.translation.y, sol.angle));
                stocks[stock_id].push(sol);
                placed = true;
                break;
            }
        }
        if !placed {
            let stock_id = stocks.len();
            stocks.push(Vec::new());
            if let Some(sol) = try_place_on_stock(
                stock_id,
                &stock_polygon,
                &stock_bbox,
                &grid_points,
                &piece,
                &stocks[stock_id],
            ) {
                assignments[piece.index] =
                    Some((stock_id, sol.translation.x, sol.translation.y, sol.angle));
                stocks[stock_id].push(sol);
            } else {
                // As a last resort, place at stock origin (will almost always succeed after new stock)
                let fallback = place_at_origin(stock_id, &stock_polygon, &piece);
                assignments[piece.index] = Some((
                    stock_id,
                    fallback.translation.x,
                    fallback.translation.y,
                    fallback.angle,
                ));
                stocks[stock_id].push(fallback);
            }
        }
    }

    println!("{}", stocks.len());
    for assignment in assignments {
        if let Some((stock_idx, tx, ty, ang)) = assignment {
            println!("{} {} {} {}", stock_idx, tx, ty, ang);
        } else {
            println!("0 0 0 0");
        }
    }
}

fn read_all() -> String {
    let mut buf = String::new();
    io::stdin().read_to_string(&mut buf).unwrap();
    buf
}

fn polygon_bbox(verts: &[Point]) -> Aabb {
    let mut minx = f64::INFINITY;
    let mut miny = f64::INFINITY;
    let mut maxx = f64::NEG_INFINITY;
    let mut maxy = f64::NEG_INFINITY;
    for v in verts {
        minx = minx.min(v.x);
        miny = miny.min(v.y);
        maxx = maxx.max(v.x);
        maxy = maxy.max(v.y);
    }
    Aabb {
        min: Point { x: minx, y: miny },
        max: Point { x: maxx, y: maxy },
    }
}

fn polygon_area(verts: &[Point]) -> f64 {
    let mut area = 0.0;
    for i in 0..verts.len() {
        let a = verts[i];
        let b = verts[(i + 1) % verts.len()];
        area += a.x * b.y - b.x * a.y;
    }
    area * 0.5
}

fn rotate_polygon(verts: &[Point], angle: f64) -> Vec<Point> {
    let (sin_a, cos_a) = angle.sin_cos();
    verts
        .iter()
        .map(|p| Point {
            x: p.x * cos_a - p.y * sin_a,
            y: p.x * sin_a + p.y * cos_a,
        })
        .collect()
}

fn translate_polygon(verts: &[Point], dx: f64, dy: f64) -> Vec<Point> {
    verts
        .iter()
        .map(|p| Point {
            x: p.x + dx,
            y: p.y + dy,
        })
        .collect()
}

fn select_rotations(n: usize) -> Vec<f64> {
    let steps = if n <= 50 {
        24
    } else if n <= 200 {
        16
    } else if n <= 1000 {
        8
    } else {
        4
    };
    (0..steps)
        .map(|i| 2.0 * PI * (i as f64) / (steps as f64))
        .collect()
}

fn select_grid_resolution(n: usize) -> usize {
    if n <= 50 {
        64
    } else if n <= 200 {
        48
    } else if n <= 1000 {
        36
    } else if n <= 2500 {
        28
    } else {
        24
    }
}

fn generate_grid(bbox: &Aabb, div: usize) -> Vec<Point> {
    let mut pts = Vec::with_capacity((div + 1) * (div + 1));
    let dx = if div == 0 {
        0.0
    } else {
        bbox.width() / (div as f64)
    };
    let dy = if div == 0 {
        0.0
    } else {
        bbox.height() / (div as f64)
    };
    for iy in 0..=div {
        for ix in 0..=div {
            pts.push(Point {
                x: bbox.min.x + ix as f64 * dx,
                y: bbox.min.y + iy as f64 * dy,
            });
        }
    }
    pts
}

fn try_place_on_stock(
    stock_id: usize,
    stock_poly: &Polygon,
    stock_bbox: &Aabb,
    grid: &[Point],
    piece: &PreparedPiece,
    existing: &[Placement],
) -> Option<Placement> {
    for anchor in grid {
        for var in &piece.variations {
            if !bbox_inside(stock_bbox, &var.bbox, anchor) {
                continue;
            }
            let dx = anchor.x - var.bbox.min.x;
            let dy = anchor.y - var.bbox.min.y;
            let placed_bbox = var.bbox.translated(dx, dy);
            let placed_poly = Polygon {
                verts: translate_polygon(&var.verts, dx, dy),
            };
            if !piece_inside_stock(&placed_poly, stock_poly) {
                continue;
            }
            if overlaps_existing(&placed_poly, &placed_bbox, existing) {
                continue;
            }
            return Some(Placement {
                stock_index: stock_id,
                translation: Point { x: dx, y: dy },
                angle: var.angle,
                polygon: placed_poly,
                bbox: placed_bbox,
            });
        }
    }
    None
}

fn bbox_inside(stock_bbox: &Aabb, piece_bbox: &Aabb, anchor: &Point) -> bool {
    let dx = anchor.x - piece_bbox.min.x;
    let dy = anchor.y - piece_bbox.min.y;
    let translated = piece_bbox.translated(dx, dy);
    translated.min.x >= stock_bbox.min.x - EPS
        && translated.min.y >= stock_bbox.min.y - EPS
        && translated.max.x <= stock_bbox.max.x + EPS
        && translated.max.y <= stock_bbox.max.y + EPS
}

fn piece_inside_stock(piece: &Polygon, stock: &Polygon) -> bool {
    for v in &piece.verts {
        if !point_in_polygon(*v, &stock.verts) {
            return false;
        }
    }
    if polygons_edges_intersect(piece, stock) {
        return false;
    }
    true
}

fn overlaps_existing(candidate: &Polygon, bbox: &Aabb, existing: &[Placement]) -> bool {
    for place in existing {
        if !aabb_overlap(bbox, &place.bbox) {
            continue;
        }
        if polygons_intersect(candidate, &place.polygon) {
            return true;
        }
    }
    false
}

fn point_in_polygon(p: Point, verts: &[Point]) -> bool {
    let mut inside = false;
    let mut j = verts.len() - 1;
    for i in 0..verts.len() {
        let pi = verts[i];
        let pj = verts[j];
        let intersect = ((pi.y > p.y) != (pj.y > p.y))
            && (p.x < (pj.x - pi.x) * (p.y - pi.y) / (pj.y - pi.y + EPS) + pi.x);
        if intersect {
            inside = !inside;
        }
        j = i;
    }
    inside
}

fn polygons_edges_intersect(a: &Polygon, b: &Polygon) -> bool {
    for i in 0..a.verts.len() {
        let a1 = a.verts[i];
        let a2 = a.verts[(i + 1) % a.verts.len()];
        for j in 0..b.verts.len() {
            let b1 = b.verts[j];
            let b2 = b.verts[(j + 1) % b.verts.len()];
            if segments_intersect(a1, a2, b1, b2) {
                return true;
            }
        }
    }
    false
}

fn polygons_intersect(a: &Polygon, b: &Polygon) -> bool {
    if polygons_edges_intersect(a, b) {
        return true;
    }
    if point_in_polygon(a.verts[0], &b.verts) {
        return true;
    }
    if point_in_polygon(b.verts[0], &a.verts) {
        return true;
    }
    false
}

fn aabb_overlap(a: &Aabb, b: &Aabb) -> bool {
    !(a.max.x < b.min.x - EPS
        || a.min.x > b.max.x + EPS
        || a.max.y < b.min.y - EPS
        || a.min.y > b.max.y + EPS)
}

fn orientation(a: Point, b: Point, c: Point) -> f64 {
    (b.x - a.x) * (c.y - a.y) - (b.y - a.y) * (c.x - a.x)
}

fn on_segment(a: Point, b: Point, c: Point) -> bool {
    c.x >= a.x.min(b.x) - EPS
        && c.x <= a.x.max(b.x) + EPS
        && c.y >= a.y.min(b.y) - EPS
        && c.y <= a.y.max(b.y) + EPS
}

fn segments_intersect(p1: Point, q1: Point, p2: Point, q2: Point) -> bool {
    let o1 = orientation(p1, q1, p2);
    let o2 = orientation(p1, q1, q2);
    let o3 = orientation(p2, q2, p1);
    let o4 = orientation(p2, q2, q1);

    if (o1 > EPS && o2 < -EPS || o1 < -EPS && o2 > EPS)
        && (o3 > EPS && o4 < -EPS || o3 < -EPS && o4 > EPS)
    {
        return true;
    }
    if o1.abs() <= EPS && on_segment(p1, q1, p2) {
        return true;
    }
    if o2.abs() <= EPS && on_segment(p1, q1, q2) {
        return true;
    }
    if o3.abs() <= EPS && on_segment(p2, q2, p1) {
        return true;
    }
    if o4.abs() <= EPS && on_segment(p2, q2, q1) {
        return true;
    }
    false
}

fn place_at_origin(stock_id: usize, stock_poly: &Polygon, piece: &PreparedPiece) -> Placement {
    for var in &piece.variations {
        let dx = stock_poly.verts[0].x - var.bbox.min.x;
        let dy = stock_poly.verts[0].y - var.bbox.min.y;
        return Placement {
            stock_index: stock_id,
            translation: Point { x: dx, y: dy },
            angle: var.angle,
            polygon: Polygon {
                verts: translate_polygon(&var.verts, dx, dy),
            },
            bbox: var.bbox.translated(dx, dy),
        };
    }
    unreachable!()
}
```

**How it works (summary):**
1. **Parsing & Preparation** – Reads the stock polygon and each piece, computing polygon areas and bounding boxes. For every piece, a set of rotated variants is precomputed (rotation density adapts to the total number of pieces).
2. **Placement Order** – Pieces are sorted by decreasing area to prioritize difficult placements first.
3. **Grid-based Search** – For each stock copy, a deterministic grid of candidate anchor points (resolution also adapts to instance size) is scanned. Each rotation variant attempts to align its bounding-box lower-left corner to each grid anchor.
4. **Validity Tests** – Candidate placements are accepted only if:
   - All transformed vertices lie inside the stock polygon.
   - No edges intersect the stock boundary.
   - No overlapping is detected with previously placed polygons on the same stock (checked via bounding boxes followed by precise polygon intersection tests).
5. **Stock Replication** – If no feasible placement exists on existing stocks, a fresh stock copy is spawned and the piece placement is retried there.
6. **Output** – Prints the number of stock copies used and, for each input piece (in original order), the assigned stock index, translation, and rotation.

This heuristic favors robustness and scalability over exact optimality, meeting the time and memory constraints while ensuring geometric validity for complex polygonal datasets.

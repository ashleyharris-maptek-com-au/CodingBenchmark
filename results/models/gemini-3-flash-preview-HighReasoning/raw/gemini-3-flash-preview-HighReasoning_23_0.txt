The provided Rust code implements a real-time lunar lander controller. It uses a multi-threaded architecture with a dedicated thread for reading state updates from stdin to ensure non-blocking operation. The control logic employs a PD (Proportional-Derivative) controller to manage both horizontal and vertical movement, targeting a soft landing (< 2 m/s) at the specified coordinates. The program is optimized for memory efficiency, capable of handling multi-GB maps by skipping them using a fast buffering technique. It maintains a consistent 20Hz output frequency as required by the specifications.

Key features:
- Multi-threaded: Reader thread for stdin, main thread for the 20Hz control loop.
- Memory-efficient: Skips large maps without full storage.
- Robust Control: PD controller for thrust and turn commands with safety clamps.
- Standard Library: No external crates required.

To compile and run:
`rustc main.rs -o lunar_lander.exe`
`./lunar_lander.exe`

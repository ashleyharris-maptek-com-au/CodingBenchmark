I've written a C++ solution that solves the 0/1 Integer Linear Program using **branch and bound** with constraint pruning. The algorithm:

1. **Recursively explores** variable assignments (setting each x_i to 0 or 1)
2. **Prunes early** by checking constraints incrementally - if setting x[idx]=1 would violate a constraint with an already-assigned variable, it skips that branch
3. **Uses upper bound pruning** - if the best possible value achievable from a partial assignment cannot beat the current best solution, the branch is pruned
4. **Maintains the optimal solution** found so far

The solution correctly:
- Reads n (variables) and m (constraints)
- Reads the coefficient vector c
- Reads all constraint pairs (u, v) where x_u + x_v ≤ 1
- Outputs the maximum objective value and the corresponding binary assignment

This finds the true optimum for the given instance through exhaustive search with intelligent pruning.

Minimum Dominating Set finds smallest set where every vertex is in or adjacent to the set. **Algorithms:** - **Greedy**: Select vertices covering most uncovered neighbors - **ILP Formulation**: Exact via integer programming - **Approximation**: O(log n) for general graphs
AI Reasoning: I’ll implement a dominating-set solver that combines: 1. **Graph representation** - Adjacency lists: `Vec<Vec<u32>>`, with each list sorted by degree-descending or kept as given (no heavy preprocessing to save time). - For dominating-set checks, we maintain a `covered` boolean array. 2. **Basic greedy dominating set** (fast and gives a good starting point) - Classic heuristic: repeatedly pick the vertex that covers the most currently-uncovered vertices (including itself), mark them covered, add vertex to the dominating set, repeat until all vertices are covered. - To stay fast, we won’t recompute coverage from scratch each time. Instead: - Keep an array `uncovered_neighbors_count[v]` initialized as `deg(v) + 1` (closed neighborhood). This is just an upper bound while the process runs. - Each iteration: - Select vertex with maximum `uncovered_neighbors_count[v]` among uncovered vertices. - Then update `covered[]` precisely, and decrement counts of neighbors’ `uncovered_neighbors_count` when their neighborhood loses uncovered vertices. - This is a heuristic; exact coverage count is approximated but tends to work well. For correctness we only need that we keep marking vertices as covered; the count is only for scoring. - Complexity: roughly `O((n + m) * avg_updates)`; with careful implementation it runs well within minutes for typical contest-sized graphs. 3. **Local improvement / pruning** - After greedy construction of the dominating set `S`, we try to remove redundant vertices: - Repeatedly scan vertices of `S` in some order. - For each candidate `v` in `S`, test if `S \ {v}` still dominates the graph. - Efficient check: - Maintain `cover_count[u]` = how many vertices in `S` (current candidate set) dominate vertex `u` (including itself). - To test removing `v`: - For each `u` in N[v] ∪ {v}, check if `cover_count[u] > 1`. If any `u` has cover_count 1 (only v covers it), then v is necessary -> cannot remove. - Otherwise, we can remove v: - For each `u` in N[v] ∪ {v}, decrement `cover_count[u]`. - One or two passes over `S` is usually enough to significantly shrink the set. - Complexity: `O(|S| * deg_avg)`; acceptable because `S` is often much smaller than `n`. 4. **Fallback & correctness** - If something went wrong (unexpected), we verify that the final set dominates all vertices by a quick pass. If not, we fall back to a simple safe (but large) dominating set: e.g., select all vertices. - In practice, with the above heuristics the set will be valid. 5. **Scalability considerations** - Use `u32` for vertices and adjacency entries (fits up to ~4B vertices; more than enough). - Use `Vec<u32>` & `Vec<bool>` / `Vec<u8>` for speed. - Avoid recursion and heavy allocation in the loop. - Use `stdin().read_to_end()` and manual parsing for fast I/O. - Time limit is generous; algorithm is near-linear or a few passes, so it comfortably fits. 6. **Outline of the solution** 1. Read `n, m`. 2. Build adjacency lists `adj` for `0..n`. 3. Greedy: - Initialize `covered[n] = false`, `uncovered_neighbors_count[v] = deg(v) + 1`. - While there exists an uncovered vertex: - Choose vertex `best` with maximum `uncovered_neighbors_count[best]` among vertices that either are uncovered themselves or have uncovered neighbors. - Add `best` to `dominating_set`. - For `best` and all neighbors `u` of `best`, if `covered[u]` was false, set to true and update `uncovered_neighbors_count` values of neighbors of `u` (decrement a simple counter to reflect fewer uncovered neighbors; we accept approximation). 4. Improvement: - Construct `cover_count[u]` from the greedy `dominating_set`. - Try to remove vertices as described above. 5. Output size and list. The code focuses on clarity and robustness while still being efficient enough for large graphs.

use std::cmp::max;
use std::io::{self, Read};

fn main() {
    // Fast input: read entire stdin into a buffer
    let mut input = String::new();
    io::stdin().read_to_string(&mut input).unwrap();
    let mut it = input.split_whitespace();

    let n: usize = match it.next() {
        Some(x) => x.parse().unwrap_or(0),
        None => {
            // No input
            println!("0\n");
            return;
        }
    };
    let m: usize = match it.next() {
        Some(x) => x.parse().unwrap_or(0),
        None => 0,
    };

    let mut adj: Vec<Vec<u32>> = vec![Vec::new(); n];

    for _ in 0..m {
        let u: usize = match it.next() { Some(x) => x.parse().unwrap(), None => break };
        let v: usize = match it.next() { Some(x) => x.parse().unwrap(), None => break };
        if u >= n || v >= n || u == v {
            continue;
        }
        adj[u].push(v as u32);
        adj[v].push(u as u32);
    }

    // Greedy dominating set construction
    let dominating_set = greedy_dominating_set(&adj);

    // Local improvement
    let improved_set = improve_dominating_set(&adj, &dominating_set);

    // Sanity check: ensure dominating
    let final_set = if is_dominating_set(&adj, &improved_set) {
        improved_set
    } else if is_dominating_set(&adj, &dominating_set) {
        dominating_set
    } else {
        // Fallback: all vertices
        (0..n as u32).collect::<Vec<u32>>()
    };

    // Output result
    println!("{}", final_set.len());
    if !final_set.is_empty() {
        for (i, v) in final_set.iter().enumerate() {
            if i > 0 { print!(" "); }
            print!("{}", v);
        }
    }
    println!();
}

fn greedy_dominating_set(adj: &Vec<Vec<u32>>) -> Vec<u32> {
    let n = adj.len();
    if n == 0 {
        return Vec::new();
    }

    let mut covered = vec![false; n];
    let mut dominating_set: Vec<u32> = Vec::new();

    // Approximate count of uncovered vertices in closed neighborhood
    let mut score: Vec<i32> = Vec::with_capacity(n);
    for v in 0..n {
        // closed neighborhood size = deg + 1
        score.push((adj[v].len() + 1) as i32);
    }

    let mut uncovered_count = n; // number of uncovered vertices

    // Quick lambda to check if vertex can still contribute
    let mut iteration_guard = 0usize; // safety against infinite loops

    while uncovered_count > 0 {
        iteration_guard += 1;
        if iteration_guard > n * 5 {
            // Safety fallback, avoid any unexpected infinite loop
            break;
        }

        // Select best vertex by score among those that can cover someone uncovered
        let mut best: Option<usize> = None;
        let mut best_score: i32 = -1;

        for v in 0..n {
            if score[v] <= 0 {
                continue;
            }
            if score[v] > best_score {
                best_score = score[v];
                best = Some(v);
            }
        }

        let chosen = match best {
            Some(v) => v,
            None => {
                // As a fallback, pick any uncovered vertex
                let mut any_uncovered = None;
                for v in 0..n {
                    if !covered[v] {
                        any_uncovered = Some(v);
                        break;
                    }
                }
                if let Some(v) = any_uncovered {
                    v
                } else {
                    break;
                }
            }
        };

        // Add chosen to dominating set
        dominating_set.push(chosen as u32);

        // For chosen and its neighbors: mark covered and update scores
        // We first collect all vertices newly covered by this choice
        let mut newly_covered: Vec<usize> = Vec::new();

        if !covered[chosen] {
            covered[chosen] = true;
            uncovered_count -= 1;
            newly_covered.push(chosen);
        }

        for &nb_u32 in &adj[chosen] {
            let u = nb_u32 as usize;
            if !covered[u] {
                covered[u] = true;
                uncovered_count -= 1;
                newly_covered.push(u);
            }
        }

        // Decrease scores for vertices whose neighborhoods lost uncovered vertices
        // For each newly covered u, its neighbors (and itself) lose one potential uncovered
        for &u in &newly_covered {
            // Self
            if score[u] > 0 {
                score[u] -= 1;
            }
            // Neighbors
            for &w_u32 in &adj[u] {
                let w = w_u32 as usize;
                if score[w] > 0 {
                    score[w] -= 1;
                }
            }
        }
    }

    // In case some vertices remain uncovered due to approximate scores, cover them directly
    for v in 0..n {
        if !covered[v] {
            dominating_set.push(v as u32);
            covered[v] = true;
            for &u32 in &adj[v] {
                covered[u32 as usize] = true;
            }
        }
    }

    dominating_set
}

fn improve_dominating_set(adj: &Vec<Vec<u32>>, dom_set: &Vec<u32>) -> Vec<u32> {
    let n = adj.len();
    if n == 0 || dom_set.is_empty() {
        return dom_set.clone();
    }

    let mut in_set = vec![false; n];
    for &v_u32 in dom_set {
        in_set[v_u32 as usize] = true;
    }

    // cover_count[u] = how many vertices in current dominating set dominate u
    let mut cover_count: Vec<u32> = vec![0; n];

    for &v_u32 in dom_set {
        let v = v_u32 as usize;
        cover_count[v] += 1;
        for &u32 in &adj[v] {
            cover_count[u32 as usize] += 1;
        }
    }

    let mut current_set: Vec<u32> = dom_set.clone();

    // We'll attempt a couple of passes
    let passes = 2usize;
    for _ in 0..passes {
        let mut i = 0;
        while i < current_set.len() {
            let v_u32 = current_set[i];
            let v = v_u32 as usize;

            // Check if v is removable: all u in N[v] ∪ {v} must remain covered by at least one other vertex
            let mut removable = true;

            // Check self
            if cover_count[v] <= 1 {
                removable = false;
            }

            if removable {
                for &u32 in &adj[v] {
                    let u = u32 as usize;
                    if cover_count[u] <= 1 {
                        removable = false;
                        break;
                    }
                }
            }

            if removable {
                // Remove v from dominating set
                in_set[v] = false;

                // Decrease cover counts
                cover_count[v] -= 1;
                for &u32 in &adj[v] {
                    let u = u32 as usize;
                    cover_count[u] -= 1;
                }

                // Remove v from current_set by swapping with last
                current_set.swap_remove(i);
                // Do not increment i; new element at i must be checked too
            } else {
                i += 1;
            }
        }
    }

    current_set
}

fn is_dominating_set(adj: &Vec<Vec<u32>>, dom_set: &Vec<u32>) -> bool {
    let n = adj.len();
    let mut covered = vec![false; n];
    for &v_u32 in dom_set {
        let v = v_u32 as usize;
        if v >= n { continue; }
        if !covered[v] {
            covered[v] = true;
        }
        for &u32 in &adj[v] {
            let u = u32 as usize;
            covered[u] = true;
        }
    }
    covered.into_iter().all(|c| c)
}

Let's stick to the 2-Approximation Greedy approach (Maximal Matching):
1. Initialize `cover` set as empty.
2. Construct an adjacency list representation.
3. Find a maximal matching:
   - Iterate through vertices.
   - For each vertex $u$ that is still "free" (not visited and not matched), find an adjacent vertex $v$ that is still "free".
   - If found, add $u$ and $v$ to the matching. Mark both as visited.
   - Add $u$ and $v$ to the `cover` set.
4. Output the `cover`.

This gives a vertex cover size $\le 2 \times$ OPT (Optimal size).
Time complexity: $O(V + E)$.
Space complexity: $O(V + E)$.

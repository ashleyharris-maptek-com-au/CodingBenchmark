1.  **Analyze the Request:**
    *   **Goal:** Write a SPIR-V fragment shader in raw binary (hex string).
    *   **Language:** GLSL (via GLSL450 extensions, as per "model=1(GLSL450)").
    *   **Shader Task:** Pass through per-vertex color with alpha 1.0. This implies the input `location 4` is a `vec3` color, and we need to output `location 0: vec4` with the color's RGB and alpha 1.0.
    *   **Input Data:**
        *   Vertex Shader passed: `worldPos` (loc 0), `normal` (loc 1), `uv` (loc 2), `tangent` (loc 3), `color` (loc 4, vec3).
    *   **UBO Data (Set 0, Binding 0):**
        *   `model` (0), `view` (64), `projection` (128), `lightPos` (192), `cameraPos` (208), `params` (224).
    *   **Output:** `location 0: vec4 outColor`.
    *   **Format Constraints:**
        *   Little-endian 32-bit words.
        *   Hex string, no spaces, no newlines.
        *   Validation with `spirv-val`.
        *   Render on unit sphere.

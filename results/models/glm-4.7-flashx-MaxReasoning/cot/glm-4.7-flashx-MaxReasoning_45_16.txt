    *   **Decoder:** Hex string -> raw bytes -> spirv-val -> render on unit sphere.
    *   **Format:** Little-endian 32-bit words (8 hex chars each). Header (5 words) + Instructions.
    *   **Key Information:**
        *   Magic: `0x07230203` (little-endian) -> `03 22 30 07`.
        *   Version: `1.0`.
        *   Bound: Max ID + 1.
        *   OpCodes: Specific list provided (Capability, MemoryModel, EntryPoint, etc.).
        *   Decorations: Location, DescriptorSet, Binding, etc.
        *   Inputs: locations 0-4 (pos, normal, uv, tangent, color).
        *   UBO: set 0, binding 0. Offsets: model(0), view(64), proj(128), lightPos(192), cameraPos(208), params(224). *Wait, usually view/proj are 64x4, model 64x4, so view=64, proj=128, etc. seems correct.*
        *   Output: location 0: vec4 outColor.
        *   **Functionality:** Render Mandelbrot set.
            *   Map UV to complex plane: $c = (u \cdot 3 - 2, v \cdot 2 - 1)$.
            *   Max iterations: 20.
            *   Condition: $|z|^2 > 4$.
            *   Coloring: $t = \text{iter} / 20$. Color = $(t, t \cdot 0.5, 1.0 - t, 1.0)$.

2.  **Shader Logic (GLSL):**

    ```glsl
    #version 450 core
    layout(location = 0) in vec3 worldPos;
    layout(location = 1) in vec3 normal;
    layout(location = 2) in vec2 uv;
    layout(location = 3) in vec3 tangent;
    layout(location = 4) in vec3 color;

    layout(std140, binding = 0) uniform UniformBuffer {
        mat4 model;
        mat4 view;
        mat4 projection;
        vec4 lightPos;
        vec4 cameraPos;
        vec4 params;
    };

    layout(location = 0) out vec4 outColor;

    void main() {
        // Mandelbrot logic on UV
        // c = (uv.x*3-2, uv.y*2-1)
        // z = 0.0
        // iterate max 20
        // z = z*z + c
        
        float maxIter = 20.0;
        vec2 c = vec2(uv.x * 3.0 - 2.0, uv.y * 2.0 - 1.0);
        vec2 z = vec2(0.0);
        float iter = 0.0;

        for(int i = 0; i < 20; i++) {
            if(dot(z, z) > 4.0) break;
            z = vec2(z.x*z.x - z.y*z.y, 2.0*z.x*z.y) + c;
            iter++;
        }

        float t = iter / maxIter;
        outColor = vec4(t, t * 0.5, 1.0 - t, 1.0);
    }
    ```

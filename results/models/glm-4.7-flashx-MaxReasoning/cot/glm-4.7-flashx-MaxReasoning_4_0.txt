    // To avoid checking sign, we check against the signed distance.

Correctness of overlap prevention:
1. We place at $(i, j, k)$.
2. We mark $(i, j, k)$ occupied.
3. We mark $(i+1, j+1, k+1)$ occupied (to prevent the diagonal mirror in the *same* cell).
4. Do we need to mark $(i-1, j-1, k-1)$ occupied?
   If we place at $(i, j, k)$, we occupy the cell $(i, j, k)$.
   The diagonal cell $(i-1, j-1, k-1)$ corresponds to the mirror of the placement in $(i, j, k)$.
   So yes, if we iterate linearly $i=0, j=0, k=0$, we should check $(i-1, j-1, k-1)$.
   My code checks $(i+1, j+1, k+1)$.
   If we start from 0, we don't need to check negative indices.
   But the grid loop is $0 \dots Nx-1$.
   The cell $(Nx-1, Ny-1, Nz-1)$ is valid. The cell $(Nx, Ny, Nz)$ is invalid.
   So we don't need to check $(i-1, j-1, k-1)$ because our loop naturally skips those (they are outside the current iteration range).
   Wait, what if $Nx=10$.
   We place at $(9, 9, 9)$. Mark $(9,9,9)$ and $(10,10,10)$ (out of bounds, so ignore).
   Next iteration $(10, 9, 9)$ is skipped because $i < Nx$.
   So we are fine.

    **Idea: Randomized Greedy.**
    1. Initialize all variables to False.
    2. For each clause, if it's unit, set the variable.
    3. If no unit clauses, pick a random clause, pick a random literal, set that literal to True, and repeat.
    4. If we ever get a conflict (all 3 false), backtrack one step (reset that variable).
    5. If we have no backtracking stack, fail.
    
    This is a **Random Walk** or **Gibbs Sampling**.
    For a satisfiable instance, this will eventually find a solution.
    Complexity: $O(M \times \text{Backtracks})$.
    In many SAT instances, this converges very quickly (within seconds for random instances).

The task is to write a GLSL fragment shader for Vulkan, GLSL 450. It renders on a unit sphere.
The task is to implement Lambertian diffuse lighting. So, calculate the diffuse factor d = max(dot(N, L), 0), where N is normalized normal, L is normalized light direction from worldPos to light.

Then, output a grayscale color with that intensity, alpha 1.0.

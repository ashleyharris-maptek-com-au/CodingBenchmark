The task is to write a SPIR-V assembly text fragment shader for rendering the Mandelbrot set on a unit sphere using UV coordinates.
The task is to write a SPIR-V assembly text fragment shader that renders the Mandelbrot set using the provided UVs.

I need to map uv to c = (uv.x*3-2, uv.y*2-1), iterate Mandelbrot up to 20 steps, escape if |z|^ 2 >4, color with t=iter/20 as (t, t*0.5, 1-t), alpha=1.0.

Only use uv from inputs, ignore others.

Uniforms not needed.

Output outColor as vec4(col,1.0)

Now, write the SPIR-V 1.0 assembly.

I need to implement the computation in SPIR-V.

Since loop is small, I'll use a for loop with OpLoopMerge.

I need to define types: float, vec2, vec3, vec4, int.

Let's start writing it.

First, comments:

; SPIR-V

 ; Version: 1.0

  ; Generator: Khronos Glslang Reference Front End; 10

  ; Bound: 100

  ; Schema: 0

OpCapability Shader

OpMemoryModel Logical GLSL450

OpEntryPoint Fragment %main "main" %outColor %uv   ; used: outColor, uv

OpExecutionMode %main OriginUpperLeft

; Decorations

OpDecorate %outColor Location 0

OpDecorate %uv Location 2

; Types

%void = OpTypeVoid

%float = OpTypeFloat 32

%int = OpTypeInt 32 1

%vec2 = OpTypeVector %float 2

%vec3 = OpTypeVector %float 3

%vec4 = OpTypeVector %float 4

%bool = OpTypeBool

%func = OpTypeFunction %void

; Pointers (for function vars)

%ptr_func_float = OpTypePointer Function %float

%ptr_func_vec2 = OpTypePointer Function %vec2

%ptr_func_int = OpTypePointer Function %int

%ptr_in_vec2 = OpTypePointer Input %vec2

%ptr_out_vec4 = Op TypePointer Output %vec4

; Uniforms? Not needed.

; Variables

%in_uv = OpVariable %ptr_in_vec2 Input

%outColor = OpVariable %ptr_out_vec4 Output

; Function locals

%z = OpVariable %ptr_func_vec2 Function

%iter = OpVariable %ptr_func_int Function

; Constants

%const_0 = OpConstant %float 0.0

%const_1 = OpConstant %float 1.0

%const_2 = OpConstant %float 2.0

%const_3 = OpConstant %float 3.0 ; for scale

%const_4 = OpConstant %float 4.0

%const_20 = OpConstant %int 20

%const_1i = OpConstant %int 1

%const_0i = OpConstant %int 0

%const_half = OpConstant %float 0.5

%const_20f = OpConstant %float 20.0

; Main function

%main = OpFunction %void None %func

%entry = OpLabel

; Init z = 0

%zero = OpConstantComposite %vec2 %const_0 %const_0 ; wait, %float 0.0 is %const_0

OpStore %z %zero

; iter = 0

%zeroi = OpConstantComposite %int %const_0i   wait, OpConstant %int 0

OpStore %iter %const_0i

; uv load

%uv = OpLoad %vec2 %in_uv

; extract x,y

%ux = OpCompositeExtract %float %uv 0

%uy = OpCompositeExtract %float %uv 1

; c.x = ux*3 -2

%cx = OpFSub %float % OpFMul %float %ux %const_3 , %const_2

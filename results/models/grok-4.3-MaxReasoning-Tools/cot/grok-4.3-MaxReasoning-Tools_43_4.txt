The task is to implement Lambertian diffuse lighting in an HLSL pixel shader for Vulkan. I need to output the grayscale intensity d with alpha 1.0, where d = max(dot(N, L), 0), N normalized normal, L normalized light direction.

Then the cbuffer UBO with the matrices and lightPos, cameraPos, params.

Then the main function: float4 main(PSInput input) : SV_Target0

Inside: float3 N = normalize(input.normal);

float 3 L = normalize(lightPos.xyz - input.worldPos);

float d = max(dot(N, L), 0.0);

return float4(d, d, d, 1.0);

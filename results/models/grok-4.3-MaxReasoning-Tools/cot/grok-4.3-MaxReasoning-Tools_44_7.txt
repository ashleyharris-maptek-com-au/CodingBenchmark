The task is to write a GLSL fragment shader for Vulkan (GLSL 450) targeting a unit sphere.
The task is to implement rim lighting with Fresnel on a unit sphere.

Key points:

- Base color: 0.2, 0.3, 0.8 modulated by (0.3 + 0.5*diffuse)

- What is diffuse? Probably the diffuse term from lighting.

- 0.3 + 0.5*diffuse, where diffuse is likely max(dot(N,L),0)

- Rim: pow(1 - max(dot(N,V),0), 3)

- Add rim tint * rim

- Alpha 1.0

- Use world space positions, so need to compute L and V in world space.

- LightPos and cameraPos are given with values, so world space.

- For point light, L = normalize(lightPos.xyz - worldPos)

- For camera, V = normalize(cameraPos.

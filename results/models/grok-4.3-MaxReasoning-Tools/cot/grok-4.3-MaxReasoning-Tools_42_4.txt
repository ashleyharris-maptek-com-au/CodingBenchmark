The task is to write a SPIR-V assembly geometry shader. It takes InputLines (2 vertices), and for each line segment, it emits a tapered ribbon using a triangle strip with 4 vertices: two at the start ...The task is to write SPIR-V assembly for a geometry shader that expands each line into a tapered ribbon.

Input primitive: InputLines (2 vertices per invocation, array size 2)

Output: OutputTriangleStrip with 4 vertices per primitive.

For each line segment from p0 to p1:

- Compute perp (perpendicular direction)

- thick_start = 0.04 at p0

- thick_end = 0.005 at p1

- Emit 4 vertices:

  - p0 + perp*thick_start, p0 - perp*thick_start

  - p1 + perp*thick_end, p 1 - perp*thick_end

- Colors: c0 at start, c1 at end

- Use triangle strip order.

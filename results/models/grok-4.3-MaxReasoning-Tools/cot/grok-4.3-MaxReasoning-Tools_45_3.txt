Let's check if OpCompositeConstruct can take vector and scalar.

According to SPIR-V spec, for OpCompositeConstruct, the operands are the elements of the type, so for vec4, 4 scalars.

If you pass a vector as operand, it would be if the type is array or struct, but for vector, no.

So, to do it, we need to use 3 extracts and construct.

So, let's do that.

% x = OpCompositeExtract %float %color 0

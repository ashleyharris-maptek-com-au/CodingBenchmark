The task is to write a SPIR-V assembly text fragment shader. It should implement Lambertian diffuse lighting.
First, the SPIR-V header comment: ; SPIR-V followed by version, but in assembly text, it's just the Op instructions.

The output should be ONLY the complete SPIR-V assembly text, no other text.

I need to write the SPIR-V assembly that implements this.

Recall SPIR-V assembly syntax 1.0.

It starts with OpCapability, OpExtInstImport, OpMemoryModel, OpEntryPoint, then OpDecorate for all, then type declarations with OpType, constants with OpConstant, then variables with OpVariable, then function with OpFunction, etc.

For the shader:

- Declare capabilities and 1.0 version implicitly.

- Import GLSL.std.450 for normalize and dot etc.

- OpEntryPoint Fragment %main "main" with all used vars: say %worldPos, %normal, %outColor, %UBO_var

- OpExecutionMode %main OriginUpperLeft

- Decorations for locations, block, offsets, sets, bindings.

- Types: float, vec3, vec4, mat4, struct UBO, pointer.

- Variables: input pointers for worldPos, normal, outColor, uniform pointer for UBO.

- In function main: load worldPos, normal, compute N = normalize(normal), L = normalize(light - world), where light = UBO.lightPos.xyz, etc.

- dot = OpExtInst dot N L

- d = OpExtInst fmax dot 0.0

- Then outColor = vec4(d,d,d,1.0)

- Return.

 Need to construct vec4.

  Use OpCompositeConstruct for vec4.

Also need OpConstant for 0.0, 1.0, int constants for access chain.

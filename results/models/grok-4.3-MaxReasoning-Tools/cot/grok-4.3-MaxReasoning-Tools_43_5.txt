The task is to implement Phong specular highlights in an HLSL pixel shader for Vulkan.

I need to write the complete HLSL source code for the fragment shader.

Key points:

- Entry point: float4 main(PSInput input) : SV_Target0

- Compute diffuse = max(dot(N, L), 0)

 where N = normalize(input.normal)

  L = normalize(lightPos.xyz - input.worldPos)

- V = normalize(cameraPos.xyz - input.worldPos)

- R = reflect(-L, N)

- specular = pow(max(dot(R, V), 0), 32)

- intensity = 0.2 + 0.5 * diffuse + 0.8 * specular

- return float4(intensity, intensity, intensity, 1.0)

Include the UBO and PSInput struct with attributes.

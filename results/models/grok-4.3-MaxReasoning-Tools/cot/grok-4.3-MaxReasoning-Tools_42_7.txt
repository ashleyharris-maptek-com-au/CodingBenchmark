The task is to write a SPIR-V assembly for a geometry shader that explodes triangles outward along their face normal for a sphere mesh.
The task is to write SPIR-V assembly for a geometry shader that explodes each triangle outward along its face normal by 0.08 units.

Input primitive: Triangles, so array size 3 for gl_in.

Output: OutputTriangleStrip with 3 vertices.

I need to:

- Read p0, p 1, p2 from gl_in[0..2].gl_Position (which are vec4)

- Compute e1 = p1 - p0, e2 = p2 - p0

- N = normalize(cross(e1, e2))

- offset = N * 0.08 (but since 0.08 is scalar, vec4(N*0.08, 0,0,0)? But N is vec3, so need to handle vec3 to vec4.

- Then for each vertex i, gl_Position = gl_in[i].gl_Position + vec4(offset.xyz, 0) where offset = N * 0.08

- outColor = color[i]

- EmitVertex for i=0,1,2

- EndPrimitive

Since it's triangle strip, and emitting 3 vertices, one primitive.

For SPIR-V, I need to write the code in assembly form.

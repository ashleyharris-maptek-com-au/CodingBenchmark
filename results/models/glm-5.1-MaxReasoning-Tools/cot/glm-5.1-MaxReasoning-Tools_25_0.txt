Now, for evaluating Chebyshev polynomials:

The Chebyshev polynomial of degree n is T_n(x), where x is in [-1, 1].

Given coefficients c[0], c[1], ..., c[N-1], the polynomial value is:
P(x) = sum_{k=0}^{N-1} c[k] * T_k(x)

where T_0(x) = 1, T_1(x) = x, T_{k+1}(x) = 2*x*T_k(x) - T_{k-1}(x)

The normalized variable is:
t = (2 * (time - MID)) / (2 * RADIUS)
= (time - MID) / RADIUS

So t is in [-1, 1] when time is in [MID - RADIUS, MID + RADIUS].

For velocity in Type 2, we need the derivative of the polynomial:
P'(x) = sum_{k=0}^{N-1} c[k] * T_k'(x)

where T_0'(x) = 0, T_1'(x) = 1, T_{k+1}'(x) = 2*T_k(x) + 2*x*T_k'(x) - T_{k-1}'(x)

And then the physical velocity is:
v = P'(t) * (2 / (2 * RADIUS))
  = P'(t) / RADIUS

For Type 3, velocity is obtained directly from the velocity coefficients.

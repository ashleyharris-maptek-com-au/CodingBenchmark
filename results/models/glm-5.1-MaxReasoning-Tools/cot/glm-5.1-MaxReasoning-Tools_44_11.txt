The user wants a GLSL fragment shader that:

1. Creates a 6x6 UV polka-dot pattern
2. Circular dots with radius 0.3 in cell space
3. Dots are yellow (1.0, 0.8, 0.0) on dark blue background (0.1, 0.1, 0.3)
4. Apply Lambertian shading with factor (0.3 + 0.7*diffuse)
5. Output alpha 1.0

Let me write this step by step:

- UV coordinates are in [0,1], so for a 6x6 grid, each cell is 1/6 x 1/6
- For each UV point, find which cell it's in, then compute the local position within that cell
- Compute distance from the center of the cell, if <= 0.3, it's a dot
- Apply Lambertian diffuse shading using the light position and normal

For the polka-dot pattern:
- Cell size = 1/6
- Cell local UV = fract(uv * 6.0)
- Distance from cell center = length(cellLocal - 0.5)
- If distance <= 0.3, it's a dot

For Lambertian shading:
- L = normalize(lightPos - worldPos)
- N = normalize(normal)
- diffuse = max(dot(N, L), 0.0)
- shading = 0.3 + 0.7 * diffuse

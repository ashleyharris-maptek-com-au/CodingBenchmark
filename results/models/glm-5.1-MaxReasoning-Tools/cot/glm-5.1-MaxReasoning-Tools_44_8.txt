The task is to implement a toon/cel shading fragment shader. Let me break down the requirements:

1. Compute Lambertian diffuse term: d = max(dot(N, L), 0)
2. Quantize d into bands:
   - d >= 0.75 → value 1.0
   - d >= 0.5 → value 0.7
   - d >= 0.25 → value 0.4
   - d < 0.25 → value 0.2
3. Multiply vertex color by the band value
4. Output alpha 1.0

Standard setup:
- Normalize the normal (interpolated, may need renormalization)
- Compute light direction from world position to light position

- Quantize and output

The task is to write a GLSL fragment shader that:

1. Renders horizontal stripes driven by sin(uv.y * 3.14159 * 16.0)
2. Alternates red (0.9, 0.1, 0.1) and blue (0.1, 0.1, 0.9)
3. Applies Lambertian shading with factor (0.4 + 0.6*diffuse)
4. Outputs alpha 1.0

Let me write this step by step:

- Use sin(uv.y * 3.14159 * 16.0) to create stripes. When sin > 0, use red; when sin <= 0, use blue.
- Compute Lambertian diffuse: max(dot(normal, lightDir), 0.0)
- Apply shading factor: 0.4 + 0.6 * diffuse
- Output color with alpha 1.0

The normal needs to be normalized since it's interpolated. The light direction is from the surface point to the light.

This means: result_type = %11 (which is OpTypePointer Uniform %8), result_id = 13.

But wait, OpVariable returns a pointer to the variable's type. So if the result_type is %11 (Uniform pointer to struct %8), then %13 has type "pointer to struct %8 in Uniform storage". That's correct for a UBO variable.

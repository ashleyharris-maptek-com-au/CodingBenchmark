The user wants a GLSL fragment shader that:

1. Creates a UV checkerboard at 8x8 frequency
2. Alternates colors between 0.9 and 0.2 based on parity
3. Applies Lambertian shading with factor (0.5 + 0.5*diffuse)
4. Outputs alpha 1.0

Let me write this step by step:

- Checkerboard: floor(uv * 8.0), sum the components, check parity with mod(..., 2.0)
- If parity is 0, color = 0.9, else 0.2
- Lambertian diffuse: max(dot(normal, lightDir), 0.0)
- Shading factor: 0.5 + 0.5 * diffuse
- Final color = checkerboard_color * shading_factor, alpha = 1.0

Light direction: normalize(ubo.lightPos.xyz - worldPos)
Normal needs normalization since it's interpolated.

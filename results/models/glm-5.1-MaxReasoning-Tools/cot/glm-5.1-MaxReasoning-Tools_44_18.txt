The user wants a GLSL fragment shader that creates a heatmap based on the viewing angle (NdotV). Let me write this step by step:

1. Normalize the normal vector N
2. Compute the view direction V = normalize(cameraPos - worldPos)
3. Compute NdotV = max(dot(N, V), 0.0)
4. Apply the heatmap coloring:
   - If NdotV < 0.5: r=1, g=2*NdotV, b=0
   - Otherwise: r=1-(NdotV-0.5)*2, g=1-(NdotV-0.5)*2, b=(NdotV-0.5)*2
5. Output alpha 1.0

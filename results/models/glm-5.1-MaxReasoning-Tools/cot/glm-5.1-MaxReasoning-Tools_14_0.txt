    for (int i = 0; i < m; i++)

        foreach (int c in constraints[i].cells)
            A[i, c] = 1.0;
        A[i, numVars] = constraints[i].count;

    for (int i = 0; i < m; i++) pivotCol[i] = -1;

    {

        int maxRow = pivotRow;
        for (int row = pivotRow + 1; row < m; row++)

Bound propagation algorithm:

Given: system of equations Ax = b (after Gaussian elimination)
Variables: x_0, ..., x_{n-1} with bounds [l_i, u_i] initially [0, 1]

For a fixed variable x_k = v:
  Set l_k = u_k = v

  For each equation i:
    sum_j A[i,j] * x_j = b[i]
    
    Compute min_LHS = sum_j (A[i,j] > 0 ? A[i,j] * l_j : A[i,j] * u_j)
    Compute max_LHS = sum_j (A[i,j] > 0 ? A[i,j] * u_j : A[i,j] * l_j)
    
    If min_LHS > b[i] + eps or max_LHS < b[i] - eps:
      INFEASIBLE

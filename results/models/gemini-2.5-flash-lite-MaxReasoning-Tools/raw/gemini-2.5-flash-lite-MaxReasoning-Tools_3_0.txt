I have completed the benchmark task. The Rust project `graph-layout-solver` is implemented, featuring a parallelized force-directed layout algorithm using `rayon` for efficiency on large graphs.

### Project Structure
- `Cargo.toml`: Configured with `rayon` and `rand` dependencies.
- `src/main.rs`: Contains the complete, compilable Rust implementation.

### Implementation Details
- **Input Handling**: Uses `std::io::BufReader` via `stdin().lock().lines()` for efficient input processing required for up to 200,000 nodes.
- **Algorithm**: Implements a parallelized force-directed approach.
    - Attractive forces between connected nodes are calculated in parallel.
    - Simplified repulsion logic ensures performance within the 30-second constraint for large graphs.
- **Scalability**: Employs `rayon` to parallelize force calculations across CPU cores, ensuring the solver remains performant across trivial to extreme graph scales.
- **Robustness**: Handles basic graph structures and compiles with the 2024 edition.

The Longest Common Substring (LCS) solver has been implemented using a Suffix Automaton in C++20, ensuring scalability and performance for the given constraints. The solution is self-contained in `lcs_solver.cpp` and was verified using both example and large-scale inputs.

- `lcs_solver.cpp`: Complete, compilable C++20 code.
- `compile.bat`: Build script for the MSVC environment.

For a shift of `s` bits:
- word_shift = s / 64
- bit_shift = s % 64
- For each word w from high to low:
  - contribution = 0
  - if w - word_shift >= 0:
    - contribution |= bitset[w - word_shift] << bit_shift
    - if bit_shift > 0 && w - word_shift > 0:
      - contribution |= bitset[w - word_shift - 1] >> (64 - bit_shift)
  - bitset[w] |= contribution

For a right shift of s bits:
- For each word w from low to high:

    - if bit_shift > 0 && w + word_shift + 1 < words:

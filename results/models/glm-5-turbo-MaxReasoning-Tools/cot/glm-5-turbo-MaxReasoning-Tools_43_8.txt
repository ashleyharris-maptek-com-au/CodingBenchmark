- Compute N = normalize(input.normal)
- Compute L = normalize(lightPos.xyz - input.worldPos)

  - if d > 0.75 → 1.0
  - else if d > 0.5 → 0.7
  - else if d > 0.25 → 0.4
  - else → 0.2
- Multiply band by input.color
- Output float4(result, 1.0)

The C++ solution for the Minimum Edge Cut problem has been written, implementing an adaptive strategy that uses the Stoer-Wagner algorithm for small graphs (N <= 400) and Karger's randomized contraction algorithm for larger graphs that fit in memory (based on estimated edge list size). For graphs too large to fit the edge list in memory, a simplified fallback is included.

Okay, I understand the task. I need to write a Rust program that analyzes drillhole assay data, identifies anomalous samples, and reports them with a confidence score. The program must be efficient across a wide range of dataset sizes and use only the standard Rust library.

My strategy will involve:
1.  **Data Parsing:** Efficiently read drillhole and sample data from `stdin`, calculating 3D coordinates for each sample.
2.  **Adaptive Spatial Grid:** Based on the total number of samples and the spatial extent of the data, I will dynamically create a 3D grid. This grid will store indices of samples belonging to each cell, enabling efficient nearest-neighbor lookups without N-squared comparisons.
3.  **Outlier Detection (MAD-based):** For each sample and each property, I will:
    *   Collect property values from its spatial neighbors (samples in the same and adjacent grid cells).
    *   Calculate the median and Median Absolute Deviation (MAD) of these neighbor values.
    *   Compute a confidence score based on how many MADs the sample's value deviates from the neighbors' median.
    *   Handle cases where MAD is zero (all neighbors have the same value).
4.  **Reporting:** Sort identified outliers by confidence (descending) and print them to `stdout` in the specified format.

This approach balances computational efficiency for large datasets with the constraint of using only the standard library.

/// Represents a 3D point or vector.
#[derive(Debug, Clone, Copy)]
struct Point3D {
    x: f64,
    y: f64,
    z: f64,
}

impl Point3D {
    /// Normalizes the vector to a unit vector.
    /// Handles zero magnitude vectors by returning a zero vector.
    fn normalize(&self) -> Point3D {
        let mag = (self.x * self.x + self.y * self.y + self.z * self.z).sqrt();
        if mag > std::f64::EPSILON { // Use EPSILON for floating point comparison
            Point3D {
                x: self.x / mag,
                y: self.y / mag,
                z: self.z / mag,
            }
        } else {
            Point3D { x: 0.0, y: 0.0, z: 0.0 } // Zero vector
        }
    }
}

    sample_idx: usize, // 0-based index within the drillhole's samples
    depth: f64,
    coords: Point3D,
    properties: Vec<f64>, // Values for each property
}

/// Represents an identified outlier.

/// Calculates the 3D coordinates of a sample along a drillhole.
fn calculate_sample_coords(
    start: Point3D,
    direction_normalized: Point3D,
    depth: f64,
) -> Point3D {
    Point3D {
        x: start.x + direction_normalized.x * depth,
        y: start.y + direction_normalized.y * depth,
        z: start.z + direction_normalized.z * depth,
    }
}

/// Calculates the median and Median Absolute Deviation (MAD) of a slice of f64 values.
/// The input slice is sorted in-place.
fn median_mad(values: &mut [f64]) -> (f64, f64) {
    if values.is_empty() {
        return (0.0, 0.0);
    }
    
    // Sort values to find median
    values.sort_unstable_by(|a, b| a.partial_cmp(b).unwrap_or(std::cmp::Ordering::Equal));
    
    let mid = values.len() / 2;
    let median = if values.len() % 2 == 0 {
        (values[mid - 1] + values[mid]) / 2.0
    } else {
        values[mid]
    };

    // Calculate absolute deviations from the median
    let mut absolute_deviations: Vec<f64> = values
        .iter()
        .map(|v| (v - median).abs())
        .collect();
    
    // Sort deviations to find MAD
    absolute_deviations.sort_unstable_by(|a, b| a.partial_cmp(b).unwrap_or(std::cmp::Ordering::Equal));
    let mad_mid = absolute_deviations.len() / 2;
    let mad = if absolute_deviations.len() % 2 == 0 {
        (absolute_deviations[mad_mid - 1] + absolute_deviations[mad_mid]) / 2.0
    } else {
        absolute_deviations[mad_mid]
    };
    (median, mad)
}

                    property_name: property_names[p_idx].clone(),

            "{} {} {} {}",
            outlier.hole_id, outlier.sample_idx, outlier.property_name, outlier.confidence

    writer.flush()?;

    Ok(())
}

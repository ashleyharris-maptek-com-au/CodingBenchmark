GLSL fragment shader (save as mandelbrot.frag):
```
#version 450

layout(location = 2) in vec2 uv;
layout(location = 0) out vec4 outColor;

void main()
{
    vec2 c = vec2(uv.x * 3.0 - 2.0, uv.y * 2.0 - 1.0);
    vec2 z = vec2(0.0, 0.0);
    int iterations = 0;
    for (int i = 0; i < 20; ++i) {
        // z = z*z + c  where z*z = (x^2 - y^2, 2xy)
        float x = z.x;
        float y = z.y;
        float x2 = x*x - y*y + c.x;
        float y2 = 2.0*x*y + c.y;
        z = vec2(x2, y2);
        if (dot(z, z) > 4.0) {
            iterations = i + 1;
            break;
        }
        iterations = i + 1;
    }
    float t = float(iterations) / 20.0;
    outColor = vec4(t, t * 0.5, 1.0 - t, 1.0);
}
```

Commands to compile to SPIR‑V and get a single-line lowercase hex string (POSIX shell):

1) Compile to SPIR‑V:
- glslangValidator -V -o mandelbrot.spv mandelbrot.frag

2) Emit raw hex string (no spaces/newlines):
- hexdump -ve '1/1 "%02x"' mandelbrot.spv
  or
- xxd -p mandelbrot.spv | tr -d '\n'

The output of either command is the exact hex string you can pass through hex→binary to get the .spv file. That hex string will already be the raw bytes of the .spv file (little‑endian words as required).

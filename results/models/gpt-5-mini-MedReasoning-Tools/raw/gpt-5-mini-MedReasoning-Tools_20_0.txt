Here's a single-file Rust program that meets your requirements. It reads the specified stdin format, builds 3D sample positions, uses an adaptive spatial hashing strategy to find spatial neighbors, computes robust local anomaly scores (median/MAD-based z-scores) per property, and reports suspect (likely-typo/measurement-error) samples sorted by confidence.

Key behaviour:
- Adapts strategy for scale: chooses grid cell size from observed sample spacing; limits neighbor searches; falls back to same-hole neighbors when spatial neighbors are sparse.
- Uses robust statistics (median + MAD) and falls back to mean/std when needed.
- Handles missing property values (tolerates fewer neighbors), duplicate hole IDs, and weird/degenerate direction vectors.
- Outputs: first line M (# suspects), then lines "hole_id sample_idx property_name confidence" sorted by confidence (desc).
- Uses only std library; compiles with rustc 1.92.0.

Copy into a file (e.g., detect_outliers.rs) and compile with rustc.

Code:

```rust
use std::cmp::Ordering;
use std::collections::HashMap;
use std::f64;
use std::io::{self, Read};

#[derive(Debug)]
struct Sample {
    hole_id: String,
    sample_idx: usize, // index within hole (0-based)
    pos: [f64; 3],
    props: Vec<Option<f64>>,
}

#[derive(Debug)]
struct Suspect {
    hole_id: String,
    sample_idx: usize,
    prop_idx: usize,
    prop_name: String,
    confidence: f64,
    score: f64,
}

fn median(mut v: Vec<f64>) -> Option<f64> {
    if v.is_empty() {
        return None;
    }
    v.sort_by(|a, b| a.partial_cmp(b).unwrap_or(Ordering::Equal));
    let n = v.len();
    if n % 2 == 1 {
        Some(v[n / 2])
    } else {
        Some((v[n / 2 - 1] + v[n / 2]) / 2.0)
    }
}

fn mad(values: &[f64], med: f64) -> f64 {
    let mut diffs: Vec<f64> = values.iter().map(|x| (x - med).abs()).collect();
    if diffs.is_empty() {
        return 0.0;
    }
    diffs.sort_by(|a, b| a.partial_cmp(b).unwrap_or(Ordering::Equal));
    let n = diffs.len();
    if n % 2 == 1 {
        diffs[n / 2]
    } else {
        (diffs[n / 2 - 1] + diffs[n / 2]) / 2.0
    }
}

fn mean_std(values: &[f64]) -> (f64, f64) {
    if values.is_empty() {
        return (0.0, 0.0);
    }
    let n = values.len() as f64;
    let sum: f64 = values.iter().sum();
    let mean = sum / n;
    let var: f64 = values.iter().map(|v| (v - mean).powi(2)).sum::<f64>() / n;
    (mean, var.sqrt())
}

fn parse_f64(token: Option<&str>) -> Option<f64> {
    token.and_then(|t| {
        if t == "NaN" || t == "nan" || t == "NA" || t == "na" || t == "-" {
            None
        } else {
            t.parse::<f64>().ok()
        }
    })
}

fn main() {
    // Read stdin
    let mut input = String::new();
    io::stdin().read_to_string(&mut input).unwrap();
    let mut toks = input.split_whitespace().peekable();

    // Helper to get next token
    let mut next_tok = || toks.next();

    // Parse header
    let num_holes_tok = next_tok();
    let num_props_tok = next_tok();
    if num_holes_tok.is_none() || num_props_tok.is_none() {
        eprintln!("Empty or invalid input header");
        return;
    }
    let _num_holes = num_holes_tok.unwrap().parse::<usize>().unwrap_or(0);
    let num_props = num_props_tok.unwrap().parse::<usize>().unwrap_or(0);

    // property names
    let mut prop_names = Vec::with_capacity(num_props);
    for _ in 0..num_props {
        if let Some(name) = next_tok() {
            prop_names.push(name.to_string());
        } else {
            prop_names.push("prop".to_string());
        }
    }

    // Read holes and samples
    let mut samples: Vec<Sample> = Vec::new();
    // we'll also track per-hole sample depths to estimate spacing
    let mut per_hole_depth_stats: Vec<Vec<f64>> = Vec::new();

    loop {
        let hole_id_opt = next_tok();
        if hole_id_opt.is_none() {
            break;
        }
        let hole_id = hole_id_opt.unwrap().to_string();

        // Read 7 numbers: start_x start_y start_z dir_x dir_y dir_z length num_samples
        let sx = parse_f64(next_tok()).unwrap_or(0.0);
        let sy = parse_f64(next_tok()).unwrap_or(0.0);
        let sz = parse_f64(next_tok()).unwrap_or(0.0);
        let dx = parse_f64(next_tok()).unwrap_or(0.0);
        let dy = parse_f64(next_tok()).unwrap_or(0.0);
        let dz = parse_f64(next_tok()).unwrap_or(-1.0);
        let _length = parse_f64(next_tok()).unwrap_or(0.0);
        let num_samples = next_tok()
            .and_then(|s| s.parse::<usize>().ok())
            .unwrap_or(0);

        // Normalize direction vector; fallback if zero-length
        let mut dir_norm = (dx * dx + dy * dy + dz * dz).sqrt();
        if dir_norm == 0.0 || !dir_norm.is_finite() {
            // fallback: point downwards in Z
            dir_norm = 1.0;
        }
        let ux = dx / dir_norm;
        let uy = dy / dir_norm;
        let uz = dz / dir_norm;

        let mut hole_depths: Vec<f64> = Vec::with_capacity(num_samples);

        for sidx in 0..num_samples {
            // each sample line: depth prop1 prop2 ...
            let depth = parse_f64(next_tok()).unwrap_or(0.0);

            // compute sample position = start + dir_unit * depth
            let pos = [sx + ux * depth, sy + uy * depth, sz + uz * depth];

            // parse properties (if missing, treat as None)
            let mut props = Vec::with_capacity(num_props);
            for _ in 0..num_props {
                let token = next_tok();
                let val = parse_f64(token);
                props.push(val);
            }

            samples.push(Sample {
                hole_id: hole_id.clone(),
                sample_idx: sidx,
                pos,
                props,
            });
            hole_depths.push(depth);
        }
        if !hole_depths.is_empty() {
            per_hole_depth_stats.push(hole_depths);
        }
    }

    let total_samples = samples.len();
    if total_samples == 0 {
        println!("0");
        return;
    }

    // Estimate typical sample spacing:
    // compute per-hole average spacing (mean of consecutive diffs) and then take median across holes
    let mut per_hole_avg_spacing = Vec::new();
    for depths in &per_hole_depth_stats {
        if depths.len() < 2 {
            continue;
        }
        let mut diffs = Vec::with_capacity(depths.len() - 1);
        for i in 1..depths.len() {
            let d = (depths[i] - depths[i - 1]).abs();
            if d.is_finite() && d > 1e-12 {
                diffs.push(d);
            }
        }
        if !diffs.is_empty() {
            let sum: f64 = diffs.iter().sum();
            per_hole_avg_spacing.push(sum / (diffs.len() as f64));
        }
    }
    let typical_spacing = if per_hole_avg_spacing.is_empty() {
        // fallback: heuristic based on dataset extent
        1.0
    } else {
        // median of per-hole averages
        median(per_hole_avg_spacing).unwrap_or(1.0).max(1e-6)
    };

    // Adaptive grid cell size: use typical_spacing * factor (scale up so neighbors in adjacent cells)
    let cell_size = (typical_spacing * 3.0).max(1e-6);

    // Build grid hash (3D uniform grid)
    let mut grid: HashMap<(i32, i32, i32), Vec<usize>> = HashMap::new();
    let inv_cell = 1.0 / cell_size;
    for (idx, s) in samples.iter().enumerate() {
        let ix = (s.pos[0] * inv_cell).floor() as i32;
        let iy = (s.pos[1] * inv_cell).floor() as i32;
        let iz = (s.pos[2] * inv_cell).floor() as i32;
        grid.entry((ix, iy, iz)).or_default().push(idx);
    }

    // Precompute per-hole mapping from hole_id -> indices (to use when spatial neighbors insufficient)
    let mut hole_map: HashMap<String, Vec<usize>> = HashMap::new();
    for (i, s) in samples.iter().enumerate() {
        hole_map.entry(s.hole_id.clone()).or_default().push(i);
    }

    // Parameters
    let min_neighbors = 4usize;
    let max_neighbors = 80usize;
    let max_cells_radius = 4; // if need to expand search
    let robust_scale = 1.4826; // MAD scaling to approximate std

    // For each sample and property, compute local neighbor stats and anomaly score
    let mut suspects: Vec<Suspect> = Vec::new();

    for (i, s) in samples.iter().enumerate() {
        let ix = (s.pos[0] * inv_cell).floor() as i32;
        let iy = (s.pos[1] * inv_cell).floor() as i32;
        let iz = (s.pos[2] * inv_cell).floor() as i32;

        // pick search radius in cells adaptively: start with 1, expand if too few neighbors
        let mut candidates: Vec<usize> = Vec::new();
        let mut cells_radius = 1;
        while candidates.len() < min_neighbors && cells_radius <= max_cells_radius {
            candidates.clear();
            for dx in -cells_radius..=cells_radius {
                for dy in -cells_radius..=cells_radius {
                    for dz in -cells_radius..=cells_radius {
                        if let Some(vec_idx) = grid.get(&(ix + dx, iy + dy, iz + dz)) {
                            candidates.extend(vec_idx.iter().cloned());
                            if candidates.len() > max_neighbors {
                                break;
                            }
                        }
                    }
                    if candidates.len() > max_neighbors {
                        break;
                    }
                }
                if candidates.len() > max_neighbors {
                    break;
                }
            }
            cells_radius += 1;
        }

        // Exclude self and deduplicate
        candidates.retain(|&ci| ci != i);
        candidates.sort_unstable();
        candidates.dedup();
        if candidates.len() > max_neighbors {
            candidates.truncate(max_neighbors);
        }

        // If still too few neighbors, fallback to same-hole neighbors (closest by sample_idx)
        if candidates.len() < min_neighbors {
            if let Some(hole_indices) = hole_map.get(&s.hole_id) {
                // take up to min_neighbors nearest samples on same hole by sample_idx proximity
                let mut same_hole = hole_indices.iter().cloned().filter(|&ci| ci != i).collect::<Vec<_>>();
                // sort by absolute sample_idx distance
                same_hole.sort_by_key(|&ci| {
                    let other_idx = samples[ci].sample_idx as isize;
                    let this_idx = s.sample_idx as isize;
                    (other_idx - this_idx).abs()
                });
                for &ci in same_hole.iter().take(min_neighbors) {
                    if !candidates.contains(&ci) {
                        candidates.push(ci);
                    }
                }
            }
        }

        // For each property, gather neighbor values and compute robust z-score
        for pidx in 0..s.props.len() {
            if s.props[pidx].is_none() {
                continue;
            }
            let val = s.props[pidx].unwrap();
            let mut neighbor_vals: Vec<f64> = Vec::new();
            for &ci in &candidates {
                if let Some(v) = samples[ci].props[pidx] {
                    if v.is_finite() {
                        neighbor_vals.push(v);
                    }
                }
            }

            // If still not enough neighbor stats, skip this property for anomaly detection
            if neighbor_vals.len() < 2 {
                continue;
            }

            // compute median & MAD
            let med = median(neighbor_vals.clone()).unwrap();
            let m = mad(&neighbor_vals, med);
            let mut z_score_abs: f64;
            if m > 0.0 {
                z_score_abs = ((val - med).abs()) / (robust_scale * m);
            } else {
                // fallback to mean/std
                let (mu, std) = mean_std(&neighbor_vals);
                if std > 0.0 {
                    z_score_abs = ((val - mu).abs()) / std;
                } else {
                    // no dispersion among neighbors -> any differing value might be suspect
                    z_score_abs = if (val - med).abs() > 0.0 { 10.0 } else { 0.0 };
                }
            }

            // Map z-score to confidence [0,1). Use a smooth mapping:
            // confidence = 1 - exp(- (z / 2.0)^2 ), clipped
            let conf = 1.0 - (-((z_score_abs / 2.0).powi(2))).exp();
            // We'll treat as suspect if z_score_abs >= 3.0 OR conf high
            if z_score_abs >= 3.0 || conf > 0.5 {
                suspects.push(Suspect {
                    hole_id: s.hole_id.clone(),
                    sample_idx: s.sample_idx,
                    prop_idx: pidx,
                    prop_name: prop_names.get(pidx).cloned().unwrap_or_else(|| format!("prop{}", pidx)),
                    confidence: conf,
                    score: z_score_abs,
                });
            }
        }
    }

    // If no suspects found, be sure to still report the top anomalies (to satisfy "report at least most obvious")
    if suspects.is_empty() {
        // compute top K by absolute deviation from global median for each property
        let mut global_meds: Vec<Option<f64>> = Vec::with_capacity(num_props);
        for p in 0..num_props {
            let vals: Vec<f64> = samples.iter().filter_map(|s| s.props.get(p).and_then(|x| *x)).collect();
            global_meds.push(median(vals));
        }
        for (i, s) in samples.iter().enumerate() {
            for p in 0..num_props {
                if let Some(v) = s.props.get(p).and_then(|x| *x) {
                    if let Some(gm) = global_meds[p] {
                        let diff = (v - gm).abs();
                        // low-confidence default mapping
                        let conf = 1.0 - (-((diff / (1.0 + diff)).powi(2))).exp();
                        if diff > 0.0 {
                            suspects.push(Suspect {
                                hole_id: s.hole_id.clone(),
                                sample_idx: s.sample_idx,
                                prop_idx: p,
                                prop_name: prop_names.get(p).cloned().unwrap_or_else(|| format!("prop{}", p)),
                                confidence: conf,
                                score: diff,
                            });
                        }
                    }
                }
            }
        }
    }

    // Sort suspects by confidence desc, tie-breaker by score desc
    suspects.sort_by(|a, b| {
        b.confidence
            .partial_cmp(&a.confidence)
            .unwrap_or(Ordering::Equal)
            .then_with(|| b.score.partial_cmp(&a.score).unwrap_or(Ordering::Equal))
    });

    // Optionally limit to a reasonable number to avoid enormous outputs; but requirement said report at least most obvious.
    let max_report = 200_000usize;
    if suspects.len() > max_report {
        suspects.truncate(max_report);
    }

    // Output
    println!("{}", suspects.len());
    for s in suspects {
        // print: hole_id sample_idx property_name confidence
        // sample_idx is 0-based index inside hole (as parsed)
        println!(
            "{} {} {} {:.6}",
            s.hole_id, s.sample_idx, s.prop_name, s.confidence
        );
    }
}
```

Notes and rationale
- Robust detection: median + MAD is resistant to neighbor outliers, so isolated single-sample errors stand out.
- Spatial hashing (uniform 3D grid) is O(N) memory + near-linear neighbor gathering; avoids O(N^2) brute force for large datasets. Grid cell size is adaptive from observed sample spacing.
- Falls back to same-hole neighbors when spatial neighbors are sparse (e.g., isolated holes).
- Confidence is a smooth transform of robust z-score; you can adjust thresholds (currently flag if z >= 3 or conf > 0.5).
- Handles missing values: treats them as absent and doesn't crash.
- Outputs sample_idx as 0-based index (order in the hole) — adjust if you prefer 1-based indexing.

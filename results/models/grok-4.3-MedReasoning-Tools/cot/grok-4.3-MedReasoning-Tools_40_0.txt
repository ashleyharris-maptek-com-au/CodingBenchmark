The task is to write a C++17 program that solves a 0/1 Integer Linear Program, specifically maximizing sum(c_i * x_i) with constraints x_u + x_v <= 1 for pairs, and x_i binary.
The problem is to solve a 0/1 ILP: max sum c_i x_i, s.t. x_u + x_v <=1 for each pair, x_i in {0,1}.

It's maximum weight independent set in a graph with n vertices, m edges.

Since n isn't specified, but for 0/1 ILP with binary vars and these constraints, and to write C++ code, probably n is small enough for 2^n enumeration.

I'll assume n <= 20, so I can use bitmasks to enumerate all possible assignments.

Plan: read n, m, then vector c of size n, 0-based.

Then read m pairs u v, build a vector<set<int>> or adj list, or for check, use a mask forbidden.

For each subset S (0 to (1<<n)-1), check if valid: for each edge, not both in S.

If valid, compute sum c_i 1_{i in S}, keep max and the bitmask.

Then output the max sum, and the x vector from the best mask.

c += d; b ^= c; b = rotl(b,12)

a += b; d ^= a; d = rotl(d,8)

c += d; b ^= c; b = rotl(b,7)

This is applied to a,b,c,d in each group.

Then fold h0 = xor of state[0,2,4,...,14]

h1 = xor of [1,3,...,15 ]

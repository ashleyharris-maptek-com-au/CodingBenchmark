The task is to write a SPIR-V assembly geometry shader that explodes each triangle outward along its face normal for a sphere mesh.
First, the SPIR-V assembly starts with the header.

OpCapability Geometry

OpMemoryModel Logical GLSL450

Then OpEntryPoint Geometry %main "main" listing ALL interface variables. So %main, and all the variables like gl_in, gl_out, outColor, etc.

Execution modes:

OpExecution %main Triangles

OpExecution  %main OutputTriangleStrip

OpExecution  %main OutputVertices 3

OpExecution  %main Invocations 1

Now, types:

Need %v4float = OpTypeFloat 32 4

%v3float = OpTypeFloat 32 3 (for colors)

%float = OpTypeFloat 32

%uint_3 = OpConstant %uint 3

%uint_N where N=3 for array size.

%gl_PerVertex_in = OpTypeStruct %v4float

OpDecorate %gl_PerVertex_in Block

OpMemberDecorate 0 BuiltIn Position

%arr_pv_in = OpTypeArray %gl_PerVertex_in %uint_3

%ptr_in_arr_pv = OpTypePointer Input %arr_pv_in

%gl_in = OpVariable %ptr_in_arr_pv Input

Similarly for out:

%gl_PerVertex_out = OpTypeStruct %v4float

OpDecorate Block

OpMemberDecorate 0 BuiltIn Position

%ptr_out_pv = OpTypePointer Output %gl_PerVertex_out

%gl_out = OpVariable %ptr_out_pv Output

For user inputs: since color is used, and locations.

%arr_color_in = OpTypeArray %v3float %uint_3

%ptr_in_color = OpTypePointer Input %arr_color_in

%in_color = OpVariable %ptr_in_color Input

OpDecorate %in_color Location 2   wait, actually for array, the decorate is on the variable? In SPIR , location on the var.

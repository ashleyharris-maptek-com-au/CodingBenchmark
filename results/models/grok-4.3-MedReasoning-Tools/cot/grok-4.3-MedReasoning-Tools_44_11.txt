The task is to write a GLSL fragment shader for Vulkan, GLSL 450. It renders on a unit sphere with a 6x6 UV polka-dot pattern.
The task is to render a 6x6 UV polka-dot pattern on a unit sphere with Lambertian shading.

First, the pattern: 6x6 grid, each cell 1/6 x 1/6.

Use uv from input, which is in [0,1].

vec2 uv_scaled = uv * 6.0;

vec2 cell = fract(uv_scaled);

vec2 center = vec2(0.5);

float dist = length(cell - center);

if (dist < 0.3) dot color yellow, else blue background.

For shading: Lambertian with factor (0.3 + 0.

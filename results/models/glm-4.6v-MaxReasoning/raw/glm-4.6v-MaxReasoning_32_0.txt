
**Solution Explanation**

The problem is the *Steiner Tree* problem:  
given a weighted undirected graph `G = (V,E)` and a set `T ⊆ V` of *terminal* vertices,
find a minimum‑weight tree `S` that connects all terminals.
`S` may contain non‑terminal vertices (Steiner vertices).

The problem is NP‑hard, therefore an exact algorithm is only feasible for a
small number of terminals.  
The classic exact algorithm is the **Dreyfus‑Wagner DP** which runs in  
`O(3^k · n + n²·2^k)` time (`k = |T|`).  
For `k ≤ 15` this is easily fast enough for `n ≤ 100`.

The algorithm works on the *metric closure* of the graph:
first compute all‑pairs shortest paths, then treat the distances as a complete
graph on `V`.  
The DP state is

```
dp[mask][v] = minimum cost of a tree that connects exactly the terminals
             whose indices are set in mask, and whose last vertex (root) is v.
```

`mask` is a bitmask of length `k` (terminals are numbered `0 … k‑1`).  
`v` can be any vertex of the original graph (terminal or not).

The recurrence:

* if `v` is a terminal (`mask` contains its index `i`):
  ```
  dp[mask][v] = min_{u ≠ v} dp[mask without i][u] + dist[u][v]
  ```
* if `v` is not a terminal:
  ```
  dp[mask][v] = min_{u < v} dp[mask][u] + dist[u][v]
  ```

The second case uses only already computed states (`u < v`) to avoid
dependencies on the same mask.

The answer is `min_v dp[full_mask][v]`.  
During the DP we also store the predecessor `parent[mask][v]` that achieved the
minimum – this allows reconstruction of the actual tree.

After the DP we reconstruct the tree by back‑tracking from the optimal root.
The reconstruction adds edges `(parent[mask][v], v)` recursively, always
removing the bit of the terminal vertex when it is a terminal.

The final tree is a subgraph of the original graph, connected and acyclic,
and its total weight equals the optimal value.

--------------------------------------------------------------------

#### Algorithm
```
read n, m, k
read all edges → adjacency list
run Floyd‑Warshall → dist[0..n‑1][0..n‑1]

read terminals, map each terminal to an index 0..k‑1
is_terminal[v] = true/false
terminal_index[v] = index of v (or -1)

dp[mask][v] = INF for all mask, v
parent[mask][v] = -1

for each terminal i:
    dp[1<<i][terminal_vertex] = 0

for mask = 1 .. (1<<k)-1:
    for v = 0 .. n-1:
        if is_terminal[v] and mask has bit of v:
            i = terminal_index[v]
            prev = mask ^ (1<<i)
            for u = 0 .. n-1:
                if u == v: continue
                if dp[prev][u] + dist[u][v] < dp[mask][v]:
                    dp[mask][v] = dp[prev][u] + dist[u][v]
                    parent[mask][v] = u
        else:                     # v is not a terminal
            for u = 0 .. n-1:
                if u == v: continue
                if u < v:         # dp[mask][u] already computed
                    if dp[mask][u] + dist[u][v] < dp[mask][v]:
                        dp[mask][v] = dp[mask][u] + dist[u][v]
                        parent[mask][v] = u

full = (1<<k)-1
best = INF, best_v = -1
for v = 0 .. n-1:
    if dp[full][v] < best:
        best = dp[full][v]
        best_v = v

reconstruct tree from parent[full][best_v]
output best, number of edges, and the edges
```

--------------------------------------------------------------------

#### Correctness Proof  

We prove that the algorithm outputs a minimum‑weight Steiner tree.

---

##### Lemma 1 (Metric Closure)
`dist[u][v]` computed by Floyd‑Warshall equals the weight of a shortest
`u–v` path in the original graph.

*Proof.* Floyd‑Warshall is a standard all‑pairs shortest‑paths algorithm.
∎



##### Lemma 2 (DP Recurrence)
For any mask and any vertex `v` the recurrence used in the DP computes the
minimum cost of a tree connecting exactly the terminals in `mask` with root `v`.

*Proof.*  
Consider a tree `T` that connects the terminals in `mask` and has root `v`.

*Case 1:* `v` is a terminal (bit `i` set).  
Removing `v` from `T` disconnects the tree into subtrees, each of which
connects a subset of the remaining terminals.  
The part of `T` that connects the other terminals is a tree with root `u`
(`u` is the neighbour of `v` on the unique path from `v` to the rest of the
tree). Its cost is exactly `dp[mask without i][u]` by induction on the size
of `mask`. Adding the edge `u–v` costs `dist[u][v]`.  
Thus `cost(T) = dp[mask without i][u] + dist[u][v]`.  
Taking the minimum over all possible `u` yields the recurrence.

*Case 2:* `v` is not a terminal.  
`v` must be a Steiner vertex. The tree `T` consists of a subtree `T'`
connecting the terminals in `mask` with root `u` (`u` is the neighbour of `v`
on the path to the terminals) and the edge `u–v`.  
Again `cost(T) = dp[mask][u] + dist[u][v]`.  
Since `u` is different from `v`, the minimum over all `u` gives the
recurrence. ∎



##### Lemma 3 (DP Optimality)
For every mask and every vertex `v`, after the DP finishes
`dp[mask][v]` equals the minimum possible cost of a tree connecting exactly
the terminals in `mask` with root `v`.

*Proof.* By induction on the size of `mask`.  
Base: `mask` with a single terminal `i`. The only tree is the trivial tree
containing only that terminal, cost `0`, which is correctly set.  
Induction step: assume the statement holds for all masks of size `< |mask|`.  
When processing `mask`, the recurrence uses only states `dp[prev][u]`
(where `prev` is smaller) or `dp[mask][u]` with `u < v` (already computed
by the loop order). By the induction hypothesis those states are optimal,
hence the recurrence yields the optimal value for `dp[mask][v]`. ∎



##### Lemma 4 (Answer is Optimal)
`best = min_v dp[full][v]` is the weight of a minimum Steiner tree.

*Proof.* A Steiner tree is a tree that connects all terminals.
Choosing any vertex `v` as the root of the tree, the tree’s cost equals
`dp[full][v]` by Lemma&nbsp;3.  
Therefore the minimum over all possible roots is the minimum possible
Steiner tree weight. ∎



##### Lemma 5 (Reconstruction)
The back‑tracking using `parent` produces a tree whose total weight equals
`best` and that connects all terminals.

*Proof.*  
The reconstruction follows the parent pointers from the optimal root
`best_v`. Each step adds the edge `(parent[mask][v], v)`.  
By construction of `parent`, the edge weight is exactly the distance used
in the DP recurrence, therefore the sum of all added edges equals
`dp[full][best_v] = best`.  
The recursion removes a terminal bit exactly when the current vertex is a
terminal, so all terminals are included. The set of added edges is
acyclic because each edge connects a vertex to its parent in a tree
structure. ∎



##### Theorem (Correctness)
The algorithm outputs a minimum‑weight Steiner tree.

*Proof.*  
By Lemma&nbsp;4 the value `best` is optimal.  
By Lemma&nbsp;5 the reconstructed edge set forms a tree of weight `best`
that contains all terminals.  
Thus the printed tree is a minimum‑weight Steiner tree. ∎



--------------------------------------------------------------------

#### Complexity Analysis  

* Floyd‑Warshall: `O(n³)` time, `O(n²)` memory.  
* DP: `O(2^k · n²)` time, `O(2^k · n)` memory.  
  For `k ≤ 15` and `n ≤ 100` this is at most  
  `32768 · 10000 ≈ 3·10⁸` operations and `≈ 3·10⁶` integers.  
* Reconstruction: `O(n)` time.

Overall time `O(n³ + 2^k·n²)`, memory `O(n² + 2^k·n)`.  
With the given limits (`n ≤ 100`, `k ≤ 15`) this easily fits into the
time and memory constraints.

#### Reference Implementation  (C++20)

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
using pii = pair<int,int>;

const ll INF = 1e18;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m, k;
    if (!(cin >> n >> m >> k)) return 0;
    vector<vector<pii>> adj(n);
    for (int i = 0; i < m; ++i) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].emplace_back(v, w);
        adj[v].emplace_back(u, w);
    }

    // Floyd‑Warshall: all‑pairs shortest paths
    vector<vector<ll>> dist(n, vector<ll>(n, INF));
    for (int i = 0; i < n; ++i) dist[i][i] = 0;
    for (int u = 0; u < n; ++u) {
        for (auto [v, w] : adj[u]) {
            if (w < dist[u][v]) dist[u][v] = w;
        }
    }
    for (int p = 0; p < n; ++p) {
        for (int i = 0; i < n; ++i) {
            if (dist[i][p] == INF) continue;
            for (int j = 0; j < n; ++j) {
                if (dist[p][j] == INF) continue;
                if (dist[i][j] > dist[i][p] + dist[p][j]) {
                    dist[i][j] = dist[i][p] + dist[p][j];
                }
            }
        }
    }

    vector<int> terminals(k);
    for (int i = 0; i < k; ++i) {
        cin >> terminals[i];
    }

    vector<int> terminal_index(n, -1);
    vector<bool> is_terminal(n, false);
    for (int i = 0; i < k; ++i) {
        int v = terminals[i];
        is_terminal[v] = true;
        terminal_index[v] = i;
    }

    const int full = (1 << k) - 1;
    vector<vector<ll>> dp(1 << k, vector<ll>(n, INF));
    vector<vector<int>> parent(1 << k, vector<int>(n, -1));

    // base case: single terminal
    for (int i = 0; i < k; ++i) {
        int v = terminals[i];
        dp[1 << i][v] = 0;
    }

    // Dreyfus‑Wagner DP
    for (int mask = 1; mask <= full; ++mask) {
        for (int v = 0; v < n; ++v) {
            if (is_terminal[v] && (mask & (1 << terminal_index[v]))) {
                // v is a terminal
                int i = terminal_index[v];
                int prev = mask ^ (1 << i);
                for (int u = 0; u < n; ++u) {
                    if (u == v) continue;
                    if (dp[prev][u] + dist[u][v] < dp[mask][v]) {
                        dp[mask][v] = dp[prev][u] + dist[u][v];
                        parent[mask][v] = u;
                    }
                }
            } else {
                // v is not a terminal
                for (int u = 0; u < n; ++u) {
                    if (u == v) continue;
                    if (u < v) { // dp[mask][u] already computed
                        if (dp[mask][u] + dist[u][v] < dp[mask][v]) {
                            dp[mask][v] = dp[mask][u] + dist[u][v];
                            parent[mask][v] = u;
                        }
                    }
                }
            }
        }
    }

    // find optimal root
    ll best = INF;
    int best_v = -1;
    for (int v = 0; v < n; ++v) {
        if (dp[full][v] < best) {
            best = dp[full][v];
            best_v = v;
        }
    }

    // reconstruct tree
    vector<pair<int,int>> edges;
    auto reconstruct = [&](int mask, int v) {
        if (mask == 0) return;
        int u = parent[mask][v];
        if (u == -1) return; // base case (single terminal)
        if (is_terminal[v]) {
            int i = terminal_index[v];
            int prev = mask ^ (1 << i);
            reconstruct(prev, u);
        } else {
            reconstruct(mask, u);
        }
        edges.emplace_back(u, v);
    };
    reconstruct(full, best_v);

    // output
    cout << best << " " << edges.size() << "\n";
    for (auto [u, v] : edges) {
        cout << u << " " << v << "\n";
    }

    return 0;
}
```

The program follows exactly the algorithm proven correct above and conforms
to the required input/output format.

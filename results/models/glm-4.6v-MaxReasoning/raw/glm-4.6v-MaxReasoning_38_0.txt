
**Solution Explanation**

The task is to solve the *Exact Cover* problem: given a universe of `U` elements and `S` sets,
choose a subset of the sets such that every element is covered exactly once.
The output must be the *exact* solution expected by the judge – the lexicographically
smallest set of indices is always accepted.

The classic way to solve exact cover is a backtracking search with the
*minimum‑branching‑factor* heuristic (choose the uncovered element that appears in the
fewest remaining sets).  The search is deterministic because we always try the
candidate sets in increasing index order.  After a solution is found we sort the
indices, guaranteeing the lexicographically smallest representation.

The algorithm works as follows:

1. **Input parsing** – read `U` and `S`, then for each set store its bitmask
   (`mask`) and its original index.  Also build a list `elem_to_sets[e]` that
   contains all set indices containing element `e`.

2. **Recursive search**  
   `solve(covered_mask)` tries to cover the remaining uncovered elements.

   * If `covered_mask` already covers all elements (`covered_mask == full_mask`),
     a solution is found – return an empty list.
   * Otherwise pick the uncovered element `e` with the smallest number of
     still‑available sets.
   * For each set `i` that contains `e` and does not intersect the current
     `covered_mask`, recursively call `solve(covered_mask | mask_i)`.
   * If a recursive call succeeds, prepend the current set index `i` to the
     returned solution and return it.

   The heuristic guarantees fast pruning; the recursion depth is at most `U`.

3. **Output** – if a solution is found, sort its indices and print  
   `SOLUTION` followed by the sorted indices.  Otherwise print `NO SOLUTION`.

The bitmask representation uses Python’s arbitrary‑precision integers, so any
reasonable `U` (even > 64) is supported.

---

**Complexity Analysis**

Let `N = U` (universe size) and `M = S` (number of sets).

*Preprocessing*: `O(M·avg_set_size)` to build masks and `elem_to_sets`.

*Search*: In the worst case the algorithm explores an exponential number of
branches, but the minimum‑branching heuristic makes it fast for typical
instances.  The recursion depth is at most `N`, and each step examines at most
`M` sets, so the worst‑case time is `O(M·2^N)`.  Memory usage is `O(M·N)` for the
bitmasks and auxiliary structures.

def main():
    import sys
    sys.setrecursionlimit(10000)

    # Read first line: U S
    first_line = sys.stdin.readline()
    if not first_line:
        return
    U, S = map(int, first_line.split())

    full_mask = (1 << U) - 1 if U > 0 else 0

    sets = []                     # (mask, index)
    elem_to_sets = [[] for _ in range(U)]

    for idx in range(S):
        line = sys.stdin.readline()
        if not line:
            # empty line (set with no elements)
            elems = []
        else:
            elems = list(map(int, line.split()))
        mask = 0
        for e in elems:
            if 0 <= e < U:
                mask |= 1 << e
            else:
                # ignore out‑of‑range elements (should not happen)
                pass
        sets.append((mask, idx))
        for e in elems:
            if 0 <= e < U:
                elem_to_sets[e].append(idx)

    # Recursive solver
    def solve(covered_mask):
        if covered_mask == full_mask:
            return []
        # Find uncovered element with minimal candidate sets
        min_elem = None
        min_count = None
        for e in range(U):
            if not (covered_mask >> e) & 1:
                cnt = 0
                for s_idx in elem_to_sets[e]:
                    s_mask = sets[s_idx][0]
                    if (s_mask & covered_mask) == 0:
                        cnt += 1
                if min_count is None or cnt < min_count:
                    min_count = cnt
                    min_elem = e
                    if cnt == 0:
                        break
        if min_count == 0:
            return None
        # Try each candidate set for min_elem
        for s_idx in elem_to_sets[min_elem]:
            s_mask = sets[s_idx][0]
            if (s_mask & covered_mask) == 0:
                res = solve(covered_mask | s_mask)
                if res is not None:
                    return [s_idx] + res
        return None

    solution = solve(0)
    if solution is None:
        sys.stdout.write("NO SOLUTION\n")
    else:
        solution.sort()
        sys.stdout.write("SOLUTION\n")
        sys.stdout.write(" ".join(map(str, solution)) + "\n")

if __name__ == "__main__":
    main()


**Solution Explanation**

The Minimum Vertex Cover (MVC) problem asks for the smallest set of vertices that touches every edge of a graph.  
It is NP‑hard, so for large inputs we use a simple 2‑approximation:

* Find a **maximal matching** – a set of edges with no two sharing an endpoint.  
* Take **both endpoints** of every edge in the matching.  
  This set is a vertex cover (every edge of the matching is covered) and its size is at most twice the optimum.

The algorithm works in linear time and uses only `O(V)` additional memory:

1. Read the graph (vertices `V`, edges `E`).  
2. Keep a boolean array `matched[v]` – true if vertex `v` already belongs to the cover.  
3. Scan all edges.  
   * If neither endpoint is matched, mark both as matched (they are added to the cover).  
   * Self‑loops (`u == v`) are handled separately: a single vertex is added if not already matched.  
4. After processing all edges, output the number of matched vertices and iterate over `0..V‑1` to print them in increasing order.

The cover size is at most `2·OPT`, which is optimal for a 2‑approximation.  
The algorithm runs in `O(E)` time and uses `O(V)` memory (≈2 MiB for `V = 2 M`), easily fitting the limits.

**Rust Code**

```rust
use std::io::{self, BufRead, BufReader};

fn main() {
    // Read the first line: number of vertices and edges
    let stdin = io::stdin();
    let mut lines = BufReader::new(stdin.lock()).lines();

    let first_line = lines.next().expect("Missing first line")
        .expect("Failed to read first line");
    let mut parts = first_line.split_whitespace();

    let v: usize = parts.next().expect("Missing vertex count")
        .parse().expect("Invalid vertex count");
    let e: usize = parts.next().expect("Missing edge count")
        .parse().expect("Invalid edge count");

    // matched[v] == true  ⇔  vertex v is already in the cover
    let mut matched = vec![false; v];

    // Process each edge
    for _ in 0..e {
        let line = lines.next().expect("Missing edge line")
            .expect("Failed to read edge line");
        let mut parts = line.split_whitespace();

        let u: usize = parts.next().expect("Missing u")
            .parse().expect("Invalid u");
        let v: usize = parts.next().expect("Missing v")
            .parse().expect("Invalid v");

        if u == v {
            // Self‑loop: add the vertex once if not already matched
            if !matched[u] {
                matched[u] = true;
            }
        } else {
            // If neither endpoint is matched, add both to the cover
            if !matched[u] && !matched[v] {
                matched[u] = true;
                matched[v] = true;
            }
        }
    }

    // Count and output the cover vertices in sorted order
    let mut count = 0;
    for &m in &matched {
        if m {
            count += 1;
        }
    }
    println!("{}", count);
    for i in 0..v {
        if matched[i] {
            print!("{} ", i);
        }
    }
    println!();
}
```

**Explanation of the Code**

* **Reading input** – `BufReader` provides fast line‑by‑line input handling.
* **`matched` array** – tracks which vertices are already in the cover.
* **Edge processing** – for each edge we check the two endpoints; if both are free we add them to the cover.
* **Output** – we count the matched vertices and print them in increasing order (0‑based) to satisfy the required format.

The program compiles without warnings, runs in linear time, and produces a valid vertex cover for graphs up to the given size limits.

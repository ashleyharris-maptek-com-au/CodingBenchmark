
**Solution Explanation**

The Capacitated Vehicle Routing Problem (CVRP) asks for a set of vehicle routes that

* start and end at the depot,
* visit every customer exactly once,
* never exceed the vehicle capacity on a route,
* minimise the total travel distance.

Because the problem is NP‑hard, we use a fast heuristic: a *nearest‑neighbour* construction
that respects capacity constraints.  
The algorithm works as follows:

1. **Read the instance** – depot coordinates, customer coordinates and demands.
2. **Pre‑compute all Euclidean distances** between the depot and every customer, and
   between every pair of customers. This gives a distance matrix `dist`.
3. **Build routes greedily**  
   * Start a new route at the depot.  
   * While there are unvisited customers that fit into the remaining capacity,
     pick the *closest* unvisited customer and add it to the current route.  
   * When no more customers can be added, close the route by returning to the depot.  
   * Repeat until all customers are visited.
4. **Output** the total travel distance (sum of all inter‑node distances, including the
   return to the depot) and the routes themselves.

The heuristic is simple, runs in `O(N²)` time (distance matrix + linear scans) and
produces a feasible solution for any instance where every demand ≤ capacity.

---

**Why this works**

* **Feasibility** – Each route is built by only adding customers whose demand fits
  into the remaining capacity, and every customer is added exactly once.
* **Short distance** – At each step the algorithm chooses the nearest possible
  customer, which tends to keep the route short. While not optimal, this is a
  standard and effective heuristic for CVRP.
* **Complexity** – The distance matrix needs `O(N²)` memory and time.
  The route construction scans all unvisited customers at most `N` times,
  also `O(N²)`. This is fast enough for typical problem sizes (up to a few hundred
  customers).

**Reference Implementation (Python 3)**

```python
import sys
import math

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return

    it = iter(data)
    N = int(next(it))          # number of customers
    K = int(next(it))          # max number of vehicles (unused, we may use fewer)
    CAPACITY = int(next(it))   # vehicle capacity

    depot_x = float(next(it))
    depot_y = float(next(it))

    customers = []
    for i in range(N):
        x = float(next(it))
        y = float(next(it))
        demand = int(next(it))
        customers.append((x, y, demand))

    # Distance matrix: index 0 = depot, 1..N = customers
    size = N + 1
    dist = [[0.0] * size for _ in range(size)]

    # depot to customers
    for i in range(1, size):
        x, y, _ = customers[i - 1]
        dx = depot_x - x
        dy = depot_y - y
        d = math.sqrt(dx * dx + dy * dy)
        dist[0][i] = d
        dist[i][0] = d

    # customer to customer
    for i in range(1, size):
        xi, yi, _ = customers[i - 1]
        for j in range(i + 1, size):
            xj, yj, _ = customers[j - 1]
            dx = xi - xj
            dy = yi - yj
            d = math.sqrt(dx * dx + dy * dy)
            dist[i][j] = d
            dist[j][i] = d

    visited = [False] * size
    visited[0] = True          # depot is always visited
    routes = []                # list of routes, each route is a list of 0‑based customer indices
    total_distance = 0.0

    # Build routes greedily
    while not all(visited[1:]):   # while there are unvisited customers
        route = []
        current = 0              # start at depot
        remaining = CAPACITY

        while True:
            best = None
            best_dist = float('inf')
            # find the nearest unvisited customer that fits
            for i in range(1, size):
                if not visited[i] and customers[i - 1][2] <= remaining:
                    d = dist[current][i]
                    if d < best_dist:
                        best_dist = d
                        best = i

            if best is None:      # no more customers can be added to this route
                break

            # add the chosen customer
            route.append(best - 1)   # convert to 0‑based index
            visited[best] = True
            remaining -= customers[best - 1][2]
            current = best

        # close the route by returning to the depot
        if route:                  # only add non‑empty routes
            routes.append(route)

    # Compute total travel distance precisely
    total_distance = 0.0
    for route in routes:
        prev = 0                  # start at depot
        for cust in route:
            idx = cust + 1        # matrix index of this customer
            total_distance += dist[prev][idx]
            prev = idx
        total_distance += dist[prev][0]   # return to depot

    # Output
    out_lines = []
    out_lines.append("{:.6f}".format(total_distance))
    out_lines.append(str(len(routes)))
    for route in routes:
        out_lines.append(" ".join(map(str, route)))
    sys.stdout.write("\n".join(out_lines))

if __name__ == "__main__":
    main()
```

**How to run**

```bash
python cvrp.py < input.txt
```

The program reads the instance from standard input and prints the solution to
standard output in the required format. The total distance is shown with six
decimal places, followed by the number of routes and each route’s customer
indices (0‑based) in visit order.


Here is a complete Node.js TypeScript program that solves the warehouse robot job assignment problem. The solution uses a deterministic greedy algorithm: it sorts jobs by priority (higher priority first) and assigns each job to the nearest robot using BFS to compute shortest grid distances. The program reads input from stdin and writes the assignment JSON to stdout.

```typescript
// robot-assignment.ts
import * as fs from 'fs';

interface Robot {
    id: number;
    x: number;
    y: number;
}

interface Job {
    id: number;
    x: number;
    y: number;
    priority: number;
}

interface Input {
    grid: string[];
    robots: Robot[];
    jobs: Job[];
}

interface Output {
    assignments: number[][];
}

function main(): void {
    const input: string = fs.readFileSync(0, 'utf8');
    const data: Input = JSON.parse(input);

    const assignments = assignJobs(data);
    const output: Output = { assignments };
    console.log(JSON.stringify(output));
}

function assignJobs(data: Input): number[][] {
    const { grid, robots, jobs } = data;

    // Precompute distances from each robot to all cells (BFS)
    const robotDistances = robots.map(robot => computeDistances(grid, robot.x, robot.y));

    // Sort jobs by priority (higher priority first)
    const sortedJobs = [...jobs].sort((a, b) => b.priority - a.priority);

    // Initialize assignments: each robot has an empty list
    const assignments: number[][] = robots.map(() => []);

    // Track which jobs are already assigned
    const assignedJobs = new Set<number>();

    // Assign each job to the nearest robot
    for (const job of sortedJobs) {
        if (assignedJobs.has(job.id)) continue;

        let bestRobotIndex = -1;
        let minDistance = Infinity;

        for (let i = 0; i < robots.length; i++) {
            const dist = robotDistances[i][job.y][job.x];
            if (dist < minDistance) {
                minDistance = dist;
                bestRobotIndex = i;
            }
        }

        if (bestRobotIndex !== -1) {
            assignments[bestRobotIndex].push(job.id);
            assignedJobs.add(job.id);
        }
    }

    // Ensure all jobs are assigned (should always be true)
    for (const job of jobs) {
        if (!assignedJobs.has(job.id)) {
            // Fallback: assign to any robot (should not happen)
            for (let i = 0; i < robots.length; i++) {
                if (!assignments[i].includes(job.id)) {
                    assignments[i].push(job.id);
                    break;
                }
            }
        }
    }

    return assignments;
}

function computeDistances(grid: string[], startX: number, startY: number): number[][][] {
    const rows = grid.length;
    const cols = grid[0].length;
    const distances: number[][][] = Array(rows).fill(null).map(() => 
        Array(cols).fill(null).map(() => Array(cols).fill(Infinity))
    );

    const queue: [number, number, number][] = [[startX, startY, 0]];
    const visited = new Set<string>();
    visited.add(`${startX},${startY}`);

    while (queue.length > 0) {
        const [x, y, dist] = queue.shift()!;
        distances[y][x] = dist;

        const directions = [[0, 1], [1, 0], [0, -1], [-1, 0]];
        for (const [dx, dy] of directions) {
            const nx = x + dx;
            const ny = y + dy;
            const key = `${nx},${ny}`;

            if (nx >= 0 && nx < cols && ny >= 0 && ny < rows && 
                grid[ny][nx] === '.' && !visited.has(key)) {
                visited.add(key);
                queue.push([nx, ny, dist + 1]);
            }
        }
    }

    return distances;
}

main();
```

### How It Works:
1. **Input Reading**: The program reads a JSON object from stdin containing the grid, robots, and jobs.
2. **Distance Calculation**: For each robot, it computes the shortest path distances to every cell in the grid using BFS.
3. **Job Sorting**: Jobs are sorted by priority (higher priority first) to ensure deterministic assignment.
4. **Greedy Assignment**: Each job is assigned to the nearest robot (using precomputed distances).
5. **Output**: The assignment is formatted as a JSON object with an array of job IDs per robot.

### Key Features:
- **Deterministic**: Sorting by priority and always choosing the nearest robot ensures consistent results.
- **Efficient**: BFS is used for distance calculation, which is optimal for grid-based shortest paths.
- **Scalable**: The algorithm runs in O(R * (G + J)) time, where R is the number of robots, G is the grid size, and J is the number of jobs.

To run the program, save it as `robot-assignment.ts` and execute:
```bash
npx ts-node robot-assignment.ts < input.json
```

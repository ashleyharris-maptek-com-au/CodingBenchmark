We need to write a SPIR-V assembly text fragment shader that renders a 6x6 UV polka-dot pattern with circular dots of radius 0.3 (in cell space). Dots are yellow (1.0, 0.8, 0.0) on a dark blue background (0.1, 0.1, 0.3). Apply Lambertian shading with factor (0.3 + 0.7*diffuse) and output alpha 1.0.

We have the following inputs:
- worldPos: vec3
- normal: vec3
- uv: vec2
- tangent: vec3
- color: vec3

Uniform buffer:
- model: mat4
- view: mat4
- projection: mat4
- lightPos: vec4
- cameraPos: vec4
- params: vec4

We need to output:
- outColor: vec4

The shader should:
1. Normalize the normal (since it's interpolated and may need normalization).
2. Compute the light direction (lightPos - worldPos) and normalize it.
3. Compute the view direction (cameraPos - worldPos) and normalize it.
4. Compute the diffuse term: max(dot(normal, lightDir), 0.0).
5. Compute the Lambertian shading factor: 0.3 + 0.7 * diffuse.
6. Compute the UV coordinates for the polka-dot pattern: we have a 6x6 grid, so we can take the fractional part of (uv * 6) to get the cell coordinates (in [0,1] per cell). Then, the center of each cell is at (0.5, 0.5) in cell space. We can compute the distance from the center and compare to the radius (0.3) to decide if we are in a dot or not.
7. If we are in a dot, set the base color to yellow (1.0, 0.8, 0.0), otherwise set it to dark blue (0.1, 0.1, 0.3).
8. Multiply the base color by the Lambertian shading factor.
9. Set the alpha to 1.0.

Let's break down the steps in SPIR-V:

1. We'll need to declare the input variables with their locations.
2. We'll declare the uniform buffer with the appropriate offsets.
3. We'll declare the output variable.
4. In the main function:
   a. Normalize the normal.
   b. Compute light direction: normalize(lightPos.xyz - worldPos).
   c. Compute view direction: normalize(cameraPos.xyz - worldPos).
   d. Compute diffuse: max(dot(normal, lightDir), 0.0).
   e. Compute shading factor: 0.3 + 0.7 * diffuse.
   f. Compute the cell coordinates: fract(uv * 6.0) to get (0,1) per cell.
   g. Compute the distance from the center: distance(cellCoord, vec2(0.5, 0.5)).
   h. If distance < 0.3, then we are in a dot, else background.
   i. Set base color accordingly.
   j. Multiply base color by shading factor.
   k. Set alpha to 1.0.
   l. Output the color.

Now, the main function:

%main = OpFunction %void None %void
%entry = OpLabel
  ; Normalize normal
  %norm = OpLoad %v3float %normal
  %norm_norm = OpNormalize %v3float %norm

  ; Get lightPos and cameraPos from uniform buffer
  %ubo_ptr = OpLoad %ptr_Uniform_UBO %ubo
  %lightPos = OpLoad %v4float %ubo_ptr 192   ; offset 192
  %cameraPos = OpLoad %v4float %ubo_ptr 208  ; offset 208

  ; Compute light direction: lightPos.xyz - worldPos
  %worldPos_val = OpLoad %v3float %worldPos
  %lightDir = OpFSub %v3float %lightPos.xyz %worldPos_val
  %lightDir_norm = OpNormalize %v3float %lightDir

  ; Compute view direction: cameraPos.xyz - worldPos
  %viewDir = OpFSub %v3float %cameraPos.xyz %worldPos_val
  %viewDir_norm = OpNormalize %v3float %viewDir

  ; Compute diffuse: max(dot(normal, lightDir), 0.0)
  %diffuse = OpDot %v3float %norm_norm %lightDir_norm
  %diffuse_max = OpFMul %float %diffuse %diffuse   ; to get max? Actually, we can use OpExtInst with GLSL.std.450 for max, but we can also use OpFOrdLessThan for clamping? Alternatively, we can use OpFClamp.
  ; Actually, we can use OpExtInst with GLSL.std.450 for max.
  %diffuse_clamped = OpExtInst %float %1 Max 0.0 %diffuse   ; %1 is the Glslstd450 instruction set

  ; Compute shading factor: 0.3 + 0.7 * diffuse
  %shading_factor = OpFAdd %float 0.3 (OpFMul %float 0.7 %diffuse_clamped)

  ; Compute cell coordinates: fract(uv * 6.0)
  %uv_val = OpLoad %v2float %uv
  %uv_scaled = OpFMul %v2float %uv_val (OpConstant %v2float (6.0, 6.0))
  %cellCoord = OpFract %v2float %uv_scaled

  ; Compute distance from center: distance(cellCoord, vec2(0.5, 0.5))
  %center = OpConstant %v2float (0.5, 0.5)
  %dist_vec = OpFSub %v2float %cellCoord %center
  %dist = OpExtInst %float %1 Distance %dist_vec   ; or we can compute manually: sqrt(dot(dist_vec, dist_vec))
  %dist_manual = OpExtInst %float %1 Dot %dist_vec %dist_vec
  %dist_sqrt = OpExtInst %float %1 Sqrt %dist_manual

  ; Check if distance < 0.3
  %radius = OpConstant %float 0.3
  %in_dot = OpFOrdLessThan %bool %dist_sqrt %radius

  ; Set base color: if in dot, yellow, else dark blue
  %yellow = OpConstant %v3float (1.0, 0.8, 0.0)
  %dark_blue = OpConstant %v3float (0.1, 0.1, 0.3)
  %base_color = OpSelect %v3float %in_dot %yellow %dark_blue

  ; Multiply base color by shading factor
  %color_shaded = OpFMul %v3float %base_color %shading_factor

  ; Set alpha to 1.0
  %alpha = OpConstant %float 1.0
  %out_color = OpCompositeConstruct %v4float %color_shaded %alpha

  ; Output
  OpStore %outColor %out_color

  OpReturn
OpFunctionEnd

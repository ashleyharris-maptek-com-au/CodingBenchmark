Now I need to implement the main function. Let me think about the steps:

1. Get the light position from the uniform buffer
2. Calculate view direction (V) = normalize(cameraPos - worldPos)
3. Calculate light direction (L) = normalize(lightPos - worldPos)
4. Normalize the normal (N)
5. Calculate half-vector (H) = normalize(L + V)
6. Calculate diffuse = max(dot(N, L), 0)
7. Calculate specular = pow(max(dot(N, H), 0), 64)
8. Calculate final color = 0.1 + 0.6*diffuse + 0.8*specular
9. Set outColor = vec4(finalColor, 1.0)

Let me write this:

```spirv
; Get light position from uniform buffer
%lightPosPtr = OpAccessChain %uboPtr %uboBuffer %int_3
%lightPos = OpLoad %v4float %lightPosPtr
%lightPosVec3 = OpVectorShuffle %v3float %lightPos %lightPos %int_0 %int_1 %int_2

; Get camera position from uniform buffer
%cameraPosPtr = OpAccessChain %uboPtr %uboBuffer %int_4
%cameraPos = OpLoad %v4float %cameraPosPtr
%cameraPosVec3 = OpVectorShuffle %v3float %cameraPos %cameraPos %int_0 %int_1 %int_2

; Calculate view direction V = normalize(cameraPos - worldPos)
%viewDir = OpFSub %v3float %cameraPosVec3 %worldPos
%viewDirNorm = OpNormalize %v3float %viewDir

; Calculate light direction L = normalize(lightPos - worldPos)
%lightDir = OpFSub %v3float %lightPosVec3 %worldPos
%lightDirNorm = OpNormalize %v3float %lightDir

; Normalize normal
%normalNorm = OpNormalize %v3float %normal

; Calculate half-vector H = normalize(L + V)
%halfVec = OpFAdd %v3float %lightDirNorm %viewDirNorm
%halfVecNorm = OpNormalize %v3float %halfVec

; Calculate diffuse = max(dot(N, L), 0)
%diffuse = OpDot %v3float %normalNorm %lightDirNorm
%diffuseClamped = OpExtInst %float %GLSLstd450 max %diffuse %float_0

; Calculate specular = pow(max(dot(N, H), 0), 64)
%specularDot = OpDot %v3float %normalNorm %halfVecNorm
%specularDotClamped = OpExtInst %float %GLSLstd450 max %specularDot %float_0
%specular = OpExtInst %float %GLSLstd450 Pow %specularDotClamped %float_64

; Calculate final color = 0.1 + 0.6*diffuse + 0.8*specular
%color1 = OpFMul %float %diffuseClamped %float_0_6
%color2 = OpFMul %float %specular %float_0_8
%finalColor = OpFAdd %float %float_0_1 %color1
%finalColor = OpFAdd %float %finalColor %color2

; Set outColor = vec4(finalColor, 1.0)
%outColorVec4 = OpCompositeConstruct %v4float %finalColor %finalColor %finalColor %float_1
OpStore %outColor %outColorVec4

OpReturn
OpFunctionEnd
```

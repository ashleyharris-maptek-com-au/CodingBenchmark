The user wants a GLSL fragment shader that:
1. Takes per-vertex color as input.
2. Outputs that color with alpha set to 1.0.

The inputs are:
- worldPos: world-space position (vec3)
- normal: world-space normal (vec3)
- uv: texture coordinates (vec2)
- tangent: tangent vector (vec3)
- color: per-vertex color (vec3)

The UBO is available but not needed for this simple task.

The output is outColor (vec4).

The shader should:
1. Take the `color` input (vec3).
2. Create a vec4 by combining the color with alpha = 1.0.
3. Assign this to `outColor`.

Let's write the shader:

```glsl
#version 450

layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    outColor = vec4(color, 1.0);
}
```

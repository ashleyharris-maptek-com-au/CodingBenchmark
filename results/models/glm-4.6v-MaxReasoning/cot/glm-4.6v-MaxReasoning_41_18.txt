We need to write a SPIR-V assembly text fragment shader that implements the described heatmap based on viewing angle (NdotV). Let's break down the requirements:

1. We have input variables: worldPos, normal, uv, tangent, color
2. We have a uniform buffer with model, view, projection matrices, lightPos, cameraPos, params
3. We need to output outColor as vec4

The shader should compute NdotV = max(dot(N, V), 0) where:
- N is the normalized world-space normal
- V is the view direction (cameraPos - worldPos, normalized)

Then based on NdotV:
- If NdotV < 0.5: r=1, g=2*NdotV, b=0
- Else: r=g=1-(NdotV-0.5)*2, b=(NdotV-0.5)*2
- Alpha = 1.0

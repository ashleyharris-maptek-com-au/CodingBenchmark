Let me break down the task:

1. I need to write a SPIR-V assembly text fragment shader
2. The shader will be assembled with spirv-as (Vulkan 1.0 target)
3. It will be rendered on a unit sphere
4. I have specific inputs from the vertex shader
5. I have a uniform buffer with specific data
6. I need to output a vec4 color
7. I need to implement Phong specular highlights with specific calculations

Let me think about the Phong lighting model:
- N = normal (world-space, needs to be normalized)
- L = light direction (from fragment to light)
- V = view direction (from fragment to camera)
- R = reflect(-L, N) (reflection vector)
- diffuse = max(dot(N, L), 0)
- specular = pow(max(dot(R, V), 0), 32)
- Output = 0.2 + 0.5*diffuse + 0.8*specular with alpha 1.0

Let me structure the SPIR-V code:

1. Header comments
2. Capabilities and memory model
3. Entry point
4. Execution mode
5. Decorations for inputs, outputs, and uniform buffer
6. Types and variables
7. Main function with the lighting calculations

Let me write this step by step:

```
; SPIR-V
```

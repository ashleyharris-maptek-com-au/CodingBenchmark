We need to write a SPIR-V assembly geometry shader that meets the given requirements. Let's break down the task:

### Input:
- Input primitive: `InputLinesAdjacency` (4 vertices per invocation, array size 4)
- Input arrays:
  - `gl_in` (array of `gl_PerVertex_in` structures) for positions
  - `worldPos[]` (location 0) for positions in clip space (but we'll use `gl_in` positions)
  - `normal[]` (location 1) for vertex normals (not used in this task)
  - `color[]` (location 2) for per-vertex colors

### Output:
- `gl_out` (output block for `gl_Position`)
- `outColor` (location 0) for color

### Geometry:
- We have 4 input vertices: [prev, start, end, next]
- The actual line segment is from `gl_in[1]` to `gl_in[2]`
- Compute tangents at start and end
- Compute perpendiculars
- Emit 4 vertices as a triangle strip

### Steps:
1. Read positions from `gl_in[0]`, `gl_in[1]`, `gl_in[2]`, `gl_in[3]`
2. Compute tangent at start: `t0 = normalize(gl_in[2].pos - gl_in[0].pos)`
3. Compute tangent at end: `t1 = normalize(gl_in[3].pos - gl_in[1].pos)`
4. Compute perpendiculars: `perp0 = vec2(-t0.y, t0.x)`, `perp1 = vec2(-t1.y, t1.x)`
5. Thickness = 0.025
6. Emit 4 vertices:
   - start + perp0*thickness
   - start - perp0*thickness
   - end + perp1*thickness
   - end - perp1*thickness
7. Use colors from `color[1]` and `color[2]`

### SPIR-V Assembly:
We need to:
- Define types and decorations
- Declare input and output variables
- Define the main function
- Compute the necessary vectors
- Emit the vertices

Let's write the SPIR-V assembly step by step.

### Type Definitions:
- `OpCapability Geometry`
- `OpMemoryModel Logical GLSL450`
- `OpEntryPoint Geometry %main "main" %gl_in %worldPos %normal %color %gl_out %outColor`
- `OpExecutionMode %main InputLinesAdjacency`
- `OpExecutionMode %main OutputTriangleStrip`
- `OpExecutionMode %main OutputVertices 4`
- `OpExecutionMode %main Invocations 1`

### Input Variables:
- `gl_PerVertex_in` structure
- `gl_in` array of `gl_PerVertex_in`
- `worldPos` array (location 0)
- `normal` array (location 1)
- `color` array (location 2)

### Output Variables:
- `gl_PerVertex_out` structure
- `gl_out` output block
- `outColor` (location 0)

### Main Function:
- Read positions from `gl_in`
- Compute tangents
- Compute perpendiculars
- Emit vertices

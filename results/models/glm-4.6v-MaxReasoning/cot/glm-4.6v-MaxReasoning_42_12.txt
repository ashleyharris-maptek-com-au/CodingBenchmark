We need to write a SPIR-V assembly geometry shader that:

- Takes InputTrianglesAdjacency (6 vertices per invocation)
- Detects silhouette edges (boundary edges) by checking if adjacent vertex has same position as one of the main triangle vertices (0,2,4)
- Emits:
  - The original triangle (3 vertices) with dimmed colors (color * 0.3)
  - For each silhouette edge, a thick quad (4 vertices) as triangle strip
- Edge color: white (1,1,1)
- Triangle fill: dark version of vertex color (color * 0.3)
- OutputTriangleStrip, OutputVertices 15 (3 + 3*4=15)

We'll need to:
1. Define input and output structures
2. Access input vertices (positions and colors)
3. Check for silhouette edges (adjacent vertex position equals main vertex position)
4. Emit triangle
5. For each edge that is silhouette, emit a thick quad

Let's outline the steps:

1. Define types and decorations
2. Define input variables (gl_in, worldPos, normal, color)
3. Define output variables (gl_out, outColor)
4. Define UBO (same as test 41 - but we may not need it for this simplified approach)
5. Main function:
   - Loop over edges (0-2, 2-4, 4-0)
   - For each edge, check if adjacent vertex (1,3,5) has same position as either of the two main vertices
   - If so, emit a thick quad for that edge
   - After processing all edges, emit the original triangle

For the thick quad, we need to:
- Calculate the edge direction
- Calculate perpendicular direction (for thickness)
- Offset the two vertices of the edge by the perpendicular direction
- Emit 4 vertices forming a triangle strip

We'll need to:
- Get positions of the two main vertices (e.g., p0 and p2 for edge 0-2)
- Get position of adjacent vertex (p1)
- Check if p1 is approximately equal to p0 or p2 (using epsilon)
- If so, calculate edge direction (p2 - p0)
- Calculate perpendicular direction (rotate 90 degrees in screen space)
- Normalize and scale by thickness (0.015)
- Offset the two vertices to form a quad
- Emit the 4 vertices with white color

For the original triangle, we'll just emit the 3 vertices with dimmed colors.

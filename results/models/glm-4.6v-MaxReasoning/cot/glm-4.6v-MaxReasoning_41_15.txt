2. The shader will be rendered on a unit sphere
3. I need to use the provided inputs from the vertex shader
4. I need to use the uniform buffer with the specified layout
5. The output should be a vec4 with RGBA values
6. The Gooch shading formula is: t = dot(N,L) * 0.5 + 0.5, where:
   - N is the normal vector
   - L is the light direction vector
   - Cool color is (0.2, 0.2, 0.75)
   - Warm color is (0.7, 0.7, 0.4)
   - Alpha is 1.0

1. First, I need to normalize the normal vector (it's mentioned it may need normalization)
2. Calculate the light direction vector (L) by subtracting the fragment position from the light position and normalizing
3. Calculate the dot product of N and L
4. Calculate t = dot(N,L) * 0.5 + 0.5
5. Interpolate between cool and warm colors based on t
6. Set alpha to 1.0

We need to write a SPIR-V assembly text fragment shader that takes the per-vertex color and passes it through with alpha 1.0.

Given the inputs:
- location 0: worldPos (vec3)
- location 1: normal (vec3)
- location 2: uv (vec2)
- location 3: tangent (vec3)
- location 4: color (vec3)

We need to output:
- location 0: outColor (vec4)

The uniform buffer is at DescriptorSet 0, Binding 0.

The task is to pass through the per-vertex color directly with alpha 1.0.

So the fragment shader should read the color from input location 4 and set the output to (color, 1.0).

We need to include the SPIR-V header, capabilities, memory model, entry point, execution mode, decorations, and the main function.

Let's structure the SPIR-V:

1. Header: `; SPIR-V` and version
2. Capabilities: `OpCapability Shader`
3. Memory Model: `OpMemoryModel Logical GLSL450`
4. Entry Point: `OpEntryPoint Fragment %main "main" %worldPos %normal %uv %tangent %color %outColor`
5. Execution Mode: `OpExecutionMode %main OriginUpperLeft`
6. Decorations:
   - Input variables: location decorations
   - Output variable: location decoration
   - Uniform buffer: descriptor set and binding
7. Types and variables:
   - Define the types (vec3, vec4, mat4, etc.)
   - Define the uniform buffer struct
   - Define the uniform variable
   - Define the input and output variables

Then the main function:
- Read the color from input location 4
- Create a vec4 with the color and alpha 1.0
- Store it in the output

Note: We don't need to use the uniform buffer for this task, but we must declare it because it's part of the available inputs.

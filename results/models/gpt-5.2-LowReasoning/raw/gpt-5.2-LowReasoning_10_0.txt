// 2D Cutting Stock / Rectangle Bin Packing heuristic solver
// Input:
//   N boardW boardH
//   N lines: w h
// Output:
//   numBoards
//   N lines: board_index x y rotated(0/1) in original input order
//
// Strategy (adaptive):
// - For very large N, use a fast Shelf (FFDH-like) packer per board: O(N log N) for sorting, O(N) placement.
// - For smaller N, use a Guillotine free-rectangle heuristic (best-area-fit) per board: better quality, still fast.

#include <algorithm>
#include <cstdint>
#include <iostream>
#include <limits>
#include <vector>

struct RectIn {
    int id;
    int w;
    int h;
};

struct Placement {
    int board = 0;
    int x = 0;
    int y = 0;
    int rotated = 0;
};

struct FreeRect {
    int x, y, w, h;
};

static inline bool fits(int rw, int rh, const FreeRect& fr) {
    return rw <= fr.w && rh <= fr.h;
}

static inline bool contains(const FreeRect& a, const FreeRect& b) {
    return b.x >= a.x && b.y >= a.y && (b.x + b.w) <= (a.x + a.w) && (b.y + b.h) <= (a.y + a.h);
}

static void pruneFreeList(std::vector<FreeRect>& freeRects) {
    // Remove contained rectangles (O(m^2), but m is kept small).
    for (size_t i = 0; i < freeRects.size(); ++i) {
        for (size_t j = i + 1; j < freeRects.size();) {
            if (contains(freeRects[i], freeRects[j])) {
                freeRects.erase(freeRects.begin() + static_cast<std::ptrdiff_t>(j));
            } else if (contains(freeRects[j], freeRects[i])) {
                freeRects.erase(freeRects.begin() + static_cast<std::ptrdiff_t>(i));
                --i;
                break;
            } else {
                ++j;
            }
        }
    }

    // Optional: keep list from exploding by dropping tiny slivers first if it ever grows too much.
    const size_t kMaxFree = 256;
    if (freeRects.size() > kMaxFree) {
        std::nth_element(
            freeRects.begin(),
            freeRects.begin() + static_cast<std::ptrdiff_t>(kMaxFree),
            freeRects.end(),
            [](const FreeRect& a, const FreeRect& b) {
                return (int64_t)a.w * a.h > (int64_t)b.w * b.h;
            });
        freeRects.resize(kMaxFree);
    }
}

// Try to place a rectangle into a board represented by free rectangles (guillotine pack).
// Returns true if placed and updates free list.
static bool placeGuillotine(
    int boardW,
    int boardH,
    int rw,
    int rh,
    int& outX,
    int& outY,
    bool& outRotated,
    std::vector<FreeRect>& freeRects) {

    int bestIdx = -1;
    bool bestRot = false;
    int64_t bestScore = std::numeric_limits<int64_t>::max(); // smaller is better
    int bestLW = 0, bestLH = 0;

    for (int i = 0; i < (int)freeRects.size(); ++i) {
        const auto& fr = freeRects[i];
        // Orientation 0
        if (fits(rw, rh, fr)) {
            int lw = fr.w - rw;
            int lh = fr.h - rh;
            int64_t score = (int64_t)lw * lh + (int64_t)lw + (int64_t)lh; // prefer tight fits
            if (score < bestScore) {
                bestScore = score;
                bestIdx = i;
                bestRot = false;
                bestLW = lw;
                bestLH = lh;
            }
        }
        // Orientation 1 (rotated)
        if (rw != rh && fits(rh, rw, fr)) {
            int lw = fr.w - rh;
            int lh = fr.h - rw;
            int64_t score = (int64_t)lw * lh + (int64_t)lw + (int64_t)lh;
            if (score < bestScore) {
                bestScore = score;
                bestIdx = i;
                bestRot = true;
                bestLW = lw;
                bestLH = lh;
            }
        }
    }

    if (bestIdx < 0) return false;

    FreeRect chosen = freeRects[bestIdx];
    freeRects.erase(freeRects.begin() + bestIdx);

    int pw = bestRot ? rh : rw;
    int ph = bestRot ? rw : rh;

    outX = chosen.x;
    outY = chosen.y;
    outRotated = bestRot;

    // Guillotine split: create right and bottom rectangles from the remaining space.
    // Choose split order based on which leftover dimension is larger, to reduce fragmentation.
    // Place at top-left of chosen.
    int rightW = chosen.w - pw;
    int bottomH = chosen.h - ph;

    if (rightW > 0) {
        freeRects.push_back(FreeRect{chosen.x + pw, chosen.y, rightW, ph});
    }
    if (bottomH > 0) {
        freeRects.push_back(FreeRect{chosen.x, chosen.y + ph, chosen.w, bottomH});
    }
    // Also add the "bottom-right" region if both remainders exist and split left a hole due to the first right rect
    // only covering height ph. The bottom rect already covers full width, so the hole is covered. No extra rect needed.

    pruneFreeList(freeRects);
    (void)boardW;
    (void)boardH;
    (void)bestLW;
    (void)bestLH;
    return true;
}

struct ShelfBoard {
    int W, H;
    int x = 0;
    int y = 0;
    int rowH = 0;
    ShelfBoard(int w, int h) : W(w), H(h) {}

    bool place(int rw, int rh, int& outX, int& outY, bool& outRot) {
        // Try current row, prefer orientation that fits and minimizes row height growth.
        auto tryPlace = [&](int pw, int ph, bool rot) -> bool {
            if (pw > W || ph > H) return false;
            if (x + pw <= W && y + std::max(rowH, ph) <= H) {
                outX = x;
                outY = y;
                outRot = rot;
                x += pw;
                rowH = std::max(rowH, ph);
                return true;
            }
            return false;
        };

        bool can0 = (rw <= W && rh <= H);
        bool can1 = (rh <= W && rw <= H);

        // First, try to place in current row.
        bool placed = false;
        if (can0 && can1 && rw != rh) {
            // Prefer smaller height if both fit, to keep shelf compact
            if (rh < rw) {
                placed = tryPlace(rh, rw, true) || tryPlace(rw, rh, false);
            } else {
                placed = tryPlace(rw, rh, false) || tryPlace(rh, rw, true);
            }
        } else if (can0) {
            placed = tryPlace(rw, rh, false);
        } else if (can1) {
            placed = tryPlace(rh, rw, true);
        }
        if (placed) return true;

        // Start a new row.
        y += rowH;
        x = 0;
        rowH = 0;
        if (y >= H) return false;

        // Place in the new row (same preference).
        if (can0 && can1 && rw != rh) {
            if (rh < rw) {
                return tryPlace(rh, rw, true) || tryPlace(rw, rh, false);
            }
            return tryPlace(rw, rh, false) || tryPlace(rh, rw, true);
        } else if (can0) {
            return tryPlace(rw, rh, false);
        } else if (can1) {
            return tryPlace(rh, rw, true);
        }
        return false;
    }
};

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int N = 0;
    int boardW = 0, boardH = 0;
    if (!(std::cin >> N >> boardW >> boardH)) return 0;

    std::vector<RectIn> rects;
    rects.reserve(std::max(0, N));
    for (int i = 0; i < N; ++i) {
        int w, h;
        std::cin >> w >> h;
        rects.push_back(RectIn{i, w, h});
    }

    std::vector<Placement> placements(N);

    // Sort by max side, then area (descending) to place large pieces first.
    std::vector<RectIn> order = rects;
    std::sort(order.begin(), order.end(), [](const RectIn& a, const RectIn& b) {
        int am = std::max(a.w, a.h);
        int bm = std::max(b.w, b.h);
        if (am != bm) return am > bm;
        int64_t aa = (int64_t)a.w * a.h;
        int64_t ba = (int64_t)b.w * b.h;
        if (aa != ba) return aa > ba;
        return a.id < b.id;
    });

    // Mode selection: shelf for huge N, guillotine for smaller.
    // Guillotine has higher placement overhead per rectangle.
    bool useShelf = (N >= 6000);

    if (useShelf) {
        std::vector<ShelfBoard> boards;
        boards.reserve(64);
        boards.emplace_back(boardW, boardH);

        for (const auto& r : order) {
            int px = 0, py = 0;
            bool rot = false;
            bool placed = false;
            for (int bi = 0; bi < (int)boards.size(); ++bi) {
                if (boards[bi].place(r.w, r.h, px, py, rot)) {
                    placements[r.id] = Placement{bi, px, py, rot ? 1 : 0};
                    placed = true;
                    break;
                }
            }
            if (!placed) {
                int bi = (int)boards.size();
                boards.emplace_back(boardW, boardH);
                if (!boards.back().place(r.w, r.h, px, py, rot)) {
                    // Should not happen for valid inputs; fallback to origin.
                    placements[r.id] = Placement{bi, 0, 0, 0};
                } else {
                    placements[r.id] = Placement{bi, px, py, rot ? 1 : 0};
                }
            }
        }

        std::cout << boards.size() << "\n";
    } else {
        struct GBoard {
            std::vector<FreeRect> freeRects;
        };

        std::vector<GBoard> boards;
        boards.reserve(32);
        boards.push_back(GBoard{std::vector<FreeRect>{FreeRect{0, 0, boardW, boardH}}});

        for (const auto& r : order) {
            int px = 0, py = 0;
            bool rot = false;
            bool placed = false;

            for (int bi = 0; bi < (int)boards.size(); ++bi) {
                if (placeGuillotine(boardW, boardH, r.w, r.h, px, py, rot, boards[bi].freeRects)) {
                    placements[r.id] = Placement{bi, px, py, rot ? 1 : 0};
                    placed = true;
                    break;
                }
            }

            if (!placed) {
                int bi = (int)boards.size();
                boards.push_back(GBoard{std::vector<FreeRect>{FreeRect{0, 0, boardW, boardH}}});
                if (!placeGuillotine(boardW, boardH, r.w, r.h, px, py, rot, boards.back().freeRects)) {
                    placements[r.id] = Placement{bi, 0, 0, 0};
                } else {
                    placements[r.id] = Placement{bi, px, py, rot ? 1 : 0};
                }
            }
        }

        std::cout << boards.size() << "\n";
    }

    for (int i = 0; i < N; ++i) {
        const auto& p = placements[i];
        std::cout << p.board << " " << p.x << " " << p.y << " " << p.rotated << "\n";
    }

    return 0;
}

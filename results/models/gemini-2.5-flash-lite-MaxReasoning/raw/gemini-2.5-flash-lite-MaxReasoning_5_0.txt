The solution has been implemented as a complete Rust project.

### Project Files:
- **`Cargo.toml`**: Configures the project.
- **`src/main.rs`**: Contains the Hamiltonian path solver utilizing backtracking and Warnsdorff's heuristic for efficiency on grids up to 2500 cells.

The code correctly parses the input, implements the pathfinding logic with the required constraints (starting at (0,0), visiting all non-obstacle cells exactly once, and handling the optional cycle requirement), and formats the output as specified.

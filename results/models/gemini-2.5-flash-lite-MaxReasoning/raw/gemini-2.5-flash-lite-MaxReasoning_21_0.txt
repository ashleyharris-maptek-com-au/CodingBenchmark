I have completed the implementation of the generalized N-Dimensional Snake Game solver.

The `snake_solver.cpp` file provides a C++20-compatible solution that:
*   **Handles arbitrary dimensions (N)** by representing coordinates as `std::vector<int>`.
*   **Ensures efficiency** for large N by using a Manhattan-distance-based greedy heuristic to select moves, coupled with collision detection against obstacles and the snake's own body.
*   **Manages game state internally**, simulating the snake's movement and growth turn-by-turn.
*   **Avoids non-standard dependencies** and adheres to standard C++ library practices for MSVC compatibility.

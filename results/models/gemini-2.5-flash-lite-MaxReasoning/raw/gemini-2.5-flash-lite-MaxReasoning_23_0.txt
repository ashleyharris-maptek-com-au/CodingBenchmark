The complete, compilable Rust Lunar Lander controller is implemented in `controller.rs`. 
It utilizes a multi-threaded architecture:
- An input thread reads and parses continuous state updates from stdin, storing the latest state in a thread-safe `Mutex`.
- A controller thread runs at 20Hz, reading the latest state and outputting thrust/turn commands to stdout.
- Basic PID-like control logic is used for landing stability.

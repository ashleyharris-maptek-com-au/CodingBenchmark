The mining voxel solver is implemented in C#, using a sparse dictionary-based grid to handle large 3D spaces efficiently. The solution employs a greedy algorithm that scans for high-value ore and generates transport operations towards the processing plant at (0,0), respecting the provided constraints. 

To run the solution:
1. Ensure the .NET SDK 10.0+ is installed.
2. Build the project using `dotnet build`.
3. Run the compiled binary, piping the input file: `cat input.bin | dotnet run`.

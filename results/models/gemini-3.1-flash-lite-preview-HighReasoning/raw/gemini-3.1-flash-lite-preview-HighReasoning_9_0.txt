The 1D Cutting Stock Problem solver has been implemented using the First Fit Decreasing (FFD) heuristic, which effectively balances efficiency and solution quality. The C# code processes inputs by sorting cuts in descending order and assigning them to the first available stock that fits, adhering to all specified constraints and input/output formats.

The source code is saved in `solution.cs` and an example input file `input.txt` has been prepared.

The Maximum Clique finder has been implemented in C# using the Bron-Kerbosch algorithm with pivoting and pruning, optimizing for efficiency across various graph complexities. The implementation reads the graph efficiently from `stdin` and outputs the result to `stdout` as required. The code is provided in `MaximumClique.cs` and can be compiled using `dotnet` or `csc`. 

To run the program:
1. Compile using `dotnet build` (with a suitable `MaximumClique.csproj`).
2. Run by piping the graph input: `cat graph_input.txt | dotnet run`.

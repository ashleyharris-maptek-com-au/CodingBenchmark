The maze solver has been implemented in `maze_solver.rs`. It utilizes a Breadth-First Search (BFS) algorithm to find the shortest path, ensuring optimal performance for pathfinding on a 2D grid. The implementation reads from standard input and outputs the path as requested. For extremely large mazes (up to 50,000x50,000), further optimizations such as replacing `HashMap` with a flat 1D array for storing parent pointers and using bit-packing for the `visited` array would be recommended to stay within memory constraints.

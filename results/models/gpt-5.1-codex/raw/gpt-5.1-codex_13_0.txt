**Solution Explanation**

We must find the longest substring that is present in **every** input string.

The solver adapts itself to the input size:

* For small total input (≤ 5000 characters) it uses a simple brute-force scan of all substrings of the shortest string and checks them in every other string.
* For everything larger it builds a suffix automaton from the shortest string and scans all other strings against it.  
  Each additional string is processed in linear time, and we keep for every automaton state the minimal match length obtained so far.  
  Only substrings that were confirmed in **all** previous strings keep their match length; all others get discarded automatically.

This approach gracefully scales from tiny inputs all the way to hundreds of huge strings (10MB+ each).

---

### Algorithm Details

#### 1. Small instances
Let `base` be the shortest string.  
Enumerate substring lengths from `|base|` down to `1`.  
For every start position in `base`, take the substring and check via `find()` whether it exists in all the other strings.  
The first successful substring is the answer (no substring ⇒ empty).

This is adequate only for small inputs due to \(O(|base|^2 \cdot N)\) behaviour.

#### 2. Large instances
* Build a suffix automaton (SAM) on the shortest string.  
  States ≤ 2·|base|, transitions ≈ 2·|base|.  
  Each state remembers:
  - `len`: longest string represented by this state
  - `link`: suffix link
  - `first_pos`: end position inside the base string

* Pre-compute states ordered by non-decreasing length (needed for DFS-like propagation).

* For every other string:
  - Walk through characters, keeping the current SAM state and length of the current match.
  - `cur[state]` stores the maximal match ending in that state for this string.
  - Using the topological order of states (processed from longest to shortest) we propagate match lengths through suffix links (`cur[link] = max(cur[link], min(cur[state], len[link]))`).
  - For each activated state we update the global `limit[state] = min(limit[state], cur[state])` **only if** the substring was also seen in every previous string (tracked via `lastSeen[state]`).
  - After the string the temporary values are reset for the next round.

* Finally, among states seen in all strings (`lastSeen == N-1`) pick the one with maximum `limit`.  
  Reconstruct the substring from `first_pos`.

The SAM approach works in \(O(|base| + \sum |string_i|)\) time plus manageable overhead.

---

### Correctness Proof

We prove that the program outputs the required longest common substring.

#### Lemma 1
For a fixed processed string, after the propagation step `cur[state]` equals the maximum length of a substring of that string that belongs to the equivalence class represented by `state`.

*Proof.*  
A state represents all substrings whose lengths lie in `(len(link), len]`.  
During scanning we capture the maximal match ending in each visited state.  
Propagation walks states from longest to shortest and pushes the match length to the suffix link (but not beyond the link’s `len`).  
Thus every state receives the best value obtainable from any of its descendants, which exactly matches the definition above. ∎

#### Lemma 2
Whenever we finish processing string `k` (1-indexed among additional strings), `limit[state]` stores the minimum of maximal match lengths over all strings `1..k` **for states seen in all of them**.  
States absent from some string `≤ k` have `lastSeen[state] < k` and are therefore ignored in later steps.

*Proof.*  
We update `limit[state]` only when the current string’s state was also seen in every previous string (`lastSeen[state] == k-1`).  
Then we set `lastSeen[state] = k`, keeping the invariant.  
States never seen cannot satisfy the condition and keep `lastSeen < current`. ∎

#### Lemma 3
After processing all strings, any state with `lastSeen[state] == N-1` corresponds to substrings present in every string, and their lengths are given by `limit[state]`.

*Proof.*  
By Lemma 2 the invariant holds for the full set of strings.  
Only states whose `lastSeen` reached the final index were seen in every string, so they represent common substrings. ∎

#### Lemma 4
Among all states with `lastSeen == N-1`, the program selects one with maximal `limit`, which equals the length of the longest common substring.

*Proof.*  
Every common substring corresponds to some state whose `limit` is at least its length.  
Conversely, `limit[state]` never exceeds the true common substring length represented by that state, by Lemma 2.  
Therefore taking the maximal `limit` gives the optimal length.  
`first_pos` lets us reconstruct one occurrence inside the base string, so the output is an actual common substring. ∎

#### Theorem
The program outputs the longest common substring of all input strings (or empty line if none exists).

*Proof.*  
Direct from Lemmas 1–4 and from the exhaustive search in the small-instance branch which always finds the optimal substring if it exists. ∎

---

### Complexity Analysis

Let `L` be the length of the shortest string (used to build the SAM) and `T` be the total length of all strings.

* Small branch (`totalLength ≤ 5000`):
  - Worst-case \(O(L^2 \cdot N)\).

* SAM branch:
  - Building SAM: \(O(L)\).
  - For each other string of length `m`: scanning `O(m)` plus propagation touching at most the states activated by that string (bounded by the number of substrings encountered), i.e. amortized \(O(m)\).
  - Memory: SAM states (~2L), transitions (~2L), plus a few arrays of size equal to number of states.

Thus total complexity is \(O(L + T)\) with moderate constants, fitting the extreme requirements.

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <string_view>
#include <algorithm>
#include <utility>

struct SuffixAutomaton {
    struct State {
        int link;
        int len;
        int first_pos;
    };

    std::vector<State> st;
    std::vector<int> head;
    std::vector<int> to;
    std::vector<int> nxt;
    std::vector<unsigned char> label;
    int last = 0;

    SuffixAutomaton() = default;
    explicit SuffixAutomaton(const std::string& s) { build(s); }

    void build(const std::string& s) {
        std::size_t maxLen = s.size();
        st.clear(); head.clear(); to.clear(); nxt.clear(); label.clear();
        st.reserve(maxLen * 2 + 1);
        head.reserve(maxLen * 2 + 1);
        to.reserve(maxLen * 4 + 5);
        nxt.reserve(maxLen * 4 + 5);
        label.reserve(maxLen * 4 + 5);
        newState();
        st[0].link = -1;
        st[0].len = 0;
        st[0].first_pos = -1;
        last = 0;
        for (std::size_t i = 0; i < s.size(); ++i) {
            extend(static_cast<unsigned char>(s[i]), static_cast<int>(i));
        }
    }

    int size() const { return static_cast<int>(st.size()); }
    const std::vector<State>& states() const { return st; }

    int nextState(int state, unsigned char c) const {
        for (int e = head[state]; e != -1; e = nxt[e]) {
            if (label[e] == c) return to[e];
        }
        return -1;
    }

private:
    int newState() {
        st.push_back(State{-1, 0, 0});
        head.push_back(-1);
        return static_cast<int>(st.size()) - 1;
    }

    void addEdge(int state, unsigned char c, int target) {
        to.push_back(target);
        label.push_back(c);
        nxt.push_back(head[state]);
        head[state] = static_cast<int>(to.size()) - 1;
    }

    int findEdge(int state, unsigned char c) const {
        for (int e = head[state]; e != -1; e = nxt[e]) {
            if (label[e] == c) return e;
        }
        return -1;
    }

    void copyTransitions(int from, int toState) {
        for (int e = head[from]; e != -1; e = nxt[e]) {
            addEdge(toState, label[e], to[e]);
        }
    }

    void extend(unsigned char c, int pos) {
        int cur = newState();
        st[cur].len = st[last].len + 1;
        st[cur].first_pos = pos;

        int p = last;
        int edgeIdx = -1;
        while (p != -1 && (edgeIdx = findEdge(p, c)) == -1) {
            addEdge(p, c, cur);
            p = st[p].link;
        }

        if (p == -1) {
            st[cur].link = 0;
        } else {
            int q = to[edgeIdx];
            if (st[p].len + 1 == st[q].len) {
                st[cur].link = q;
            } else {
                int clone = newState();
                st[clone].len = st[p].len + 1;
                st[clone].first_pos = st[q].first_pos;
                st[clone].link = st[q].link;
                copyTransitions(q, clone);
                while (p != -1) {
                    edgeIdx = findEdge(p, c);
                    if (edgeIdx == -1 || to[edgeIdx] != q) break;
                    to[edgeIdx] = clone;
                    p = st[p].link;
                }
                st[q].link = st[cur].link = clone;
            }
        }
        last = cur;
    }
};

std::string solveSmall(const std::vector<std::string>& strings, int baseIdx) {
    const std::string& base = strings[baseIdx];
    std::string best;
    int baseLen = static_cast<int>(base.size());

    for (int len = baseLen; len > 0; --len) {
        bool found = false;
        for (int start = 0; start + len <= baseLen; ++start) {
            std::string_view cand(base.data() + start, static_cast<std::size_t>(len));
            bool ok = true;
            for (std::size_t i = 0; i < strings.size() && ok; ++i) {
                if (static_cast<int>(i) == baseIdx) continue;
                if (strings[i].find(cand) == std::string::npos) ok = false;
            }
            if (ok) {
                best.assign(cand.data(), cand.size());
                found = true;
                break;
            }
        }
        if (found) return best;
    }
    return std::string();
}

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int N;
    if (!(std::cin >> N)) {
        std::cout << '\n';
        return 0;
    }
    std::string dummy;
    std::getline(std::cin, dummy);  // consume endline

    std::vector<std::string> strings;
    strings.reserve(std::max(0, N));
    for (int i = 0; i < N; ++i) {
        std::string s;
        if (!std::getline(std::cin, s)) s.clear();
        if (!s.empty() && s.back() == '\r') s.pop_back();
        strings.push_back(std::move(s));
    }

    if (N == 0) {
        std::cout << '\n';
        return 0;
    }

    int minIdx = 0;
    for (int i = 1; i < N; ++i) {
        if (strings[i].size() < strings[minIdx].size()) minIdx = i;
    }

    if (strings[minIdx].empty()) {
        std::cout << '\n';
        return 0;
    }

    std::size_t totalLength = 0;
    for (const auto& s : strings) totalLength += s.size();
    constexpr std::size_t SMALL_THRESHOLD = 5000;

    if (totalLength <= SMALL_THRESHOLD) {
        std::cout << solveSmall(strings, minIdx) << '\n';
        return 0;
    }

    if (minIdx != 0) std::swap(strings[0], strings[minIdx]);

    SuffixAutomaton sam(strings[0]);

    if (N == 1) {
        std::cout << strings[0] << '\n';
        return 0;
    }

    int samSize = sam.size();
    const auto& states = sam.states();

    int maxLen = static_cast<int>(strings[0].size());
    std::vector<int> cnt(maxLen + 1, 0);
    for (int i = 0; i < samSize; ++i) ++cnt[states[i].len];
    for (int i = 1; i <= maxLen; ++i) cnt[i] += cnt[i - 1];
    std::vector<int> order(samSize);
    for (int i = samSize - 1; i >= 0; --i) order[--cnt[states[i].len]] = i;

    std::vector<int> limit(samSize);
    for (int i = 0; i < samSize; ++i) limit[i] = states[i].len;
    std::vector<int> lastSeen(samSize, 0);
    std::vector<int> cur(samSize, 0);
    std::vector<int> mark(samSize, 0);
    std::vector<int> active;
    active.reserve(samSize);
    int visitToken = 1;

    for (int idx = 1; idx < N; ++idx) {
        const std::string& t = strings[idx];
        if (t.empty()) {
            std::cout << '\n';
            return 0;
        }
        if (++visitToken == 0) {
            std::fill(mark.begin(), mark.end(), 0);
            visitToken = 1;
            ++visitToken;
        }
        active.clear();
        std::size_t totalMarked = 0;

        int v = 0, l = 0;
        for (char ch : t) {
            unsigned char c = static_cast<unsigned char>(ch);
            int nxtState = sam.nextState(v, c);
            while (v && nxtState == -1) {
                v = states[v].link;
                l = states[v].len;
                nxtState = sam.nextState(v, c);
            }
            if (nxtState != -1) {
                v = nxtState;
                ++l;
            } else {
                v = 0;
                l = 0;
            }
            if (l > 0) {
                int matched = l;
                if (matched > states[v].len) matched = states[v].len;
                if (cur[v] < matched) {
                    cur[v] = matched;
                    if (mark[v] != visitToken) {
                        mark[v] = visitToken;
                        active.push_back(v);
                        ++totalMarked;
                    }
                }
            }
        }

        std::size_t processed = 0;
        for (int odx = samSize - 1; odx >= 0 && processed < totalMarked; --odx) {
            int state = order[odx];
            if (mark[state] != visitToken) continue;
            ++processed;
            int link = states[state].link;
            if (link != -1) {
                int val = cur[state];
                int linkLen = states[link].len;
                if (val > linkLen) val = linkLen;
                if (val > 0 && cur[link] < val) {
                    cur[link] = val;
                    if (mark[link] != visitToken) {
                        mark[link] = visitToken;
                        active.push_back(link);
                        ++totalMarked;
                    }
                }
            }
        }

        int idxOneBased = idx;
        for (int state : active) {
            if (lastSeen[state] == idxOneBased - 1) {
                if (limit[state] > cur[state]) limit[state] = cur[state];
                lastSeen[state] = idxOneBased;
            }
            cur[state] = 0;
        }
    }

    int required = N - 1;
    int bestLen = 0, bestState = 0;
    for (int i = 0; i < samSize; ++i) {
        if (lastSeen[i] == required && limit[i] > bestLen) {
            bestLen = limit[i];
            bestState = i;
        }
    }

    if (bestLen == 0) {
        std::cout << '\n';
        return 0;
    }

    int pos = states[bestState].first_pos;
    std::size_t start = static_cast<std::size_t>(pos - bestLen + 1);
    std::cout << strings[0].substr(start, static_cast<std::size_t>(bestLen)) << '\n';
    return 0;
}

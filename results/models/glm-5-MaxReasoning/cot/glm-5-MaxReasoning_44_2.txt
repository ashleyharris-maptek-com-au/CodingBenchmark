The user wants me to write a GLSL fragment shader that visualizes texture coordinates as colors.

Requirements:
- GLSL 450 for Vulkan
- Visualize UV coordinates: R = uv.x, G = uv.y, B = 0, A = 1
- Use the provided inputs and UBO structure

This is a very simple task - I just need to output the UV coordinates as the RGB color.

Let me write the shader:

```glsl
#version 450

layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    outColor = vec4(uv.x, uv.y, 0.0, 1.0);
}
```

This is straightforward. The uv coordinates are already in [0,1] range according to the input description, so they can be used directly as color values.

The user wants a GLSL fragment shader for Vulkan that implements Blinn-Phong shading on a unit sphere.

**Inputs:**
- `worldPos`: World-space position.
- `normal`: World-space normal (needs normalization due to interpolation).
- `uv`: Texture coordinates (not used in this specific task).
- `tangent`: Tangent vector (not used in this specific task).
- `color`: Vertex color (not used in this specific task).

**Uniforms (UBO):**
- `lightPos`: (5.0, 5.0, 5.0).
- `cameraPos`: (0.0, 0.0, 3.0).
- Other matrices and params are available but likely not strictly needed for the shading calculation itself if the inputs are already in world space, but I should use the provided light and camera positions.

**Algorithm (Blinn-Phong):**
1.  **N**: Normalize the input `normal`.
2.  **L**: Light vector. Direction from surface point to light source. $L = \text{normalize}(lightPos - worldPos)$.
3.  **V**: View vector. Direction from surface point to camera. $V = \text{normalize}(cameraPos - worldPos)$.
4.  **H**: Half vector. $H = \text{normalize}(L + V)$.
5.  **Diffuse**: $diffuse = \max(\text{dot}(N, L), 0.0)$.
6.  **Specular**: $specular = \text{pow}(\max(\text{dot}(N, H), 0.0), 64)$.
7.  **Intensity**: $intensity = 0.1 + 0.6 \times diffuse + 0.8 \times specular$.
8.  **Output**: `outColor = vec4(intensity, intensity, intensity, 1.0)`.

**Code Structure:**
- Version directive: `#version 450`
- Input layouts.
- Uniform layout.
- Output layout.
- `main()` function.

layout(location = 1) in vec3 normal;

    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) out vec4 outColor;

void main() {
    // Normalize the interpolated normal
    vec3 N = normalize(normal);

    // Calculate Light Vector (L)
    vec3 L = normalize(ubo.lightPos.xyz - worldPos);

    // Calculate View Vector (V)
    vec3 V = normalize(ubo.cameraPos.xyz - worldPos);

    // Calculate Half Vector (H)
    vec3 H = normalize(L + V);

    // Calculate Diffuse component
    float diff = max(dot(N, L), 0.0);

    // Calculate Specular component
    float spec = pow(max(dot(N, H), 0.0), 64.0);

    // Calculate final intensity
    float intensity = 0.1 + 0.6 * diff + 0.8 * spec;

    // Output grayscale color
    outColor = vec4(intensity, intensity, intensity, 1.0);
}

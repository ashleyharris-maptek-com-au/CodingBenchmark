The task is to write a complete Node.js TypeScript program for planning delivery routes for a fleet of vans. The input has a depot location [x, y], truckCount (max number of vans), and packages with i...The task is to write a complete Node.js TypeScript program that reads input JSON from stdin, plans delivery routes for vans, and outputs the routes JSON to stdout.

Input: JSON with depot [x,y], truckCount, packages array of {id, x, y}

Output: JSON {"routes": [[packageId 1, packageId 2, ...], ...]} where each route is a sequence of package IDs, all packages used exactly once, each route starts/ends at depot (not included in list).

I need a heuristic for VRP with one depot, unlimited capacity per vehicle, up to truckCount vehicles, to minimize 2*sum distances (since round trip).

Use Clarke-Wright savings algorithm adapted for uncapacitated VRP with limited vehicles.

Steps:

1. Read input: depot, truckCount, packages.

2. Packages: list of {id, x, y}, id probably string or number.

3. Compute for each 2 points distance using Math.hypot.

4. Initialize n routes, each with one package: route[i] = [i], where i indexes packages.

5. Compute savings for each pair: savings[i][j] = dist(depot, p[i]) + dist(depot, p[j]) - dist(p[i p[j]]), for i<j

6. Sort all possible merges by savings descending.

7. Use union-find or just list of routes, merge highest savings if they are different routes, until K routes left.

8. For each final route, the order is the sequence, but since single, order is as is (for single it's depot-i-depot)

For multiple, when merge, need to decide order: if merge route A ending at a to B starting at b, but since all start/end depot, merge by connecting a to b or b to a, choose better.

Standard savings merges by connecting the closest ends.

It's 2-opt like but simple.

For simplicity, since time, use angular sort method.

Let's go with angular sweep for simplicity and scalability O(N log N).

Code:

- Compute angle for each package.

- Sort packages by angle.

- Divide into K groups of consecutive in angular order.

The autopilot solution has been implemented in `autopilot.py` with robust sensor fusion and PID control logic. A basic verification script `test_autopilot.py` confirms that the function executes as expected.

### Key Features:
- **Sensor Fusion:** Fuses wheel speeds and vision-based lane offsets using simple valid-data filtering and averaging.
- **Control Logic:** Implements PID controllers for steering and speed control.
- **Safety:** Includes basic obstacle avoidance (braking) and traction management based on `estimated_mu`.
- **Persistence:** Uses a module-level dictionary to maintain PID state between function calls.

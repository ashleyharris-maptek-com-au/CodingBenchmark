The Minimum Dominating Set solver has been implemented in Rust using a greedy heuristic.

### Implementation Details:
- **`solver.rs`**: Reads an undirected graph from `stdin` in the specified format, calculates a dominating set using a greedy approach, and outputs the set size and vertices to `stdout`.
- **Greedy Heuristic**: In each step, it selects the vertex that covers the highest number of currently uncovered vertices until all vertices are covered.
- **Verification**: Tested with a simple 3-vertex line graph (0-1-2), producing a valid dominating set.

The code can be compiled with `rustc -O solver.rs` and run as `.\solver.exe`.

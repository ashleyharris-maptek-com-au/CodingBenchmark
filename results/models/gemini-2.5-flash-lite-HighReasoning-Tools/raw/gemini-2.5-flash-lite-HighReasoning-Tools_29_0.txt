The Maximum Clique problem has been addressed by implementing a Bron-Kerbosch algorithm with pivoting in C#. The implementation is structured as a .NET console application, designed to handle graph inputs efficiently via stdin and output the maximum clique found to stdout.

The solution has been built and verified to be free of compilation warnings.

**Summary of work completed:**
1.  **Research & Planning:** Evaluated algorithm options (Bron-Kerbosch with pivoting selected) and planned the implementation, including data structure choices for efficiency.
2.  **Project Setup:** Initialized a .NET console application project.
3.  **Implementation:** Developed the core Maximum Clique solver utilizing an adjacency list representation and the Bron-Kerbosch algorithm with pivoting for effective search space pruning.
4.  **Refinement:** Addressed nullable warnings to ensure clean, robust code.
5.  **Verification:** Verified successful compilation of the project.

Implemented a scalable C++20 rectangle bin packing solver in `solver.cpp`.

Algorithm:
- Greedy First-Fit Decreasing by area.
- Per board placement uses a MaxRects free-rectangle list with Best Short Side Fit, allowing 90° rotation.
- For N <= 5000, applies a small tail-repacking improvement pass (~2s) to reduce board count when possible.

Input: stdin as specified (N board_w board_h, then N lines w h)
Output: stdout as specified (num_boards, then N lines: board_index x y rotated in input order)

Note: The environment here doesn’t have `cl`/`g++` available on PATH, so I couldn’t compile/run locally, but the code is standard-library-only and MSVC-compatible (no nonstandard headers).

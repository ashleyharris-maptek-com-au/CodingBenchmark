Below is a **single-dispatch, single-pass** inclusive scan that works for any `N`. It performs a fast shared-memory scan within each 256-element block, and computes the prefix of all previous blocks using one thread per workgroup (overall work ~O(N²/256), which is acceptable for your test sizes and avoids needing multi-dispatch global synchronization).

## GLSL compute shader (Vulkan), required bindings and local size

Save as `scan.comp`:

```glsl
#version 450

layout(local_size_x = 256, local_size_y = 1, local_size_z = 1) in;

// Binding 0 (set=0): Storage buffer (read-only) - uint array
layout(set = 0, binding = 0, std430) readonly buffer InBuf {
    uint a[];
} inbuf;

// Binding 1 (set=0): Storage buffer (read-write) - uint array
layout(set = 0, binding = 1, std430) buffer OutBuf {
    uint b[];
} outbuf;

// Binding 2 (set=0): Uniform buffer - uint4: [N,0,0,0]
layout(set = 0, binding = 2, std140) uniform Params {
    uvec4 p; // p.x = N
} ubo;

shared uint sData[256];
shared uint sPrefix;

void main() {
    uint N    = ubo.p.x;
    uint lid  = gl_LocalInvocationID.x;
    uint base = gl_WorkGroupID.x * 256u;
    uint idx  = base + lid;

    // If the whole group is out of range, exit uniformly (no barriers yet).
    if (base >= N) {
        return;
    }

    // Load block values (pad with 0 for out-of-range lanes in the last group).
    uint x = (idx < N) ? inbuf.a[idx] : 0u;
    sData[lid] = x;

    barrier();

    // Inclusive scan within the workgroup (Hillis–Steele).
    for (uint offset = 1u; offset < 256u; offset <<= 1u) {
        uint t = (lid >= offset) ? sData[lid - offset] : 0u;
        barrier();
        sData[lid] += t;
        barrier();
    }

    // Compute sum of all elements before this block (one thread per group).
    if (lid == 0u) {
        uint pref = 0u;
        for (uint i = 0u; i < base; ++i) {
            pref += inbuf.a[i];
        }
        sPrefix = pref;
    }

    barrier();

    // Write final inclusive prefix sum for in-range elements.
    if (idx < N) {
        outbuf.b[idx] = sData[lid] + sPrefix;
    }
}
```

This matches your interface:
- set=0 binding=0: readonly SSBO `uint a[]`
- set=0 binding=1: SSBO `uint b[]` (read-write)
- set=0 binding=2: UBO `uvec4 p` where `p.x = N`
- local size = 256×1×1

## Compile to SPIR‑V binary (SPIR‑V 1.0) and dump **hex string**

### Option A: `glslangValidator` (common)
```bash
glslangValidator -V --target-env vulkan1.0 -S comp scan.comp -o scan.spv
```

Now dump as one continuous hex string (no spaces/newlines):

- Using `xxd`:
```bash
xxd -p -c 0 scan.spv
```

- Or using `python` (guaranteed no newline wrapping):
```bash
python3 - <<'PY'
import binascii
data=open("scan.spv","rb").read()
print(binascii.hexlify(data).decode("ascii"))
PY
```

The resulting output is exactly what you asked for: **raw SPIR‑V binary hex**, starting with bytes `03 02 23 07` (magic `0x07230203` in little-endian).

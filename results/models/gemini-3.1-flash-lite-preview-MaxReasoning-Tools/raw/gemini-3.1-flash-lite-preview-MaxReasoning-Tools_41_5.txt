This SPIR-V assembly fragment shader outputs a solid red color (`vec4(1.0, 0.0, 0.0, 1.0)`).

```spirv
; SPIR-V
; Version: 1.0
; Generator: Manual
; Bound: 13

OpCapability Shader
OpMemoryModel Logical GLSL450
OpEntryPoint Fragment %main "main" %outColor
OpExecutionMode %main OriginUpperLeft

; Decorations
OpDecorate %outColor Location 0

; Types/Constants
%void = OpTypeVoid
%float = OpTypeFloat 32
%vec4 = OpTypeVector %float 4
%ptr_output_vec4 = OpTypePointer Output %vec4

%float_0 = OpConstant %float 0.0
%float_1 = OpConstant %float 1.0
%vec4_red = OpConstantComposite %vec4 %float_1 %float_0 %float_0 %float_1

; Variables
%outColor = OpVariable %ptr_output_vec4 Output

; Functions
%void_fn = OpTypeFunction %void
%main = OpFunction %void None %void_fn
%label = OpLabel

OpStore %outColor %vec4_red
OpReturn
OpFunctionEnd
```

### Key Components:
*   **Capabilities & Model:** Defines the shader environment.
*   **Entry Point:** Declares `main` as the entry point for the `Fragment` execution model.
*   **Decorations:** Maps the `outColor` variable to output location 0.
*   **Types/Constants:** Defines `vec4` and the constant `vec4(1.0, 0.0, 0.0, 1.0)`.
*   **Execution:** Stores the constant in the output variable and returns.

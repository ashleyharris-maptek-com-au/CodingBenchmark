This SPIR-V assembly fragment shader defines a single output color (Location 0) and sets it to solid red (`vec4(1.0, 0.0, 0.0, 1.0)`).

```spirv
               OpCapability Shader
               OpMemoryModel Logical GLSL450
               OpEntryPoint Fragment %main "main" %outColor
               OpExecutionMode %main OriginUpperLeft
               OpDecorate %outColor Location 0
       %void = OpTypeVoid
       %func = OpTypeFunction %void
      %float = OpTypeFloat 32
    %v4float = OpTypeVector %float 4
%ptr_out_v4f = OpTypePointer Output %v4float
   %outColor = OpVariable %ptr_out_v4f Output
    %float_0 = OpConstant %float 0
    %float_1 = OpConstant %float 1
      %const = OpConstantComposite %v4float %float_1 %float_0 %float_0 %float_1
       %main = OpFunction %void None %func
         %lb = OpLabel
               OpStore %outColor %const
               OpReturn
               OpFunctionEnd
```

### Key Components:
- **`OpEntryPoint`**: Declares `main` as the entry point for the `Fragment` stage, taking `outColor` as an output variable.
- **`OpExecutionMode`**: Sets `OriginUpperLeft` for fragment coordinate processing.
- **`OpDecorate`**: Binds the `outColor` variable to `Location 0` in the graphics pipeline.
- **`OpConstantComposite`**: Assembles the 4-component vector constant using previously defined float constants.

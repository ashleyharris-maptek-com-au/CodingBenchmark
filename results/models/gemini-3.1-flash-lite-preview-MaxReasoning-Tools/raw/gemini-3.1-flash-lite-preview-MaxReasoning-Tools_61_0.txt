The benchmark task requires a Node.js TypeScript program to plan vehicle delivery routes. I have implemented a greedy nearest-neighbor algorithm that effectively assigns packages to the available fleet to minimize overall distance while ensuring all packages are delivered.

The solution adheres to the following:
1.  **Input/Output:** Reads JSON from stdin and writes JSON to stdout, as requested.
2.  **Scalability:** Uses a set for remaining packages and iterative route assignment, which performs well as the number of packages increases.
3.  **Dependencies:** Uses standard Node.js libraries (`fs`) without external dependencies.
4.  **Language:** Written in TypeScript.

The implementation is contained in `solution.ts`.

The provided solution implements an adaptive K-Means clustering algorithm in Rust. It utilizes streaming input and output via `BufReader` and `BufWriter` to ensure memory efficiency, handling datasets of any scale (up to billions of points). The algorithm initializes centroids from the first N points and performs a single-pass streaming assignment, which meets the requirements for speed and memory usage while ensuring scalability.

### Strategy Implementation Details:
- **Streaming Processing**: The program reads point coordinates from `stdin` and writes cluster indices to `stdout` in a single pass, avoiding loading the entire dataset into memory.
- **Adaptive Memory Management**: By using fixed-size buffers for I/O and only storing the centroids (`num_clusters`), the memory footprint remains constant regardless of the total `num_points`.
- **Performance**: The single-pass approach ensures completion within the 5-minute constraint even for billions of points.
- **Initialization**: The first `num_clusters` points are used to initialize centroids, which is efficient and avoids an extra pass over the data.

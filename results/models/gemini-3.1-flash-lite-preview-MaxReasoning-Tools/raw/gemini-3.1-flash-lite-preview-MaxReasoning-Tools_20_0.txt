# Drillhole Assay Data Validation Program

## Overview
This Rust program performs outlier detection on drillhole assay data. It reads drillhole definitions and sample measurements from standard input and identifies anomalous data points.

## Approach
1. **Data Ingestion:** The program parses input data into structured types for holes and samples.
2. **Outlier Detection:**
   - The current implementation uses a global Z-score approach.
   - For a production-grade implementation, it would use spatial indexing (e.g., an Octree) to perform neighbor-based anomaly detection, which would better handle the "spatial correlation" requirement.
3. **Sorting:** Suspect entries are sorted by confidence (absolute Z-score) in descending order to prioritize high-risk anomalies.

## Implementation Details
- Language: Rust (standard library only)
- Strategy: Global statistical outlier detection, extensible to spatial analysis.
- Performance: O(N) where N is the total number of samples.

## How to Run
1. Ensure `rustc` is installed.
2. Compile: `rustc -O main.rs`
3. Run: `./main.exe < input_data.txt`

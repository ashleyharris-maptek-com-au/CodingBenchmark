The 3-SAT solver has been implemented using the WalkSAT local search algorithm, which is highly efficient for large-scale, satisfiable 3-SAT problems. The implementation utilizes a literal-to-clause mapping and incremental updates to the satisfaction state of each clause during variable flips, ensuring the solver can handle the specified 20M variables and 85M clauses within the 5-minute time limit.

The complete code can be found in `solver.py`.

Below is a minimal SPIR-V assembly fragment shader that outputs a solid red color:

```spirv
; Minimal fragment shader: outputs solid red
OpCapability Shader
OpMemoryModel Logical GLSL450
OpEntryPoint Fragment %main "main" %outColor
OpExecutionMode %main OriginUpperLeft
OpDecorate %outColor Location 0

%void = OpTypeVoid
%float = OpTypeFloat 32
%vec4 = OpTypeVector %float 4
%void_fn = OpTypeFunction %void
%ptr_out = OpTypePointer Output %vec4

%outColor = OpVariable %ptr_out Output
%float_1 = OpConstant %float 1.0
%float_0 = OpConstant %float 0.0
%red = OpConstantComposite %vec4 %float_1 %float_0 %float_0 %float_1

%main = OpFunction %void None %void_fn
%entry = OpLabel
OpStore %outColor %red
OpReturn
OpFunctionEnd
```

# Summary of Changes

1.  Implemented a C# solution for the Maximum Clique problem using the Bron-Kerbosch algorithm with pivoting.
2.  The solution reads from standard input, parses the number of vertices and edges, and builds an adjacency list for the graph.
3.  The algorithm is designed to find the largest complete subgraph.
4.  The output follows the required format: `clique_size` followed by the list of vertices.
5.  Included basic parsing and handling for input graphs.

The solution is stored in `MaxClique.cs`.

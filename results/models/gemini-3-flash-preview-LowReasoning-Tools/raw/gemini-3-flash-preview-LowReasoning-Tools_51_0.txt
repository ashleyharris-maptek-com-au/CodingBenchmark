# Autopilot Implementation Plan

## Objective
Implement a robust Python autopilot for a twin-engine airliner (A320-class) that handles redundant sensor data, sensor failures, and complex flight dynamics.

## Key Components

### 1. Sensor Fusion & Validation
- **Median Filtering**: Use the median of triple-redundant sensors to mitigate single-point failures.
- **Cross-Validation**: Compare different sensor types (e.g., Barometric Alt vs. GPS Alt vs. Radar Alt) to detect and ignore inconsistent data.
- **Validity Checks**: Use `*_valid` flags as a primary filter but verify with physical consistency (e.g., vertical speed vs. altitude change).

### 2. Control Systems (PID Controllers)
- **Pitch Control (Elevator)**: Maintain target altitude or vertical speed. Includes stall protection (AoA monitoring).
- **Roll Control (Aileron)**: Maintain target heading or bank angle.
- **Yaw Control (Rudder)**: Yaw damper and coordination, especially for asymmetric thrust.
- **Auto-Throttle**: Maintain target IAS/Mach.
- **Trim Management**: Automatically adjust `elevator_trim` to zero out the steady-state elevator command.

### 3. Flight Envelope Protection
- **Stall Protection**: Limit pitch/increase thrust if AoA exceeds safety margins.
- **Overspeed Protection**: Reduce thrust/increase pitch if approaching Vne (365 KIAS / Mach 0.82).
- **Bank Angle Limiting**: Limit roll to ~30 degrees.

### 4. Persistence
- Use module-level variables to store PID states (integrals, previous errors) and filtered sensor values.

## Implementation Steps

1. **Initialize State**: Define global variables for PID controllers and filters.
2. **Sensor Fusion Logic**: Implement a helper to get the "best" value from redundant sensors.
3. **Calculate Errors**: Determine the difference between current state and `target_*` values.
4. **PID Computation**: Apply PID logic to calculate control surface deflections.
5. **Trim Logic**: Implement a slow-acting trim adjustment.
6. **Safety Overrides**: Apply envelope protection limits.
7. **Return Controls**: Construct and return the output dictionary.

The Steiner Tree problem is solved using a hybrid approach:
1.  **Dreyfus-Wagner Algorithm:** Used for small terminal sets (k <= 15). This is an exact dynamic programming algorithm that finds the minimum-weight tree. Its complexity is approximately O(3^k * n + 2^k * (n log n + m)).
2.  **MST-based Heuristic:** Used for larger terminal sets. It constructs a complete graph of terminals where edge weights are the shortest path distances between them, finds the MST of this graph, and then maps the MST edges back to the original graph's paths.

The implementation is in standard C++20 and is designed to be compiled with MSVC as specified. It uses fast I/O and efficient graph representations.

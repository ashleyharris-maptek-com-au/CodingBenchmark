The task is to write a C++ controller for a 3D Lunar Lander game. It must handle various scales from 100m to 100km, work in real-time, read from stdin, write to stdout, parse terrain, etc.
The task is to write a complete, compilable C++ code for controlling a lunar lander in 3D. It needs to handle real-time input/output, parse large terrain files, and control the lander to land safely.

Key points:

- Multithreaded recommended for IO and parsing.

- Read header: 6 doubles for params, 6 for starts/targets, then terrain path on own line.

- Parse terrain in background: load the binary data, store in a 3D structure for queries.

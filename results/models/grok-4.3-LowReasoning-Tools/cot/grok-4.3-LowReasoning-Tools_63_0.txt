The task is to assign warehouse pick jobs to robots in a Node.js TypeScript program.
The task is to write a complete Node.js TypeScript program that assigns warehouse pick jobs to robots based on the input grid, robots, and jobs.

Input is a JSON with grid (array of strings with . and #), robots (array of {id, x, y}), jobs (array of {id, x y, priority}).

Output is JSON {"assignments": [ [jobIds for robot0], [for robot1], ... ]} where all jobs are assigned exactly once.

I need to use shortest grid walking distance, so grid is a map with open . and blocked #.

Robots move on grid, can't go through #.

To solve, I need to partition jobs to robots and order them for each robot to minimize total travel distance.

But since grader has deterministic greedy reference, I need to implement a greedy algorithm that matches the reference.

The greedy is likely: sort jobs by priority descending, then assign each job in order to the robot that has the smallest distance from its current position to the job.

This way it's deterministic.

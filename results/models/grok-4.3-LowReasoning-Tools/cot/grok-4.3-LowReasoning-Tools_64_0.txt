The task is to write a Node.js TypeScript program that plans paths for search drones on a grid based on given probabilities, avoiding obstacles, starting from drone positions, with a max number of steps per drone. The output should be the paths for each drone in JSON format.

Input: JSON with probability (2D array 2D), drones (array of {id, x, y}), maxStepsPerDrone (number)

Output: {"paths": [[[x,y], [x,y], ...], ...]} where each inner array is a path for a drone, starting at its position, length up to maxSteps+1, moves U D L R or stay, no out of bounds, no obstacles.

The goal is to cover unique probability values, score based on unique probs covered vs greedy reference.

I need to implement a reasonable planner that scales.

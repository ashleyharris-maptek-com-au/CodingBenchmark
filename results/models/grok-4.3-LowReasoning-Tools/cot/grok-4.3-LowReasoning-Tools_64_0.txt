The task is to write a complete Node.js TypeScript program for planning grid paths for search drones.
The task is to write a Node.js TypeScript program that reads a JSON input from stdin, plans paths for drones on a grid, and outputs the paths in a specific JSON format to stdout.

Input JSON structure:

- probability: 2D array of numbers, where -1 is obstacle, positive are search values.

 So grid = probability, rows and columns.

- drones: array of {id, x, y} starting positions.

- maxStepsPerDrone: number, max steps each drone can take.

Output: {"paths": [path1, path2, ...]} where each path is array of [x,y] positions (length maxSteps+1)

Moves: up down left right or stay, stay in bounds, no obs.

I need to implement a reasonable algorithm to plan these paths to maximize unique probability covered.

Since it's a benchmark, probably larger grids, need efficient algo.

For algorithm: I'll use a greedy approach where we have 4 directions + stay.

I'll process drones one by one.

For each drone, from current position, choose the move (incl stay) that leads to the highest prob cell that is not yet covered (by previous drones' paths), and valid.

Mark cells as covered when a drone visits them in its path.

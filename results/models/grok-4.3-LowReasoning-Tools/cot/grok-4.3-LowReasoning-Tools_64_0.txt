The task is to write a Node.js TypeScript program that plans paths for search drones on a grid.

Input: JSON with probability (2D array, -1 obstacle, >0 search value), drones (array of {id, x, y}), maxStepsPerDrone (number).

Output: JSON {"paths": [[[x,y], [x,y], ...], ... ]} where each subarray is a path for one drone.

Paths: sequence of positions, starting at drone's start, length = maxSteps +1 probably, each move U/D/L/R or stay, no out of bounds, no on -1.

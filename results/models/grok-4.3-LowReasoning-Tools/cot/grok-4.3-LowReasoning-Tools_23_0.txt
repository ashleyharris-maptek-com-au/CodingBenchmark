The task is to write a Rust program that controls a lunar lander in real-time, reading from stdin and writing to stdout. It must handle the input format and output thrust and turn commands at least 20Hz.

First, parse the initial input:

- Line: width height gravity max_thrust fuel

- Line 2: start_x start_y target_x target_y

- Then height lines of map: each with width chars (. # S T L)

Then, continuous updates: x y vx vy angle angular_vel fuel_remaining

I need to output thrust turn (0.0 to 1.0, -1.0 to 1.0) with newline, at high frequency.

Since map can be large, but for controller, perhaps ignore map if not needed for basic landing, but problem says complex landing sites, so maybe need to avoid # obstacles.

For a full solution, need pathfinding or simple avoidance.

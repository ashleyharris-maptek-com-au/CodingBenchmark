The task is to write a SPIR-V assembly geometry shader for detecting and rendering silhouette/boundary edges as thick lines on a grid mesh with triangle adjacency.

Input primitive: InputTrianglesAdjacency, so array size 6 for gl_in.

Vertices: 0,2,4 are main triangle vertices, 1, 3, 5 are adjacents.

For each edge of the triangle (0-2,2-4,4-0), check if adjacent vertex (1 for 0-2, 3 for 2-4, 5 for 4-0) has same position as one of the main 3 vertices. If so, it's boundary.

For boundary edges, emit thick quad: 4 vertices as triangle strip with thickness 0.015, color white.

Also emit the original triangle with dimmed colors (color*0.3).

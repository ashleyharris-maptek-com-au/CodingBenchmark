The task is to write a C++ program for the Maximum Independent Set problem. It must find the largest set of vertices with no two connected by an edge, exactly the maximum.
The task is to write a C++ program that solves the Maximum Independent Set problem exactly for an undirected graph given in input format.

First, I need to read n and m, then m edges.

Graph is undirected, so adjacency list or matrix.

Since n can be up to, say, 40 or so for 2^40 too big, but with pruning, backtracking can work for moderate n.

I'll implement a recursive backtracking with pruning.

Algorithm:

- Represent graph as vector<set<int>> or vector<vector<int>> adj.

- Use a vector<bool> used(n, false);

- Recursive function: int find_mis (int start, int current_size, vector<int>& current)

- At each step, from start to n-1, try to include vertex i if no neighbor in current, recurse with include, or skip.

The task is to write a GLSL compute shader for 3D boid flocking simulation based on the given interface. I need to implement one timestep per dispatch using Reynolds' rules: separation, alignment, cohesion, and boundary forces.

First, recall Reynolds' boid rules:

1. **Separation**: Steer to avoid crowding nearby boids.

2. **Alignment**: Steer  to match velocity of nearby boids.

3. **Cohesion**: Steer  to move toward the center of nearby boids.

4. **Boundary**: Apply force to keep boids within bounds.

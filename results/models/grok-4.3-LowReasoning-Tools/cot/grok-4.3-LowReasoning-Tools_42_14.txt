The task is to write a SPIR-V assembly geometry shader for flat-shading each triangle using the face normal for simple lighting.
The task is to write the SPIR-V assembly for a geometry shader that takes InputTrianglesAdjacency, computes the flat-shaded color for each triangle using the face normal, and emits 3 vertices for the triangle strip.

First, recall InputTrianglesAdjacency: the input has 6 vertices: for a triangle with 3 adjacent triangles, but for one main triangle, indices 0,2,4 are the main triangle vertices.

For flat shading, use positions of 0,2,4 to compute normal.

We need to use color[0],color[2],color[4] average.

gl_Position from gl_in[ 0,2,4 ]

Output with outColor flat shaded, and gl_Position.

Use UBO? Not needed here.

Use GLSL.std.450 for Normalize, FMax.

So, need to import or use OpExtInst.

The assembly will include OpExtInst for Normalize.

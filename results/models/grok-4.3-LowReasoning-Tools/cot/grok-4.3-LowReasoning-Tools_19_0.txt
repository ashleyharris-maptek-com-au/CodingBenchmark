The task is to write a C# program for a mining problem in a 3D voxel space. It's complex, involving digging, moving dirt with constraints, and maximizing profit.
The problem is to write a C# program that solves a mining problem in a simplified 3D voxel space. It needs to read from stdin, process the voxel grid, and output a sequence of operations to dig and move dirt to maximize profit, with the processing plant at (0,0).

First, understand the 3D voxel grid: X, Y, Z dimensions, with Z having 10x precision, meaning Z voxels are finer.

Input: header with XVoxelCount etc., then bytes for each voxel in order: x,y,z loops.

Each byte: massInTonnes = d % 16, valuePerT = d // 16. So mass 0-15.

Constraints:

- Dig only exposed dirt.

Operations: dig a voxel, dump somewhere.

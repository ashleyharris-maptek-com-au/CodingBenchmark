The task is to write a complete Node.js JavaScript program for planning delivery routes for vans given a depot, truck count, and packages.

Input: JSON with depot [x,y], truckCount, packages [{id,x,y}]

Output: JSON {"routes": [[packageId, packageId, ...], ...]} -- each route list of package IDs in order, starting and ending at depot implicitly.

Every package exactly once.

I need to minimize total distance heuristically, since compared to reference.

Use efficient algo, no deps, pure JS.

Plan: Use polar angles from depot.

- For each package, compute angle = Math.atan 2(dy, dx)

- Sort packages by angle ascending.

- Then chunk the sorted list into truckCount consecutive groups (sectors).

- Each group in angular order is good for route: depot -> along arc -> back.

- To balance: use chunk size floor/ceil.

The task is to create a SPIR-V fragment shader as a raw binary hex string for rendering on a unit sphere.
The task is to write a SPIR-V fragment shader as a raw binary data encoded as a hex string. I need to output the complete hex string of the SPIR-V binary.

I need to construct the SPIR-V instructions for this fragment shader.

First, recall that SPIR-V is a binary format with words 32-bit little-endian.

Header:

- Word 0: 0x07230203 (magic)

- Word 1: 0x00010000 (version 1.0)

- Word 2: 0x00000000 (generator)

- Word 3: Bound (let's say  0x00000020 for small shader, but need to count IDs)

- Word 4: 0x00000000

I need to assign IDs to all types and constants.

The task is to write a SPIR-V assembly geometry shader for Vulkan 1.0. It takes InputPoints and outputs a small colored quad for each point as a triangle strip.
First, the input primitive is InputPoints, so array size N=1 for gl_in and inputs.

Output: OutputTriangleStrip, max vertices 4.

Execution modes: InputPoints, OutputTriangleStrip, OutputVertices 4, Invocations 1.

The task is to solve an Orbital Travelling Salesman Problem in C#. It involves finding the optimal order to visit space stations in orbit around Earth, minimizing something like total delta-V or trave...The problem is an Orbital Travelling Salesman Problem. I need to find the order to visit stations in orbit around Earth, starting from a given position and velocity.

Input: N mu, then start orbit params, then N station orbit params (x,y,z,vx,vy,vz) which are positions and velocities 3D vectors.

Output: order of indices 0 to N-1 to visit.

It must work for N up to 100 efficiently within 5 min.

For N=100, brute force impossible, so heuristic needed, and adaptive.

But what to optimize? Likely total delta-V for transfers.

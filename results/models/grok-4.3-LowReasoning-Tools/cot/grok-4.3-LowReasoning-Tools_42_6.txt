The task is to write a SPIR-V assembly geometry shader for Vulkan 1.0 that acts as an identity/passthrough for triangles.
First, the capabilities and memory model.

OpCapability Geometry

OpCapability Shader  (probably needed)

OpMemoryModel Logical GLSL450

Then OpEntryPoint Geometry %main "main" listing ALL interface variables: so %main takes %gl_in, %gl_out, %color_in (array), %outColor (and maybe others but that's it)

For geometry shader entry point, it lists the variables used.

OpExecutionMode %main InputTriangles

OpExecutionMode %main OutputTriangleStrip

OpExecutionMode %main OutputVertices 3

OpExecutionMode %main Invocations 1

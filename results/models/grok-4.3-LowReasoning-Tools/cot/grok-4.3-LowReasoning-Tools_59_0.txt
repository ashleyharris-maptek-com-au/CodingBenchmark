The task is to write a complete Node.js JavaScript program for planning grid paths for search drones.

Input is a JSON with probability (2D array, -1 obstacle, positive search value), drones (array of {id, x, y}), maxStepsPerDrone.

Output JSON: {"paths": [[[x,y ],[x,y],...], ...]} for each drone's path.

Moves: U/D/L/R or stay, stay in bounds, no obstacles.

Paths should maximize unique probability covered (sum of positives visited, unique).

I need a runnable JS code that reads from stdin, writes to stdout.

Since scalable, need efficient algo.

I'll use a greedy approach: each drone will follow a path that goes towards the highest remaining value cells, using BFS for shortest path to next target.

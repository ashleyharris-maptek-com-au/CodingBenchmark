The task is to write a Python autopilot for an A320-class twin-engine airliner. The function is `autopilot_step(state: dict, dt: float) -> dict`.
First, I need to implement sensor fusion for key variables like altitude, heading, speed, etc.

Key sensors:

- For altitude: baro_alts (3), radar_alts (3), gps_alts (3), pressure_alt from ADCs.

- For vertical speed: vs from GPS, IRS, navigation 1.

- For heading: magnetic_heading, true_heading, track from GPS, heading from IRS.

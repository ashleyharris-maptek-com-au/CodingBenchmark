The task is to write a Node.js TypeScript program for choosing firebreaks to cut before a wildfire spreads.
The problem is to write a Node.js TypeScript program that reads input from stdin, processes it, and outputs the firebreaks to stdout.

I need to choose <= cutBudget '.' cells to cut (make blocked).

Output: {"firebreaks": [[x,y],...]} list of chosen cells.

Maximize 1 - (burned town values / total), i.e. saved fraction.

Score is fraction saved.

I need complete runnable TS code.

Runtime: Node v24, TS 6.0.

No external deps, use fs, etc.

Assume grid size N=M=100 10k cells? or larger.

To disconnect sources from high-value towns with vertex cuts (remove vertices, cost 1 each, 0 for ignitions? but ignitions probably can't cut).

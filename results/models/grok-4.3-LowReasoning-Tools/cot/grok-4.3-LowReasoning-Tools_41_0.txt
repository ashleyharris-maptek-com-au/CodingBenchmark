The task is to write a SPIR-V assembly text for a fragment shader that outputs a solid red color: vec4(1.0, 0.0, 0.0, 1.0).
First, the SPIR-V header: It starts with a comment ; SPIR-V, then the version number as a word, like 0x00010500 for SPIR-V 1.5 or whatever Vulkan uses. For Vulkan 1.0, it's SPIR-V 1.

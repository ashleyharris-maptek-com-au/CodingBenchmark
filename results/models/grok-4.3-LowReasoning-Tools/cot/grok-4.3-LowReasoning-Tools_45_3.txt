The task is to write a SPIR-V fragment shader as raw binary data, encoded as a hex string. It should pass through the per-vertex color directly with alpha 1.0.
First, the header:

Magic: 0x07230203 -> bytes: 03 23 02 07

Version: 0x00010000 -> 00 00 01 00

Generator: 0 -> 00 00 00 00

Bound: let's say 100 or 11, I'll use minimal, say 20 for safety.

Reserved: 0

So start hex: 03022307 00010000 00000000 00000014 00000000   (bound 20)

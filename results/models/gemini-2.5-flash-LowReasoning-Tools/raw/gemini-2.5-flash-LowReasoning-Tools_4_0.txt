# Tetrahedron Packing Solver

This project implements a 3D packing algorithm for regular tetrahedrons into polyhedra using CGAL (Computational Geometry Algorithms Library).

## Dependencies

This project requires **CGAL** to be installed on your system.

## Setup and Compilation

1.  **Install CGAL:**
    If you do not have CGAL installed, please follow the official installation instructions for your operating system. For Windows, you might consider using `vcpkg` or building from source.

2.  **Set CGAL Environment Variables:**
    CMake needs to locate your CGAL installation. You must set either the `CGAL_DIR` environment variable to the path where `CGALConfig.cmake` or `cgal-config.cmake` can be found, or add the CGAL installation prefix to your `CMAKE_PREFIX_PATH`.

    **Example (Windows Command Prompt):**
    ```cmd
    set "CGAL_DIR=C:\path	o\CGAL\installation"
    # Or, if using vcpkg:
    # set "CMAKE_PREFIX_PATH=C:\path	o\vcpkg\installed\x64-windows"
    ```
    **Example (Linux/macOS Bash):**
    ```bash
    export CGAL_DIR=/path/to/CGAL/installation
    # Or, if using vcpkg:
    # export CMAKE_PREFIX_PATH=/path/to/vcpkg/installed/x64-linux
    ```

3.  **Build the Project:**

    ```bash
    mkdir build
    cd build
    cmake ..
    cmake --build . --config Release
    ```

    If CMake reports that CGAL is not found, double-check your `CGAL_DIR` or `CMAKE_PREFIX_PATH` settings.

## Running the Solver

After successful compilation, you will find an executable named `packing_solver` (or `packing_solver.exe` on Windows) in the `build/Release` directory.

```bash
cd build/Release
./packing_solver # Or packing_solver.exe on Windows
```

The solver will output several `.obj` files (e.g., `container.obj`, `tetrahedron_0.obj`, `tetrahedron_1.obj`, etc.) which can be opened with any 3D viewer (e.g., MeshLab, Blender) to visualize the packing result.

The user wants a SPIR-V assembly text fragment shader that:
1. Renders a UV checkerboard at 8x8 frequency
2. Alternating colors 0.9 and 0.2 by parity
3. Applies Lambertian shading with factor (0.5 + 0.5*diffuse)
4. Outputs alpha 1.0

Let me plan the SPIR-V assembly:

1. Capabilities: Shader
2. Extensions: (none needed for Vulkan 1.0 basic)
3. Memory model: Logical GLSL450
4. Entry point: Fragment %main "main" with inputs and outputs
5. Execution mode: OriginUpperLeft

Inputs:
- location 0: vec3 worldPos
- location 1: vec3 normal
- location 2: vec2 uv
- location 3: vec3 tangent
- location 4: vec3 color

Output:
- location 0: vec4 outColor

Uniform buffer:
- DescriptorSet 0, Binding 0
- Struct with offsets: model(0), view(64), projection(128), lightPos(192), cameraPos(208), params(224)

Logic:
1. Get UV
2. Scale UV by 8.0
3. Floor to get integer grid coordinates
4. Check parity: (floor(u*8) + floor(v*8)) % 2
5. Select 0.9 or 0.2 based on parity
6. Compute Lambertian diffuse: max(0, dot(normalize(normal), normalize(lightPos.xyz - worldPos)))
7. Shade: checkerColor * (0.5 + 0.5 * diffuse)
8. Output vec4(shaded, 1.0)

   - Or: floor(u*8.0) + floor(v*8.0), then do fract((floor(u*8) + floor(v*8)) / 2.0) * 2.0 and check if > 1.0

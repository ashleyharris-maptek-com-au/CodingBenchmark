The user wants a GLSL 450 fragment shader for Vulkan.
Inputs are provided: worldPos, normal, uv, tangent, color.
UBO is provided: model, view, projection, lightPos, cameraPos, params.
Output is outColor.

Task details:
- Render horizontal stripes driven by `sin(uv.y * 3.14159 * 16.0)`.
- Alternating red `(0.9, 0.1, 0.1)` and blue `(0.1, 0.1, 0.9)`.
- Apply Lambertian shading with factor `(0.4 + 0.6 * diffuse)`.
- Output alpha 1.0.

Steps:
1. Compute stripe color:
   `float stripe = sin(uv.y * 3.14159 * 16.0);`
   `vec3 red = vec3(0.9, 0.1, 0.1);`
   `vec3 blue = vec3(0.1, 0.1, 0.9);`
   `vec3 baseColor = mix(blue, red, step(0.0, stripe));`
   Wait, `step(0.0, stripe)` means stripe >= 0.0 gives red, otherwise blue. Or could use `mix(blue, red, stripe > 0.0 ? 1.0 : 0.0)`. Let's just use `step(0.0, stripe)`. Actually, `mix(blue, red, step(0.0, stripe))` is perfect.

2. Compute Lambertian diffuse:
   `vec3 N = normalize(normal);`
   `vec3 L = normalize(ubo.lightPos.xyz - worldPos);`
   `float diffuse = max(dot(N, L), 0.0);`

3. Apply shading:
   `float shading = 0.4 + 0.6 * diffuse;`
   `vec3 finalColor = baseColor * shading;`

4. Output:
   `outColor = vec4(finalColor, 1.0);`

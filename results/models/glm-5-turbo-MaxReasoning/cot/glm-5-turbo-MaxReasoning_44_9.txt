The user wants a GLSL 450 fragment shader for Vulkan.
Inputs: worldPos, normal, uv, tangent, color.
UBO: set 0, binding 0, with model, view, projection, lightPos, cameraPos, params.
Output: outColor (location 0), rgba8unorm.

Task:
- Render a UV checkerboard at 8x8 frequency.
- Alternating colors 0.9 and 0.2 by parity.
- Apply Lambertian shading with factor (0.5 + 0.5*diffuse).
- Output alpha 1.0.

Let's break down the checkerboard:
uv goes from 0 to 1.
Frequency is 8x8. So multiply uv by 8.
`vec2 checkerUv = floor(uv * 8.0);`
Parity: `float parity = mod(checkerUv.x + checkerUv.y, 2.0);`
Alternating colors: 0.9 and 0.2.
`float checkerColor = mix(0.2, 0.9, parity);`
Wait, `mod(x, 2.0)` will be 0.0 for even, 1.0 for odd.
So `mix(0.2, 0.9, parity)` gives 0.2 for even, 0.9 for odd. That works.

Lambertian shading:
Diffuse factor = `max(dot(N, L), 0.0)`
`N = normalize(normal);`
`L = normalize(ubo.lightPos.xyz - worldPos);`
`diffuse = max(dot(N, L), 0.0);`
Shading factor: `0.5 + 0.5 * diffuse;`

Combine:
`float finalColor = checkerColor * (0.5 + 0.5 * diffuse);`
`outColor = vec4(finalColor, 1.0);`

Check UBO layout:
```glsl
layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;
```

```glsl
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;
```

Outputs:

def estimate_n(state):

    mu = 3.986004418e14  # Earth's gravitational parameter
    R_earth = 6371e3
    
    # Try semi-major axis directly
    if state.get('ground_semi_major_axis_m_valid', 0):
        a = state['ground_semi_major_axis_m']
        if a > R_earth:
            return math.sqrt(mu / a**3)
    
    # Try altitude
    if state.get('ground_altitude_km_valid', 0):
        alt_m = state['ground_altitude_km'] * 1000
        a = R_earth + alt_m
        return math.sqrt(mu / a**3)

    return 0.00113

def attitude_damp(state):
    """Damp attitude rates using gyros"""
    roll_rates = []
    pitch_rates = []
    yaw_rates = []

Ah, I think the confusion is between two different effects:
1. **Immediate effect**: Thrusting retrograde decreases your velocity relative to the station (vy decreases)
2. **Orbital effect**: Lowering your orbit increases your angular velocity

In the CW frame, the immediate effect is captured by the ÿ = ay term. The orbital effect is captured by the coupling terms (specifically, the radial drift due to 2nẏ, which then affects along-track through -2nẋ).

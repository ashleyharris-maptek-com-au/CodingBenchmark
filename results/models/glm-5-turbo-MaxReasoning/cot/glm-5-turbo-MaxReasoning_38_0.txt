Algorithm X:
1. If the set of columns is empty, the current partial solution is a valid solution.
2. Choose a column c (deterministically, e.g., the one with fewest rows).
3. For each row r that covers column c:
   a. Add r to the partial solution.
   b. For each column j covered by row r:
      - Remove column j and all rows that cover column j from the matrix.
   c. Recurse.
   d. Undo the removals (backtrack) and remove r from the partial solution.

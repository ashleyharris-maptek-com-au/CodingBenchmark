**Designing minimal SPIR-V fragment shader**

Planning a minimal SPIR-V fragment shader that takes a vec2 UV input at location 2 and outputs a vec4 color at location 0; it uses basic types, variables, and decorations without external instructions, aiming for a hex string binary output.**Crafting full SPIR-V hex binary**

The complete minimal SPIR-V fragment shader binary is encoded as a hex string of little-endian bytes for each word, carefully assembled with correct IDs, instructions, decorations, types, constants, variables, and function code to output vec4 color from input UV coordinates.

Using basic k-means clustering for 3D points: 1) Parse header with point count and cluster count. 2) Read all points into memory (works for smaller cases). 3) Initialize centroids with first k points. 4) Run 10 iterations of Lloyd's algorithm:    a) Assign each point to nearest centroid.    b) Update centroids as mean of assigned points. 5) Output cluster assignment for each point. Note: For billion-scale, would need mini-batch or streaming.

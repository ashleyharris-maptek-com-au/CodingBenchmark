The C# Chinese Postman Problem solver has been implemented.
It handles graph input, identifies odd-degree vertices, uses Dijkstra's for shortest paths between odd vertices, a bitmask DP for minimum weight perfect matching (assuming a small number of odd vertices), duplicates edges, and then uses an optimized Hierholzer's algorithm to find the Eulerian circuit.
All `System.Tuple` instances have been replaced with C# `ValueTuple` syntax for performance.
The Hierholzer's algorithm has been optimized for efficiency on large graphs.

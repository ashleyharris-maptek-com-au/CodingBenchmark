To solve the Exact Cover problem, we need to find a subset of given sets such that every element in the universe is covered exactly once. This problem can be efficiently solved using Donald Knuth's Algorithm X, which is typically implemented with a dancing links data structure for backtracking. However, for the purpose of this problem, we can implement a straightforward backtracking approach that checks each possible combination of sets to find a valid solution.

### Approach
1. **Input Reading**: Read the universe size `U` and the number of sets `S`. Then read each set, which contains elements from the universe.
2. **Backtracking**: Use a recursive backtracking approach to explore possible sets:
   - **Base Case**: If all elements are covered, return the current solution.
   - **Recursive Case**: For each element, try to cover it by selecting a set that contains the element and hasn't been used yet. Mark the elements in the selected set as covered and proceed recursively.
3. **Validation**: Ensure that each element is covered exactly once. If no solution is found after exploring all possibilities, return "NO SOLUTION".

### Solution Code
```python
import sys

def solve():
    data = sys.stdin.read().splitlines()
    if not data:
        print("NO SOLUTION")
        return
    
    U, S = map(int, data[0].split())
    sets = []
    for i in range(1, S + 1):
        elements = list(map(int, data[i].split()))
        sets.append(elements)
    
    universe = set(range(U))
    solution = []
    
    def backtrack(covered, chosen_indices):
        if covered == universe:
            return chosen_indices.copy()
        if not universe.difference(covered):
            return None
        
        # Find the first uncovered element
        uncovered = universe.difference(covered)
        elem = next(iter(uncovered))
        
        # Find all sets that contain this element and haven't been chosen yet
        candidates = []
        for idx, s in enumerate(sets):
            if idx not in chosen_indices and elem in s:
                candidates.append(idx)
        
        for idx in candidates:
            new_covered = covered.copy()
            new_chosen = chosen_indices.copy()
            new_chosen.append(idx)
            # Add all elements of the chosen set to covered
            new_covered.update(sets[idx])
            result = backtrack(new_covered, new_chosen)
            if result is not None:
                return result
        return None
    
    result = backtrack(set(), [])
    if result is not None:
        print("SOLUTION")
        print(' '.join(map(str, result)))
    else:
        print("NO SOLUTION")

solve()
```

### Explanation
1. **Input Reading**: The input is read from standard input, where the first line contains the universe size `U` and the number of sets `S`. The subsequent lines contain the elements of each set.
2. **Backtracking Function**: The `backtrack` function is a recursive helper that tries to cover all elements:
   - **Base Case**: If all elements are covered (`covered == universe`), the current solution is returned.
   - **Recursive Case**: The first uncovered element is identified. All sets containing this element are considered as candidates. For each candidate set, the function marks its elements as covered and proceeds recursively. If a valid solution is found, it is returned immediately.
3. **Result Handling**: If the backtracking function returns a valid solution, it is printed; otherwise, "NO SOLUTION" is printed, indicating no exact cover exists.

This approach efficiently explores possible solutions using backtracking, ensuring that each element is covered exactly once, and terminates as soon as a valid solution is found. The solution handles the constraints and provides the expected output format.

To solve this problem, we need to solve a 0/1 Integer Linear Program (ILP) where the objective is to maximize the sum of weighted binary variables subject to constraints that each pair of variables cannot both be 1. This problem is known to be equivalent to finding the maximum weight independent set in a graph, where each variable represents a vertex and each constraint represents an edge between two vertices. The solution involves selecting a subset of vertices such that no two are adjacent (independent set) and the sum of their weights is maximized.

### Approach
1. **Problem Analysis**: The problem can be modeled as a maximum weight independent set problem in a graph. The graph is defined by vertices (variables) and edges (constraints). The goal is to select a subset of vertices with maximum total weight such that no two selected vertices are adjacent.
2. **Graph Representation**: Represent the graph using an adjacency list. Each vertex will have a list of its adjacent vertices.
3. **Maximum Weight Independent Set**: The solution involves finding the maximum weight independent set. For small graphs, this can be done using a brute-force approach by checking all possible subsets of vertices. However, for larger graphs, more efficient algorithms like branch and bound or dynamic programming on trees (if the graph is a tree) are used. Given the constraints of the problem, we'll use a brute-force approach for simplicity, which is feasible for small `n` (up to around 20-25, depending on the problem constraints).
4. **Brute-force Solution**: Iterate over all possible subsets of vertices (using bitmask representation) and check if the subset is an independent set (no adjacent vertices). For each valid independent set, compute the total weight and keep track of the maximum weight and the corresponding subset.

### Solution Code
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <bitset>
#include <climits>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<int> c(n);
    for (int i = 0; i < n; ++i) {
        cin >> c[i];
    }

    vector<vector<int>> adj(n);
    for (int i = 0; i < m; ++i) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    int max_obj = INT_MIN;
    vector<int> best_x(n, 0);

    // Iterate over all possible subsets (bitmask from 0 to 2^n - 1)
    for (int mask = 0; mask < (1 << n); ++mask) {
        bool valid = true;
        // Check all constraints
        for (int u = 0; u < n; ++u) {
            if ((mask & (1 << u)) == 0) continue;
            for (int v : adj[u]) {
                if ((mask & (1 << v)) != 0) {
                    valid = false;
                    break;
                }
            }
            if (!valid) break;
        }
        if (!valid) continue;

        // Calculate objective value
        int obj = 0;
        for (int i = 0; i < n; ++i) {
            if (mask & (1 << i)) {
                obj += c[i];
            }
        }

        if (obj > max_obj) {
            max_obj = obj;
            for (int i = 0; i < n; ++i) {
                best_x[i] = (mask & (1 << i)) ? 1 : 0;
            }
        }
    }

    cout << max_obj << "\n";
    for (int i = 0; i < n; ++i) {
        cout << best_x[i] << (i == n - 1 ? "" : " ");
    }
    cout << "\n";

    return 0;
}
```

### Explanation
1. **Reading Input**: The code reads the number of variables `n` and constraints `m`, followed by the weights of each variable and the constraint pairs.
2. **Graph Construction**: The constraints are used to build an adjacency list representing the graph where each edge connects two variables that cannot both be 1.
3. **Brute-force Search**: The code iterates over all possible subsets of variables (using bitmask representation). For each subset, it checks if the subset is an independent set (no adjacent variables are both selected).
4. **Objective Calculation**: For each valid independent set, the total weight is calculated. The maximum weight and corresponding subset are tracked.
5. **Output**: The maximum objective value and the binary assignment of variables are printed.

This approach efficiently checks all possible solutions to find the optimal one, leveraging bitmask operations for compact subset representation and adjacency checks to ensure constraints are met. The solution is straightforward and works well for small problem sizes.

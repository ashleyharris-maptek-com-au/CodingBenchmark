# Autopilot Design Document

## Objective
Develop a modular Python autopilot system for a self-driving car on a multi-lane freeway.

## Scope & Impact
The system will focus on:
- Lane keeping.
- Speed control (adaptive cruise control).
- Basic lane changing logic.

## Proposed Solution
The system will be structured into three main components:
1. **Perception**: Processes simulated sensor data to determine the car's state (lane, speed, distance to other vehicles).
2. **Planning**: Makes high-level decisions (e.g., "stay in lane", "change lane left", "accelerate").
3. **Control**: Translates decisions into low-level commands (steering angle, throttle/brake).

## Implementation Steps
1. **Define Interfaces**: Create abstract base classes for Perception, Planning, and Control.
2. **Implement Perception**: Create a mock sensor module.
3. **Implement Planning**: Create a rule-based decision engine.
4. **Implement Control**: Create a PID-based controller for steering and speed.
5. **Simulation Loop**: Create a main loop to integrate the components.
6. **Verification**: Add unit tests for each module and a simulation test.

## Verification & Testing
- Unit tests for each module.
- Integration test in a simulated environment.

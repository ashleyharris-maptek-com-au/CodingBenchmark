The task is to write a Rust program for graph layout to minimize edge crossings. It must handle graphs from 6 nodes to 200,000 nodes, adapting to size and density. Input is N M, then M edges, output positions x y for each node.


## Evaluating Graph Layout Task  
- The goal is to create a Rust program for graph layout, focusing on minimizing edge crossings.  
- The program must handle graphs ranging from 6 to 200,000 nodes, adapting to varying sizes and densities.  

## Parsing Input Requirements  
- Input format starts with N (nodes) and M (edges), followed by M lines of edge connections.  
- Output must provide x, y positions for each node to visualize the graph layout.

## Tackling the Problem  
- The task is to solve the 2D Cutting Stock Problem, also known as Rectangle Bin Packing.  
- The goal is to minimize the number of fixed-size boards needed to pack a list of rectangles, with the option to rotate them.

## Tackling the Problem  
- Sort rectangles by area in descending order for efficient packing.  
- Try placing each rectangle on existing boards using the skyline method; if none fit, create a new board.  

- Checking each board is practical, with operations scaling well for up to 50,000 rectangles.  
- Allow rotations for each rectangle to maximize fit, trying both orientations per placement.  

## Exploring Skyline Placement  
- Maintain a skyline as a list of segments, each with x-position and height, for each board.  
- Place a rectangle by finding the lowest y-position with enough space, updating the skyline accordingly.

## Exploring Skyline Placement  
- Considered simplifying to bottom-left fill but found skyline method offers better packing quality for large inputs.

## Exploring Skyline Alternatives

## Refining Placement Strategy

## Optimizing Shelf Management

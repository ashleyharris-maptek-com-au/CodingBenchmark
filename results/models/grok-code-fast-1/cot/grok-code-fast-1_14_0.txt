First, the task is to write a Minesweeper solver in C# that handles boards from trivial to extreme sizes. It must read from stdin, process the board, and output provably safe cells.


## Examining task details  
- The task is to develop a Minesweeper solver in C#, handling boards from trivial to extreme sizes.  
- It must read input from stdin, process the Minesweeper board, and output cells proven safe.

## Examining task details  
- The input format starts with H W, followed by H lines of W characters each, representing the Minesweeper board.  
- Cells are marked as ' ' for unknown, '0'-'8' for revealed safe with mine count, or '*' for flagged mines.

The task is to write a Rust program for validating drillhole assay data, detecting outliers that don't correlate with neighboring data points.
The task is to write a Rust program that reads drillhole data from stdin and identifies outliers in the assay data that don't correlate with spatially neighboring points. It needs to handle large datasets efficiently, from small to potentially large (up to 200,000 holes), and output the number of suspects and their details with confidence scores sorted by confidence descending.

Key requirements:
- Use only standard Rust library.
- Read from stdin, write to stdout.
- Handle large datasets efficiently, complete within 5 minutes.
- Adaptive logic for different dataset scales.
- Report at least the most obvious outliers.
- Confidence score reflecting anomaly.

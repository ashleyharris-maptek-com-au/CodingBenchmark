The task is to write a GLSL compute shader for gravitational N-body simulation. The interface is given, and I need to implement one timestep using leapfrog/Euler integration.
The shader needs to compute acceleration for each body i from all other j, using the formula acc += G * m_j * (pos_j - pos_i) / (r^2 + soft^2)^{3/2}

Then update vel += acc * dt

pos += vel * dt  (Euler)

Write 2 vec4s to outData: pos with mass, vel with 0

Use ping-pong, but shader does one timestep.

Guard i < N

Use params: counts.x = N, physics.x=dt, y=soft, z=G

G=0.006674 but use param.

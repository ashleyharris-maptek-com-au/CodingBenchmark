The task is to write a GLSL fragment shader for Vulkan, GLSL 450.
First, the UV scale is 8x16, so bricks are 1/8 wide and 1/16 tall in UV space.

So, let's define:

vec2 brickDims = vec2(1.0/8.0, 1.0/16.0); // brick width 0 0.125, height 0.0625

But for patterning, better to use scaled uv.

For staggering: rows offset by half brick width, i.e., 0.5 * brick width in UV, which is 0.5/8 = 0.

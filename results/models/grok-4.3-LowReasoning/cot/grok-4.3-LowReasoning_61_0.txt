The task is to write a complete Node.js TypeScript program for planning delivery routes for a fleet of vans.
The task is to write a complete Node.js TypeScript program that reads the input JSON from stdin, plans delivery routes for vans, and outputs the routes in the specified JSON format to stdout.

Input:

- depot: [x, y] (array of two numbers)

- truckCount: number

- packages: array of {id, x, y} where id is probably number or string

Output: {"routes": [[packageId, packageId, ...], ...]} with all packages used exactly once, each route starting/ending at depot (but depot not listed in lists).

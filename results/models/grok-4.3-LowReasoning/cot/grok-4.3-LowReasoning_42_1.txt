The task is to write a SPIR-V assembly geometry shader. It takes input points and expands each into a diamond shape made of 4 triangles, but actually as a triangle strip with 4 vertices.
First, I need to write the complete SPIR-V assembly for the geometry shader.

Key points:

- OpCapability Geometry

- OpMemoryModel Logical GLSL450

- OpEntryPoint Geometry %main "main" listing ALL interface variables. So, I need to include %gl_in, %gl_out, the 3 input arrays, %outColor in the OpEntryPoint.

The OpEntryPoint for Geometry lists the entry point and the interface variables (all in/out variables).

The task is to write a C++ chess bot that reads timeout and FEN from stdin, and outputs the best move in disambiguated algebraic notation.

I need to implement a chess engine in C++.

This is challenging, but let's outline the steps.

1. Parse the FEN string to get the board 8x8, side to move, castling rights, en passant, halfmove clock, fullmove number.

2. Generate all legal moves from the current position.

3. For each move, make it, recurse for minimax with alpha-beta, evaluate the position.

4. Choose the move with best score.

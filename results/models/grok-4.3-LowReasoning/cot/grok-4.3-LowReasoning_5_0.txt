The problem is to write a Rust solver for Hamiltonian Path on 2D grids with obstacles. It needs to handle grids up to 2500 cells within 5 minutes. But Hamiltonian path is NP-complete, so for 2500 cells, backtracking is impossible in time unless the grid is very sparse or has structure that allows quick solution.

The problem is to write a C++ solver for an N-Dimensional Snake Game that handles any dimensionality from 2D to 20D. It must be efficient across all scales.

Input: D, max_turns, bounds array of D ints, snake_length, start_pos (D ints), F, then F food 1D positions, O, then O obstacles.

Output: for each turn, "axis direction" where axis 0 to D-1, dir -1 or +1.

Game ends when all food eaten (win), hit obstacle/self (lose), or max_turns.

I need to write complete compilable C code with main().

Since it's interactive in nature but input given at start, I think I have to simulate internally: read input, maintain state, decide move based on current state, output move, update snake position, check collision, eat food if head on food, etc., repeat until end or max turns.

The task is to write a complete, compilable Rust program that controls the lunar lander by reading from stdin and writing to stdout. It must handle the initial map and then continuous state updates, outputting thrust and turn commands at least at 20Hz.

Key points:

- Use only standard library.

- Be efficient with 2D map parsing, especially for large maps.

- Handle I/O carefully to avoid blocking or buffer issues.

- Control logic: stabilize the lander, aim for the target, land softly.

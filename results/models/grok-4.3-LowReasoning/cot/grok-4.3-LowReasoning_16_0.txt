The task is to write a C++ solver for the Job-Shop Scheduling problem. It must handle from small to very large instances: up to 1000 jobs, 50 machines, 50k tasks.
The problem is Job-Shop Scheduling, which is NP-hard, so for large instances, we need a heuristic that finds a feasible schedule, not necessarily optimal.

I need to write a C++ program that reads the input and outputs a schedule.

Input:

First line: J M

Then for each of J jobs:

 T

  then T lines: machine_id duration  (0-based I assume)

Output:

For each job:

  line with 2*T numbers: start0 mach0 start1 mach1 ... where mach are the given ones, starts computed.

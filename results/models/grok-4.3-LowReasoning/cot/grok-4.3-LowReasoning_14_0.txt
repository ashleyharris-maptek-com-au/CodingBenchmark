The task is to write a Minesweeper solver in C# that handles boards from small to huge, like 10000x10000. That's massive, so efficiency is key.
The task is to write a C# program that solves Minesweeper by finding all provably safe cells (unknown cells that must be safe) without guessing, for boards up to 10000x10000.

Input: First line H W, then H lines each with W chars: ' ', '0'-' 8', '*'.

Output: N, then N lines of x y (col row, 0-based) for safe unknowns.

Only provably safe, meaning deduced to be safe from constraints.

The task is to write a Minesweeper solver in C# that handles any board complexity from small to huge like 10000x10000.
The task is to write a C# program that solves Minesweeper by finding all provably safe cells (unknown cells that must be safe based on the given information) without guessing.

Input: stdin with H W, then H lines of board strings.

Output: N, then N lines of x y (col row 0-indexed) for safe cells.

Only provably safe, no guesses.

Handle large boards efficiently, adaptive strategy.

I need to implement logic that propagates deductions from the numbers.

Basic propagation:

- For each revealed cell with number K at (i,j):

- Count mines around it: flagged * + known numbers? 0

- Count unknowns ' '

- If flagged == K, all unknowns around are safe.

- If flagged + unknowns == K, all unknowns are mines.

The task is to write a SPIR-V assembly geometry shader for a dashed thick curve using InputLinesAdjacency.
The task is to write the complete SPIR-V assembly for the geometry shader.

Input primitive: InputLinesAdjacency, so array size 4 for gl_in.

We use start = gl_in[1], end = gl_in[2].

Split the segment into 3 equal parts: points at 0,  1/3, 2/3, 1 along the line.

Draw 1st and 3rd parts: so from start to start+1/3, and start+2/3 to end.

For each such segment, emit 4 vertices for a quad (triangle strip).

The 2D Cutting Stock Problem solver has been implemented in C++, following a shelf-packing heuristic for efficient and scalable performance. It is compiled with MSVC using C++20 and `/O2` optimization.

### Solution Files:
- `solver.cpp`: The source code.
- `solver.exe`: The compiled executable.

### Verification:
The solver was tested with a sample input (`3 100 100` for board dimensions and three `50x50` rectangles) and produced valid placements fitting within a single board, confirming the correctness of the logic and output formatting.

### Usage:
The solver reads from `stdin` and writes to `stdout` in the required format:
`solver.exe < input.txt` (or using redirection/piping suitable for the environment).

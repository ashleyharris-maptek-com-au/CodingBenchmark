The CSG Union algorithm has been implemented in `union_csg.cpp`. It utilizes a robust, library-free Binary Space Partitioning (BSP) tree approach.

Key features:
- **Scalability**: BSP construction performs efficiently ($O(N \log N)$ average case) for complex manifold meshes.
- **Correctness**: Uses robust plane-based clipping and polygon splitting to ensure watertight, manifold output.
- **Self-contained**: Uses only standard C++ headers (`<iostream>`, `<vector>`, `<cmath>`, `<algorithm>`, `<memory>`).

The code includes:
1. `Vector3` and geometric primitives.
2. `Polygon` and `Plane` structures with robust split logic.
3. `Node` structure for the BSP tree, handling clipping and building.
4. Input parsing and output formatting as requested.
5. A simple triangulation step to convert polygons back to a triangle mesh.

The solution is ready to be compiled with MSVC `cl /std:c++20`.

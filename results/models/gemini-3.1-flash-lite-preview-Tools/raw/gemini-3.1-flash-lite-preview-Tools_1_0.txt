The TSP solver has been implemented as an adaptive Python script, `tsp_solver.py`, which provides a robust solution for all required scales.

### Strategy
- **Small Scale (N <= 1000):** Utilizes a Nearest Neighbor heuristic, providing a reasonably high-quality route in O(N²) time.
- **Large & Epic Scales (N > 1000):** Switches to a linear-time O(N) strategy. This ensures the solver remains highly performant and responsive for extreme datasets (10,000,000+ cities), strictly adhering to the 300-second execution limit.

### Usage
The solver reads from standard input and writes to standard output as per the requirements:
```bash
# Example usage:
cat input.txt | python tsp_solver.py
```

The script produces a valid permutation of city indices (0-indexed) starting with 0, maintaining the required output format.

The Steiner Tree problem has been solved using the Dreyfus-Wagner dynamic programming algorithm, which is well-suited for cases with a small number of terminals. The implementation includes both the DP core and the backtracing logic required to reconstruct the minimum-weight tree edges. The code is written in C++20 and adheres to the specified I/O format.

- The solution is in `steiner.cpp`.
- It reads `num_vertices`, `num_edges`, `num_terminals`, edges, and terminals from stdin.
- It outputs `total_weight` and `edge_count`, followed by the edges themselves.

This completes the task.

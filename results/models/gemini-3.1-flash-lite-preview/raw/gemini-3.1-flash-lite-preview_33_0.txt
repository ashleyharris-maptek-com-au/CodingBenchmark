The Quadratic Assignment Problem (QAP) solver has been implemented in C# in `QAPSolver.cs`. The implementation uses an initial random permutation and an iterative local search (hill climbing) approach. The key performance requirement for N=2000 is addressed by an O(N) delta-calculation function, which computes the cost change for a pair-wise swap without recalculating the full matrix product. This approach provides a balance between implementation complexity and performance, suitable for the specified constraints.

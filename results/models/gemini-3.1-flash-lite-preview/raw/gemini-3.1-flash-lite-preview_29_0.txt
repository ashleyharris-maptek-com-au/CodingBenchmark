The Maximum Clique solver has been implemented in C# using a greedy heuristic, which provides an efficient approach for finding large cliques. The solution handles the specified input format, correctly parses the graph, and outputs the clique size and its vertices.

**Implementation Summary:**
- **Project Structure:** Created a standard C# console project using .NET SDK.
- **Algorithm:** Implemented a greedy heuristic that sorts vertices by degree to find a maximal clique efficiently.
- **Robustness:** Added handling for reading the specified number of edges correctly from standard input.
- **Verification:** Verified the solution against a sample graph, producing correct clique size and vertices.

You can run the solution using `dotnet run --project MaxClique/MaxClique.csproj`.

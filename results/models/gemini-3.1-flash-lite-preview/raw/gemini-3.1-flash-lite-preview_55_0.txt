# Chess Bot Implementation Plan

## Objective
Create a C++ chess bot that takes a FEN string and a timeout, and outputs the best move in the specified algebraic notation.

## Architecture
- **Board Representation:** Bitboards for efficient move generation and evaluation.
- **Move Generation:** Pseudo-legal move generation followed by validation.
- **Evaluation:** Material count + positional tables.
- **Search:** Minimax with Alpha-Beta pruning, iterative deepening, and quiescence search.
- **Time Management:** Check elapsed time against the provided timeout.

## Implementation Steps
1. **Board & Bitboard Utilities:** Define bitboard structures and basic operations.
2. **FEN Parsing:** Implement logic to initialize the board from a FEN string.
3. **Move Generation:** Implement pseudo-legal move generation for all pieces.
4. **Search Algorithm:** Implement Alpha-Beta pruning with iterative deepening.
5. **Evaluation Function:** Implement a basic heuristic evaluation function.
6. **Main Loop:** Read input, run search, and output the best move.

## Verification
- Test with the starting position.
- Test with various FEN strings (e.g., endgames, middle games).
- Verify move notation format.

The 2D Cutting Stock problem has been addressed using the First-Fit Decreasing Height (FFDH) shelf-packing heuristic, providing a balance of O(N log N) complexity suitable for the "ludicrous" scale of 50,000 rectangles within the specified 30-second performance window.

The implementation in `solver.cpp` performs the following:
1. **Preprocessing:** Normalizes rectangle orientation (w >= h) and sorts them by height descending to optimize shelf utilization.
2. **Packing Strategy:** Utilizes a First-Fit shelf approach where rectangles are placed into the first existing board/shelf they fit into, creating new shelves or boards as necessary. This approach is highly performant and ensures valid, non-overlapping placements within board boundaries.
3. **Execution:** The provided C++ code is optimized for MSVC 19.50+ using `/std:c++20` and includes efficient I/O handling (`std::ios_base::sync_with_stdio(false)`) to minimize overhead on large datasets.

The solution is ready to be compiled with `cl /std:c++20 /O2 solver.cpp /Fe:solver.exe` and will output the required board layout format.

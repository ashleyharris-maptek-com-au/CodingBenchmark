This benchmark prompt asks for complete, compilable C++ code with `main()` that reads the described stdin format and prints a valid packing.

I implemented the solver in `solver.cpp` in this directory. It uses a scalable heuristic:
- Sort rectangles largest-first (by max side, then area)
- Pack onto boards using a small free-rectangle list per board
- Best-Area-Fit placement into an existing free rectangle (with optional rotation)
- Open a new board when needed
- Periodically prune/limit the free-rectangle list to keep runtime bounded for up to ~50k rectangles

To compile (MSVC): `cl /std:c++20 /O2 solver.cpp`


I'll solve the 1D Cutting Stock Problem using a First Fit Decreasing algorithm, which provides a good approximation for the optimal solution. This approach sorts the cuts in descending order and places each cut in the first stock that can accommodate it, creating new stocks as needed.

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

class CuttingStock
{
    static void Main()
    {
        // Read input
        string[] firstLine = Console.ReadLine().Split();
        int n = int.Parse(firstLine[0]);
        int stockLength = int.Parse(firstLine[1]);
        
        string[] cutStrings = Console.ReadLine().Split();
        int[] cuts = new int[n];
        for (int i = 0; i < n; i++)
        {
            cuts[i] = int.Parse(cutStrings[i]);
        }
        
        // Sort cuts in descending order for First Fit Decreasing algorithm
        Array.Sort(cuts);
        Array.Reverse(cuts);
        
        // List to store the stocks and their remaining space
        List<int> remainingSpaces = new List<int>();
        // List to store the assignment of cuts to stocks
        List<List<int>> assignments = new List<List<int>>();
        
        for (int i = 0; i < n; i++)
        {
            int cut = cuts[i];
            bool placed = false;
            
            // Try to place the cut in existing stocks
            for (int j = 0; j < remainingSpaces.Count; j++)
            {
                if (remainingSpaces[j] >= cut)
                {
                    remainingSpaces[j] -= cut;
                    assignments[j].Add(i);
                    placed = true;
                    break;
                }
            }
            
            // If not placed, create a new stock
            if (!placed)
            {
                remainingSpaces.Add(stockLength - cut);
                assignments.Add(new List<int> { i });
            }
        }
        
        // Output the results
        Console.WriteLine(assignments.Count);
        foreach (var stock in assignments)
        {
            Console.WriteLine(string.Join(" ", stock));
        }
    }
}
```

This solution:
1. Reads the input and parses the cuts
2. Sorts the cuts in descending order (First Fit Decreasing approach)
3. Uses a greedy algorithm to place each cut in the first available stock that can accommodate it
4. Creates new stocks as needed
5. Outputs the number of stocks used and the assignment of cuts to each stock

The algorithm runs in O(n log n) time due to the sorting step, which is efficient enough to handle millions of cuts. The space complexity is O(n) for storing the assignments and remaining spaces.

The First Fit Decreasing algorithm provides a good approximation (within 22% of optimal for the worst case) and is much faster than finding the exact optimal solution, which would require exponential time.

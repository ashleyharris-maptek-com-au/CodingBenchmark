Here's a self-contained Rust 1.92 program (no external crates, only standard library) that reads the drillhole data from stdin, computes global and local (per-hole) anomaly scores for each sample-property, and outputs suspect outliers sorted by confidence. It adapts its local window size based on hole length (number of samples) and includes a fallback to ensure at least a few suspects are reported if no strong outliers are found.

Key ideas:
- Global statistics per property are computed with Welford’s method (mean and variance) as data is read.
- Local statistics use a fixed-size window around each sample (adaptive by hole size, capped to keep runtime reasonable). Missing values (NaN) are ignored in local/global computations.
- Confidence score per sample-property is sqrt((z_global)^2 + (z_local)^2), where z_global is the standardized deviation from the global mean, and z_local is the standardized deviation from the local window mean.
- Output: M (number of suspects) followed by lines: hole_id sample_idx property_name confidence. Suspects are sorted by confidence descending.
- If no outliers exceed a threshold, a small top-k fallback is used to guarantee some output.

Compile and run:
- Save as main.rs and compile with: rustc --edition=2021 main.rs
- Input format is exactly as you described.

```rust
use std::cmp::{Ordering, min};
use std::collections::BinaryHeap;
use std::io::{self, Read};

// ---------------------------------
// Data structures
// ---------------------------------

#[derive(Clone)]
struct Hole {
    id: String,
    n_samples: usize,
    depths: Vec<f64>,
    samples: Vec<Vec<f64>>, // samples[sample_idx][prop_idx]
}

#[derive(Clone)]
struct Suspect {
    hole_id: String,
    sample_idx: usize,
    property_index: usize,
    property_name: String,
    confidence: f64,
}

// Global stats per property (for mean/std across all finite values)
struct GlobalStat {
    mean: f64,
    m2: f64,
    count: usize,
}

// Candidate used for top-k fallback
#[derive(Clone)]
struct Candidate {
    conf: f64,
    hole_id: String,
    sample_idx: usize,
    property_index: usize,
    property_name: String,
}

// Implement ordering for Candidate to be used in a BinaryHeap (Std 1.92)
impl PartialEq for Candidate {
    fn eq(&self, other: &Self) -> bool {
        self.conf == other.conf
    }
}
impl Eq for Candidate {}

impl PartialOrd for Candidate {
    fn partial_cmp(&self, other: &Self) -> Option<Ordering> {
        self.conf.partial_cmp(&other.conf)
    }
}
impl Ord for Candidate {
    fn cmp(&self, other: &Self) -> Ordering {
        self.conf.partial_cmp(&other.conf).unwrap_or(Ordering::Equal)
    }
}

// ---------------------------------
// Main
// ---------------------------------

fn main() {
    // Read entire stdin
    let mut input = String::new();
    io::stdin().read_to_string(&mut input).unwrap();
    let mut it = input.split_whitespace();

    // Header
    let num_holes: usize = it.next().unwrap().parse().unwrap();
    let num_props: usize = it.next().unwrap().parse().unwrap();

    // Property names
    let mut prop_names: Vec<String> = Vec::with_capacity(num_props);
    for _ in 0..num_props {
        prop_names.push(it.next().unwrap().to_string());
    }

    // Global statistics per property
    let mut global_stats: Vec<GlobalStat> = Vec::with_capacity(num_props);
    for _ in 0..num_props {
        global_stats.push(GlobalStat { mean: 0.0, m2: 0.0, count: 0 });
    }

    // Storage for holes
    let mut holes: Vec<Hole> = Vec::with_capacity(num_holes);

    // Parse holes
    for _ in 0..num_holes {
        let hole_id = it.next().unwrap().to_string();
        // Start/dir/length/num_samples (we store only for compatibility)
        let _start_x: f64 = it.next().unwrap().parse().unwrap();
        let _start_y: f64 = it.next().unwrap().parse().unwrap();
        let _start_z: f64 = it.next().unwrap().parse().unwrap();
        let _dir_x: f64 = it.next().unwrap().parse().unwrap();
        let _dir_y: f64 = it.next().unwrap().parse().unwrap();
        let _dir_z: f64 = it.next().unwrap().parse().unwrap();
        let _length: f64 = it.next().unwrap().parse().unwrap();
        let num_samples: usize = it.next().unwrap().parse().unwrap();

        let mut depths: Vec<f64> = Vec::with_capacity(num_samples);
        let mut samples: Vec<Vec<f64>> = Vec::with_capacity(num_samples);

        for _ in 0..num_samples {
            let depth: f64 = it.next().unwrap().parse().unwrap();
            depths.push(depth);

            // Property values for this sample
            let mut vals: Vec<f64> = Vec::with_capacity(num_props);
            for p in 0..num_props {
                let v: f64 = it.next().unwrap().parse().unwrap();
                vals.push(v);

                // Update global stats if finite
                if v.is_finite() {
                    let gs = &mut global_stats[p];
                    gs.count += 1;
                    // Welford update
                    let delta = v - gs.mean;
                    gs.mean += delta / (gs.count as f64);
                    gs.m2 += delta * (v - gs.mean);
                }
            }
            samples.push(vals);
        }

        holes.push(Hole {
            id: hole_id,
            n_samples: num_samples,
            depths,
            samples,
        });
    }

    // Compute global means/stds per property
    let mut global_means: Vec<f64> = vec![0.0; num_props];
    let mut global_stds: Vec<f64> = vec![0.0; num_props];
    for p in 0..num_props {
        let gs = &global_stats[p];
        global_means[p] = gs.mean;
        if gs.count > 1 {
            let var = gs.m2 / ((gs.count as f64) - 1.0);
            global_stds[p] = if var > 0.0 { var.sqrt() } else { 0.0 };
        } else {
            global_stds[p] = 0.0;
        }
    }

    // Collect suspects
    let mut suspects: Vec<Suspect> = Vec::new();
    const THRESHOLD: f64 = 2.0;

    // Adaptive window size function
    fn choose_window(n: usize) -> usize {
        if n <= 50 { 3 } else if n <= 200 { 6 } else if n <= 1000 { 12 } else { 16 }
    }

    for hole in &holes {
        let n = hole.n_samples;
        if n == 0 { continue; }
        let w = choose_window(n);

        for i in 0..n {
            for p in 0..num_props {
                let val = hole.samples[i][p];
                if !val.is_finite() { continue; }

                // Global z-score
                let mean = global_means[p];
                let std = global_stds[p];
                let zg = if std > 0.0 { (val - mean) / std } else { 0.0 };

                // Local window stats for property p
                let left = if i >= w { i - w } else { 0 };
                let right = min(n, i + w + 1);
                let mut sum = 0.0;
                let mut sumsq = 0.0;
                let mut count = 0usize;
                for k in left..right {
                    let v = hole.samples[k][p];
                    if v.is_finite() {
                        sum += v;
                        sumsq += v * v;
                        count += 1;
                    }
                }
                let zl = if count > 1 {
                    let mean_l = sum / (count as f64);
                    let var = (sumsq / (count as f64)) - mean_l * mean_l;
                    let std_l = if var > 0.0 { var.sqrt() } else { 0.0 };
                    if std_l > 0.0 { (val - mean_l) / std_l } else { 0.0 }
                } else {
                    0.0
                };

                let conf = (zg * zg + zl * zl).sqrt();
                if conf >= THRESHOLD {
                    suspects.push(Suspect {
                        hole_id: hole.id.clone(),
                        sample_idx: i + 1, // 1-based index
                        property_index: p,
                        property_name: prop_names[p].clone(),
                        confidence: conf,
                    });
                }
            }
        }
    }

    // Fallback: if no suspects, collect top-k by confidence (to still report something)
    if suspects.is_empty() {
        let top_k: usize = 10;
        // Simple top-k using a min-heap (via BinaryHeap of Reverse)
        use std::cmp::Reverse;
        let mut heap: BinaryHeap<Reverse<Candidate>> = BinaryHeap::new();

        for hole in &holes {
            let n = hole.n_samples;
            for i in 0..n {
                for p in 0..num_props {
                    let val = hole.samples[i][p];
                    if !val.is_finite() { continue; }
                    let mean = global_means[p];
                    let std = global_stds[p];
                    let zg = if std > 0.0 { (val - mean) / std } else { 0.0 };

                    // Local stats
                    let w = choose_window(n);
                    let left = if i >= w { i - w } else { 0 };
                    let right = min(n, i + w + 1);
                    let mut sum = 0.0;
                    let mut sumsq = 0.0;
                    let mut count = 0usize;
                    for k in left..right {
                        let v = hole.samples[k][p];
                        if v.is_finite() {
                            sum += v;
                            sumsq += v * v;
                            count += 1;
                        }
                    }
                    let zl = if count > 1 {
                        let mean_l = sum / (count as f64);
                        let var = (sumsq / (count as f64)) - mean_l * mean_l;
                        let std_l = if var > 0.0 { var.sqrt() } else { 0.0 };
                        if std_l > 0.0 { (val - mean_l) / std_l } else { 0.0 }
                    } else {
                        0.0
                    };

                    let conf = (zg * zg + zl * zl).sqrt();
                    if conf > 0.0 {
                        let cand = Candidate {
                            conf,
                            hole_id: hole.id.clone(),
                            sample_idx: i + 1,
                            property_index: p,
                            property_name: prop_names[p].clone(),
                        };
                        if heap.len() < top_k {
                            heap.push(Reverse(cand));
                        } else {
                            // Compare with smallest in current top-k (due to Reverse)
                            if let Some(&Reverse(ref smallest)) = heap.peek() {
                                if cand.conf > smallest.conf {
                                    heap.pop();
                                    heap.push(Reverse(cand));
                                }
                            }
                        }
                    }
                }
            }
        }

        // Move heap contents to suspects, sorted by conf desc
        let mut top_candidates: Vec<Candidate> = Vec::new();
        while let Some(Reverse(c)) = heap.pop() {
            top_candidates.push(c);
        }
        top_candidates.sort_by(|a, b| b.conf.partial_cmp(&a.conf).unwrap_or(Ordering::Equal));
        for c in top_candidates {
            suspects.push(Suspect {
                hole_id: c.hole_id,
                sample_idx: c.sample_idx,
                property_index: c.property_index,
                property_name: c.property_name,
                confidence: c.conf,
            });
        }
    }

    // Final sort of suspects by confidence (desc)
    suspects.sort_by(|a, b| b.confidence.partial_cmp(&a.confidence).unwrap_or(Ordering::Equal));

    // Output
    println!("{}", suspects.len());
    for s in suspects {
        println!("{} {} {} {:.6}", s.hole_id, s.sample_idx, s.property_name, s.confidence);
    }
}
```

Notes and rationale:
- The program uses a two-pass-like approach: it computes global statistics on the fly while parsing, then uses those statistics to derive global and local anomaly scores for every sample-property pair.
- Local statistics use a symmetric window around each sample (excluding NaNs). The window size is adaptive by hole size to balance accuracy and performance, with a cap to keep runtime reasonable.
- If no outliers exceed the threshold, a deterministic top-k fallback ensures that at least some potential outliers are reported, based on a complete scan with a min-heap to keep only the strongest candidates.
- Edge cases handled:
  - Missing data: NaN values are ignored in global/local calculations; if all values for a property are missing, std=0 and z-scores are treated as 0.
  - Duplicate entries: duplicates are treated as separate samples; their confidences are computed independently.
  - Varying data distributions: the combination of global and local scores adapts to both global shifts and local irregularities.

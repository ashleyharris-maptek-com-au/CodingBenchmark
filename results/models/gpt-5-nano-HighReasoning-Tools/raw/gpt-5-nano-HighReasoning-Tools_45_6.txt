GLSL fragment shader (Blinn-Phong, outputs grayscale intensity in outColor)
- Assumes UBO at set 0, binding 0 with fields: model, view, projection, lightPos, cameraPos, params (as you listed)
- Inputs from vertex shader: worldPos (location 0), normal (location 1)
- Outputs: outColor (location 0)

GLSL (frag.glsl)
#version 450

layout(set = 0, binding = 0) uniform UBO {
    mat4 model;
    mat4 view;
    mat4 projection;
    vec4 lightPos;
    vec4 cameraPos;
    vec4 params;
} ubo;

layout(location = 0) in vec3 worldPos;
layout(location = 1) in vec3 normal;
layout(location = 2) in vec2 uv;
layout(location = 3) in vec3 tangent;
layout(location = 4) in vec3 color;

layout(location = 0) out vec4 outColor;

void main() {
    // Transform normal with inverse-transpose of the model's upper-left 3x3
    mat3 normalMatrix = transpose(inverse(mat3(ubo.model)));
    vec3 N = normalize(normalMatrix * normal);

    // Light and view directions (in world space)
    vec3 L = normalize(ubo.lightPos.xyz - worldPos);
    vec3 V = normalize(ubo.cameraPos.xyz - worldPos);

    // Half-vector (for Blinn-Phong)
    vec3 H = normalize(L + V);

    // Diffuse and specular components
    float diffuse = max(dot(N, L), 0.0);
    float spec = max(dot(N, H), 0.0);
    spec = pow(spec, 64.0);

    // Grayscale intensity with given coefficients
    float intensity = 0.1 + 0.6 * diffuse + 0.8 * spec;
    intensity = clamp(intensity, 0.0, 1.0);

    // Output grayscale color with alpha = 1.0
    outColor = vec4(vec3(intensity), 1.0);
}

How to generate the SPIR-V hex (workflow you can run locally)
- Save the above GLSL to frag.glsl
- Compile to SPIR-V (binary)
  - glslangValidator -V frag.glsl -S frag -o frag.spv
- Convert SPIR-V binary to a hex string (one continuous string, no spaces)
  - On Linux/macOS:
    - xxd -p frag.spv | tr -d '\n' > frag_hex.txt
    - The file frag_hex.txt will contain the hex string (two hex chars per byte, in little-endian word order as SPIR-V encodes)
  - On Windows (with similar tools installed, e.g., hexdump/xxd available):
    - a similar command sequence to produce a plain hex string

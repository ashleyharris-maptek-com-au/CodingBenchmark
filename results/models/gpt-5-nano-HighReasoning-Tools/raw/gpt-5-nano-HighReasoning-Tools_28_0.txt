Here's a simple, fast Rust solution that computes a 2-approximation for the Minimum Vertex Cover problem using a maximal matching approach. It processes edges in input order and, whenever it encounters an uncovered edge (u, v), it adds both endpoints to the cover. This guarantees every edge is covered and the resulting cover has size at most twice the optimum.

Key points:
- Time: O(n + m)
- Memory: O(n)
- Works for up to 2,000,000 vertices and 15,000,000 edges
- Output format matches the specification (two lines)

Rust code (compilable with rustc 1.92.0, edition 2024):

```rust
use std::io::{self, Read, Write};

fn main() {
    // Read entire input
    let mut input = String::new();
    io::stdin().read_to_string(&mut input).unwrap();
    let mut it = input.split_whitespace();

    // Read number of vertices and edges
    let num_vertices: usize = it.next().unwrap().parse().unwrap();
    let num_edges: usize = it.next().unwrap().parse().unwrap();

    // in_cover[v] == 1 means v is in the vertex cover
    let mut in_cover = vec![0u8; num_vertices];

    // Process edges in input order
    for _ in 0..num_edges {
        let u: usize = it.next().unwrap().parse().unwrap();
        let v: usize = it.next().unwrap().parse().unwrap();

        // If both endpoints are not yet in the cover, add both
        if in_cover[u] == 0 && in_cover[v] == 0 {
            in_cover[u] = 1;
            in_cover[v] = 1;
        }
    }

    // Collect covered vertices in increasing order (by index)
    let mut cover: Vec<usize> = Vec::new();
    for i in 0..num_vertices {
        if in_cover[i] != 0 {
            cover.push(i);
        }
    }

    // Prepare output
    let mut out = String::new();
    out.push_str(&cover.len().to_string());
    out.push('\n');
    if !cover.is_empty() {
        for (idx, val) in cover.iter().enumerate() {
            if idx > 0 {
                out.push(' ');
            }
            out.push_str(&val.to_string());
        }
    }
    out.push('\n');

    // Write output
    let mut stdout = io::BufWriter::new(io::stdout());
    stdout.write_all(out.as_bytes()).unwrap();
}
```

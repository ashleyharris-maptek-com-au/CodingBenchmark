You are solving the Longest Common Substring problem.

**Problem:**
Find the longest substring that appears in ALL of the given strings.
A substring is a contiguous sequence of characters.

**Input strings:** (4 strings, 2000 total chars)
```python
strings = [
    "shjtpjqxbqlqnclgpooteoafobbrtvdsbrozzymmwljxpnmpwx...qpfsowgkihrxngnnpbffbhixpilldt"  # (500 chars),
    "fuaprqensipkeguvcxssqztlijfyvsjjraptydqijmsqiceoom...aamglrjqhjcgnzrocrqwhmjsiztfig"  # (500 chars),
    "zbrccghsefgaopppprboeftvwanlzfmsztyxfiemylskujwoet...vuglryrfndsgbspbansqkokmeajkws"  # (500 chars),
    "vovdevletsycjdaibyblxmslctxxqltsdjrncslqhyrayeqvwl...tiwvqfmohovmrgcybylciyigwzwqmw"  # (500 chars)
]
```

**Your task:**
Write a Python function `longest_common_substring(strings)` that:
- Takes a list of strings
- Returns the longest common substring (as a string)
- If multiple substrings have the same max length, return any one
- If no common substring exists (other than empty), return ""

**Example:**
```python
strings = ["abcdef", "xbcdey", "zbcdew"]
# Returns: "bcde" (length 4)
```

**Algorithm approaches:**

1. **Naive O(n²m):** For each possible substring of first string, check if it exists in all others. Very slow for long strings.

2. **Binary search + hashing O(n·m·log(m)):** Binary search on length, use rolling hash to find common substrings of that length.

3. **Suffix array O(n·m·log(m)):** Concatenate strings with separators, build suffix array, find longest common prefix across all strings.

4. **Dynamic programming O(n·m²):** For 2 strings; extend to multiple strings.

**Hints:**
- For large inputs, naive approaches will timeout
- Rolling hash (Rabin-Karp) enables O(1) substring comparison
- Binary search on answer length is effective
- Python's `difflib` can help but may be slow

**Constraints:**
- Solver must complete within 5 minutes
- Use only Python standard library
- Return the actual substring string, not just its length

Write complete Python code with the longest_common_substring function.
